　　“小子。”
<br/>　　云渊长老忽然冷不丁定叫了一句。
<br/>　　“怎么？”萧逸疑惑地看着云渊长老。
<br/>　　云渊长老道，“离开万剑碑林后，你又要四处跑了吧？”
<br/>　　“嗯。”萧逸点点头，“还是得历练。”
<br/>　　“备战古帝之墓的事吗？”云渊长老问道。
<br/>　　萧逸点了点头。
<br/>　　古帝之墓，他自然是要去的。
<br/>　　一来，或许那里有别的机缘。
<br/>　　即便得不到机缘，去见识一番，或者提升实力也是好的。
<br/>　　二来，他答应过风刹殿主和修罗殿主。
<br/>　　虽说修罗总殿主的请求，是让他去古帝之墓看看有没有修罗战体前七层的消息。
<br/>　　而修罗战体前七层，也根本就在他身上。
<br/>　　无论他之后选不选择将功法交出。
<br/>　　但无论如何，还是要去一趟，走个过场。
<br/>　　当然，萧逸不知道的是，修罗总殿主其实早便看出了他的想法。
<br/>　　“也好。”云渊长老这时说道。
<br/>　　“杀戮之路，是剑道之路上的一个标志。”
<br/>　　“我们这些极致剑修更早触发罢了，别的剑修，拿气宗宗主来说，如果他不死的话，日后突破绝世强者到达下一境界，终归也会遇到。”
<br/>　　“你现今提前触发，也踏过去了，你的剑道之路，在很长一段时间内都不会有任何瓶颈。”
<br/>　　“只要你有足够的修炼资源，足以一路高歌到绝世强者的层次，乃至突破之。”
<br/>　　“出外历练，得到修炼的途径资源也简单些。”
<br/>　　讳执事笑笑，“云渊真正想说的是，出了万剑碑林，你我三人就要分道扬镳了。”
<br/>　　“之后，事事小心。”
<br/>　　“嗯。”萧逸认真地点了点头。
<br/>　　三人，御空飞行，不过片刻，便已出了万剑碑林。
<br/>　　“告辞了…”萧逸刚打算与二人告辞离去。
<br/>　　但，话语，却并未说完。
<br/>　　萧逸的眼眸，眯了眯，直视前方。
<br/>　　云渊长老和讳执事，亦是双眼一眯，直视前方。
<br/>　　前方，有一道身影，负手而立。
<br/>　　身影，缓缓地转过了身，看向三人。
<br/>　　“老院长。”云渊长老与讳执事，同时行了一礼。
<br/>　　“洛前辈。”萧逸也微微行了一礼。
<br/>　　不错，来人正是黑云学教老院长，洛前辈。
<br/>　　虽说萧逸已不是黑云学教弟子。
<br/>　　但洛前辈毕竟指导过他的修炼，自然的，这一声‘前辈’，这一‘前辈之礼’他还是要给的。
<br/>　　“嗯。”洛前辈点了点头。
<br/>　　洛前辈的脸色，很是平淡，一如既往般平静。
<br/>　　负手而立，并未言语。
<br/>　　云渊长老皱了皱眉，率先拱了拱手，道，“老院长不是在学教吗？怎地忽然来剑域了？”
<br/>　　讳执事直接道，“老院长可是寻我们有事？”
<br/>　　“若无，萧逸小子还有要事在身，便要离去了。”
<br/>　　“若有，我和云渊二人随时听候老院长吩咐。”
<br/>　　“哦对了，我和云渊也准备回学教了。”
<br/>　　说着，讳执事对萧逸打了个眼色。
<br/>　　萧逸会意。
<br/>　　然而，刚挪动脚步，一道气息却瞬间锁定了他。
<br/>　　洛前辈笑笑，“你们是聪明人，我来此的目的，不必我说你们也清楚，何必多言。”
<br/>　　“非要老夫腆着脸，张口道明来意吗？”
<br/>　　云渊长老二人心头一突。
<br/>　　萧逸也是脸色一变。
<br/>　　“不知老院长是何意。”云渊长老问道。
<br/>　　洛前辈微微摇了摇头，“非要老夫说，那便说吧。”
<br/>　　“交出剑帝本源。”
<br/>　　洛前辈的声音，很轻。
<br/>　　却足以让萧逸三人身躯猛地一颤。
<br/>　　“我等不明白老院长的意思。”云渊长老面露疑惑之色。
<br/>　　“剑帝本源，可是在剑帝碑之内…”
<br/>　　洛前辈直接出言打断道，“我都知道。”
<br/>　　“云渊，无视，你二人心思缜密，但却骗不过我。”
<br/>　　云渊长老脸色一变，“老院长早便来了？”
<br/>　　“不错。”洛前辈点点头，“在你们离开剑帝碑时…，不应该更早些，你们进入剑帝碑时，我便在了。”
<br/>　　云渊长老和讳执事，同时脸色大变。
<br/>　　洛前辈这等人物，修为通天，若是不现身，根本无人能发现。
<br/>　　云渊长老和讳执事同时踏前一步，将萧逸护在了身后。
<br/>　　“可我二人，还是不懂老院长的意思。”
<br/>　　“还是不懂吗？”洛前辈只淡淡一笑。
<br/>　　但下一秒，一股滔天气势，从天而降，瞬间压在了云渊长老二人身上。
<br/>　　惊人的气势，竟瞬间将云渊长老二人压得脸色通红。
<br/>　　单单一股气势，便将能轻松重伤气宗宗主和欧阳烈的云渊长老二人，牢牢压制，甚至让得二人脸色难看。
<br/>　　可想而知，洛前辈实力之滔天。
<br/>　　“现在懂了吗？”洛前辈淡淡问道。
<br/>　　“还是不懂。”云渊长老二人张开了嘴巴。
<br/>　　“嗯？”洛前辈脸上的平淡脸色，已化作了皱眉。
<br/>　　轰…
<br/>　　天地气势，再次加剧。
<br/>　　云渊长老二人，已然体内一阵气血翻滚。
<br/>　　“懂了吗？”洛前辈再次问道。
<br/>　　“不懂。”云渊长老二人摇了摇头，“剑帝本源，是萧逸的东西，为何要交出？”
<br/>　　“萧逸已非学教弟子，他如何，学教无权干涉。”
<br/>　　洛前辈想说些什么。
<br/>　　忽然。
<br/>　　云渊长老和讳执事，同时暴起。
<br/>　　两人手中瞬间握剑，两道惊天剑气打出。
<br/>　　“就是现在，小子，快走。”二人暴喝一声，一把拎起萧逸，就要逃离。
<br/>　　轰…
<br/>　　不见洛前辈有任何动作，只是冷笑一声。
<br/>　　方圆百里，霎时武道崩塌，空间封锁。
<br/>　　三人的速度，本是极快。
<br/>　　一个呼吸间，已能横跨百里。
<br/>　　然而，当三人身影刚出百里范围，却是猛地身影一闪。
<br/>　　三人还未反应过来，身影，已出现出现在原地。
<br/>　　而周遭，早已被封锁。
<br/>　　“咫尺天涯？”云渊长老脸色大变。
<br/>　　萧逸虽然不知道怎么回事，但‘咫尺天涯’四个字，他听说过。
<br/>　　这是极其高深的手段。
<br/>　　他们三人，重新回到了原地。
<br/>　　但他们若想离去的话，每走一步，都将难如天堑。
<br/>　　小小一步的距离，不亚于横跨天涯海角。
<br/>　　周遭空间以及武道规则，早已被改变。
<br/>　　也就是说，从现在起，三人哪怕挪动一步都休想做到。
<br/>　　洛前辈的声音，缓缓传出，“乖乖交出剑帝本源。”
<br/>　　“或者，老夫亲自取。”
