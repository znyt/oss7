　　“借我点钱。”
<br/>　　楚涵一进房间还没见到人，就直接开口说出这番话。
<br/>　　哪知话刚出口，他就发现事情不对劲，房间里不止国王，还有其他人在！
<br/>　　“你是谁！”国师盯着楚涵，眼神森冷。
<br/>　　楚涵上下打量了国师一眼，没说话，直径走到了国王面前，道：“借我点钱。”
<br/>　　国师一挑眉，杀意已经浮现在面上。
<br/>　　楚涵却根本没管身后的人，就这么朝着国王伸出手。
<br/>　　国王这时候的表情那叫一个精彩，今天国师竟然无视了守卫冲进她的房间，她正想不到解决办法，没想到楚涵竟然来了。
<br/>　　“你要钱干嘛？”国王为了拖延时间，将问题延长。
<br/>　　楚涵老实回答：“想找个住的地方，才发现自己没钱，你知道的，我们那不用天运国的货币。”
<br/>　　“恩，我懂。”国王双眼噌亮，忽然下一句就开口道：“你不用住外面啊，住我这好了。”
<br/>　　说着还拍了拍床：“睡得下。”
<br/>　　楚涵愣住，这啥意思？
<br/>　　国师此时已经忍无可忍，怒道：“国王也太不知廉耻了吧！”
<br/>　　国王当即脸一沉，回骂道：“国师管的也太宽了吧！”
<br/>　　两人的火药味在空中交替，倒是让闯入的楚涵有些尴尬，于是他看了看窗户，道：“那我先走了，明天再来借钱。”
<br/>　　“别走！”国王大急。
<br/>　　国师却已经出手，杀机暗藏！
<br/>　　一把短刀直接从袖口飞出，直逼楚涵的后颈！
<br/>　　此人的确很强，其战力若放在地球上，已经算得上是一名不得了的十阶新人类。
<br/>　　但也只是个十阶……
<br/>　　于是下一秒，黑色的修罗战斧凭空出现，‘叮’的一声挡住这柄短刀。
<br/>　　没断，是把好刀。
<br/>　　国师诧异万分，先是不明白这黑斧从何而来，再就是惊讶于楚涵的臂力，竟然将他整个人都单手挡住！
<br/>　　楚涵想了想，人毕竟是国师，不能打破再像珈蓝国那样打破一国秩序。
<br/>　　于是没下杀手，但却伸出脚，‘啪’的一下将国师整个踢出窗外！
<br/>　　黑夜之中一个人影就这么飞出去老远，也不知道掉在了哪里。
<br/>　　国王这时候已经惊呆了，看怪物一样的看着楚涵，国师是天运国最强战力者之一，竟然被楚涵轻飘飘的一脚踢飞？
<br/>　　“你跟国师关系不好啊？”楚涵想了想，还是关心了一下这个国家的大事。
<br/>　　国王愣愣点头：“是需要铲除的对象。”
<br/>　　“额……”楚涵呆了一下：“你不早说。”
<br/>　　早知道就直接杀了啊！
<br/>　　“但是不能鲁莽。”国王连忙解释道：“国师的背后还有一些残存势力，影响朝运，也是我这二十年一直在调查的对象，只是可惜没能查出个所以然。”
<br/>　　楚涵本来想问白允儿怎么没帮你查，但一想到白允儿那性格，有些天然呆，杀人在行，查真·相还真不行。
<br/>　　国王似乎还想到了什么，苦笑道：“我怀疑二十年前要追杀我的人，就跟国师有关系，朝中还有一些我没能察觉到的危险，也同样直指当年的事，国师是唯一的线索，他死了，就什么都查不到了。”
<br/>　　楚涵想了想，也没继续问，只是手一伸：“钱还是要借我的。”
<br/>　　国王无奈，道：“你要不继承白允儿的天女之位吧？任命你为天子怎样？”
<br/>　　她不想放楚涵走，以前有白允儿，可现在身边没有一个战力强大的人，非常危险。
<br/>　　楚涵嘴角直抽：“天子在我们那是国王的意思。”
<br/>　　国王想了想又道：“那叫天男。”
<br/>　　楚涵摆摆手：“还不如教主。”
<br/>　　“主教大人也不错。”国王兴奋起来：“明日我需要出行一趟，主教大人随我一同前往吧？”
<br/>　　“啊？”楚涵也是没想到这种发展。
<br/>　　国王则是笑道：“是这样，我每年都要去各地逛一趟，寻找下落不明的皇冠。”
<br/>　　楚涵更匪夷所思了：“你皇冠丢了啊？”
<br/>　　国王眼神暗沉：“就是二十年前的那次刺杀，若不是白允儿一直在我身边让有心人无法近身，恐怕我的王位早就没了，果然白允儿一走，国师就胆大包天的来到我房间。”
<br/>　　楚涵终于明白了过来，感情这国王过的相当苦逼啊，看这天运国发展这么好，国王却近乎被国师架空，确实惨。
<br/>　　当天夜里国王就这么拖着楚涵不让走，将天运国大事小事都说了一通，也不管楚涵爱不爱听。
<br/>　　对此楚涵也很无奈，这国王估计也是被吓惨了。
<br/>　　“你难道天天不睡觉？”楚涵终于受不了，“你不睡我要睡啊。”
<br/>　　国王抿了抿嘴，指着床道：“那你睡这。”
<br/>　　楚涵翻了个白眼：“这样，你要是真害怕，就给我支个屋，我也有落脚的地方。”
<br/>　　“好。”国王终于喜笑颜开，当场自己动手，将一张塌椅一横，意思就是楚涵睡这。
<br/>　　楚涵更无奈了：“你意思是我还是要睡在这个房间？”
<br/>　　国王点头：“咱俩一个屋。”
<br/>　　一夜无话，楚涵也不知道自己是怎么睡着的。
<br/>　　第二天一大早，一群侍女进来的时候，纷纷震惊于睡在榻椅上的楚涵。
<br/>　　天运国的国王也是个奇女子，竟然毫无反应的该干嘛干嘛，梳洗打扮之中，侍女们几乎每隔几秒就偷看楚涵一眼，好奇心爆棚。
<br/>　　这个男人是睡，怎么睡在国王房间里？
<br/>　　楚涵懒得解释，翻了个身继续睡：“你打扮完了再喊我。”
<br/>　　女人要梳洗打扮很久，他还能再睡个回笼觉。
<br/>　　国王也不甚在意的应了一声，引得侍女们一个个惊讶无比，这两人什么关系？
<br/>　　前段时间还听大臣们在吵国王的婚事，才几天竟然房间里就藏男人了！
<br/>　　这八卦卖出去能卖多少钱？
<br/>　　等到楚涵再次睡醒睁眼，国王也终于梳洗完毕，今天的国王特地乔装了一翻，不再身穿王衣。
<br/>　　楚涵也明白，他俩今天要去的地方，恐怕也不方便透露身份。
<br/>　　国王走出王宫，没有带任何随从，楚涵跟在其后免不了好奇，那些侍女侍卫竟然也不担心，就这么放任国王这么大摇大摆的走出去。
<br/>　　“以前都是白允儿陪我的。”国王开口，解答了楚涵的疑惑。
<br/>　　的确有白允儿在的话，就不需要任何随从。
<br/>　　谁打得过她啊？
