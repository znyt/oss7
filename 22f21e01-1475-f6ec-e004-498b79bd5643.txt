        明司寒敏捷地一脚踹开后，小巷的阴影之中，又走出两个黑衣人来。
<br/>
<br/>        当被他撂倒的两个黑衣人吃痛地从地面上爬起来，与后来的两个黑衣人准备向他发出群攻的时候，耳畔突然传来“嗖嗖嗖”的声音。
<br/>
<br/>        三个黑衣人倒地，其中一个慌慌张张地四下探看，见情形不妙，撒腿便跑了。
<br/>
<br/>        “噌”地一声，一道黑影恍若从天而降。
<br/>
<br/>        明司寒惊了一下，立即挥起拳头，摆出迎战的姿势。
<br/>
<br/>        “是我！”女人的声音清澈利索。
<br/>
<br/>        明司寒定睛一看，才发现是裴萱妍。
<br/>
<br/>        “没想到，你还挺能打啊！”裴萱妍将手中的麻醉枪插入腰后的枪夹盒中，红唇微扬。
<br/>
<br/>        “你怎么在这？”明司寒不禁剑眉微蹙。
<br/>
<br/>        裴萱妍抬起手来，搭在明司寒的肩膀上，围着他的身子，妖媚地转了一圈，一双桃花眼在他身上仔细地打量了一番：“嗯，还好你没受伤，不然我要心疼了。”
<br/>
<br/>        明司寒不耐烦地甩开裴萱妍的手，质问道：“那些人是你惹来的？你到底是什么人？”
<br/>
<br/>        “我是什么人不要紧，要紧的是，现在你已经成为那群人的掌中之物了。他们要抓你，恐怕是因为你就是国际上排名第三的黑客‘C’，你虽然国际排名只在第三，但你在去年的那场排名赛上，是故意输掉的吧？以你的实力，排第一完全没问题，我没说错吧？”裴萱妍淡定自若道。
<br/>
<br/>        “……”明司寒此刻剑眉紧锁，默不吭声。
<br/>
<br/>        裴萱妍接着说道：“我接到上头的命令，现在的任务是贴身保护你。”
<br/>
<br/>        “不需要。”明司寒冷语拒绝。
<br/>
<br/>        裴萱妍会心一笑：“如果我不在你身边的话，今晚这种事情，日后还会发生。难道，你就不怕，他们会去对你喜欢的那个小丫头动手吗？”
<br/>
<br/>        “他们若是敢，我一定不会放过他们！”明司寒咬牙切齿地握紧了双拳。
<br/>
<br/>        裴萱妍从腰间掏出一枚智能手环，递给明司寒：“这是我两的通讯电话，你戴上。你放心，你不仅救过我的命，还是我是上司亲口下令要保护的对象。所以，我不会害你。”
<br/>
<br/>        “我凭什么信你？”明司寒瞥了裴萱妍一眼，径直往前走。
<br/>
<br/>        裴萱妍随后跟了上去，走在明司寒的身侧，条条是道地解释：“你所有的资料，上司都已经发给我了。想必，今晚那群人的幕后大佬也一定对你了如指掌。他们隐秘在暗处，不想被人知道，自然不会波及你身边的人。但是，你喜欢的那个丫头来历可不简单。她是傅夜沉的女儿，傅夜沉有暗势力，妻女是他唯一的弱点……”
<br/>
<br/>        裴萱妍的话到这，明司寒突然停下了脚步。
<br/>
<br/>        “你们的人也想动星儿？”明司寒凝眉质问。
<br/>
<br/>        裴萱妍微微摇了摇头：“我们的上司和傅夜沉的暗势力是联盟关系，所以，我们不会动自己盟友的人。”
<br/>
<br/>        “……”明司寒不禁若有所思起来。
<br/>
<br/>        裴萱妍接着说道：“你爷爷突然想起要跟傅家以联姻的方式去和解冷明两家的世仇，只怕也是想借用傅夜沉的暗势力，为你们明家在军政界保驾护航吧！我都已经把关系挑得如此明了了，你若还不信，我也没办法了。”
<br/>
<br/>        “你想怎么着，都可以，但麻烦你不要妨碍到我。”明司寒淡淡地说道。
<br/>
<br/>        裴萱妍会心一笑：“你知道连星儿为什么不喜欢你吗？”
<br/>
<br/>        “关你什么事。”明司寒冷语道。
<br/>
<br/>        裴萱妍却啧嘴道：“女孩子得哄着。你连哄女孩都不会，还指望她会喜欢你？”
<br/>
<br/>        “……”明司寒薄唇紧抿，懒得跟这个女人废话。
<br/>
<br/>        连星儿原本以为自己和明司寒的那个流言蜚语会有个结果。
<br/>
<br/>        结果，这样的结果，比她想象中的还要糟糕。
<br/>
<br/>        连带着同寝室的顾千凝、赵安彤和路娅玟一起遭了殃。
<br/>
<br/>        她们的寝室门，不知道被哪个缺德的人塞了口香糖，就连挂在走廊上晾晒的衣服也不知道被谁给取了下来甩在了地上，还有无数个肮脏的脚印踩在了上面。
<br/>
<br/>        这特么地是女大学生该干的事吗？
<br/>
<br/>        连星儿二话不说，直接掏出手机拍下照片后，直接发给了明司寒兴师问罪。
<br/>
<br/>        就这么整我，你不觉得你很幼稚吗？——连星儿。
<br/>
<br/>        明司寒收到连星儿的这条短信时，正在上高数课。
<br/>
<br/>        有配图，有质问，那丫头似乎挺恼火。
<br/>
<br/>        可他真的很冤。
<br/>
<br/>        “明司寒同学，你上来替同学解答一下。”戴着黑框眼镜的教授，对着话筒铿锵有力地喊道。
