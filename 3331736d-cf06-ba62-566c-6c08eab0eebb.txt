　　南署基地的大战在三天后终于结束，当最后一只丧尸被打破脑袋的时候，这座往日里干净而繁华的基地，当今已然遍地腐尸。
<br/>　　那个有和上赶制而出的大音响，当下停止了循环了好几天的播放，那古怪的音律消失后，基地也刹那间变得空寂和可怕。
<br/>　　引入眼帘的尸山，打碎了脑袋而流淌而出的脑浆，黑色的黏稠物到处都是。
<br/>　　黑色的丧尸血浸染了整个基地，延着地面的细缝渗透入地下，让这块地域在今后很长的一段时间里都寸草不生，并长期弥漫着一股腐臭味道。
<br/>　　眼下南署基地的场景一点不亚于地狱，心理承受能力不好的人，恐怕一刻也待不下去，不被这到处都是的恐怖场面逼疯才怪。
<br/>　　但在这里战斗了足足一周的狼牙战团成员，却在这极短的时间内习以为常，结束战斗的他们从各个角落里钻出。
<br/>　　有基地的建筑物，有房屋楼顶，有狼牙工程队建造的高台，也有地底地洞，更有从尸堆底下爬出来的。
<br/>　　他们衣衫褴褛，皮肤上或多或少的沾着丧尸的血肉，他们寻找着自己的小队，聚集后拨开这些堆积成山阻挡了道路的尸体，带着疲惫及骄傲，慢慢的向着指挥台集合。
<br/>　　三战队的战斗也落下帷幕，那一圈高耸宽大的基地外墙，当下已经破烂不堪，外墙顶端更是铺满了尸体，在百万高阶丧尸一拥而上的战斗中，没有人能抽·出空闲时间来清理这些死去的丧尸，于是三战队的成员便在这越堆越高的城墙上，大战了七天七夜！
<br/>　　多亏了大量的银净水，否则哪怕强悍入徐锋这样的新人类，也无法坚持到最后。
<br/>　　高少辉也累的半途离战，当帮助了三战队解决了大批高阶丧尸之后，他便早在一天前就已经离开。
<br/>　　两墙之间的第一战区，也被丧尸填满，一只只的堆积在一起垒砌老高。
<br/>　　这座基地连站的地方都没有了，全部都是被他们杀光殆尽的丧尸！
<br/>　　“战斗结束，集合。”何峰的声音响起在扬声器内，回荡在这空寂安静的基地各处。
<br/>　　很快，狼牙各小队都从内部走出，一队队的排列成方阵靠近指挥台。
<br/>　　这里同样被丧尸的尸体占据，但在那些从迷宫区走出的新兵到达之前，在第一战区战斗的狼牙战团老兵们，就已经着手粗略清理了一遍。
<br/>　　于是整个狼牙战团的成员，就这么拥挤的站在指挥台附近的空地上，哪怕旁边就是小山一般高的尸体堆。
<br/>　　何峰站在指挥台上，望着下方密密麻麻的全体狼牙战团成员，双目布满红血丝的他还需要为这次大战画上句号。
<br/>　　“清点伤亡情况。”他的声音带着固有的冷静，沉稳不似一个才二十来岁的年轻人。
<br/>　　这次大尸潮的战役史无前例，更是毁灭性的攻击力度，狼牙战团哪怕老兵都在所难免有不小的伤亡，更别说那些才训练了两个月的新兵。
<br/>　　很快这次大战的具体数据便得以统计，新兵伤亡人数为一万，这是一个不小的数字，哪怕是何峰在得知的那一刻，也心下沉重无比。
<br/>　　这死去的一万人中，只有一千是死于战术调整之前，而余下的九千之多，却是在战术调整后，七百万丧尸一股脑冲进基地之后一次性造成的大创。
<br/>　　才堪堪一阶或两阶的新人类，在数量碾压的尸潮风暴之中很难保命，尤其是在那最为黑暗恐怖的几分钟内，大多数小队都被冲散，个人更加难以招架这种攻势。
<br/>　　而与丧尸的战斗也和其他战斗不同，要么只是轻伤不足为患，要么便是死亡。
<br/>　　因为丧尸带有病毒，新人类的免疫力随着等阶逐步升高，但也有在多次受伤后感染的情况，那九千人有大半是被丧尸当场啃噬致死，剩下的一小部分则是受到了感染，逐步尸变而死……
<br/>　　再如何残酷也必须接受的一个事实便是，一旦尸变为丧尸，便无法挽回！
<br/>　　而丧尸，是不被允许在这场大战中存活的。
<br/>　　哪怕最新研究而出的解毒剂，也只对刚受到感染，病毒还未入侵大脑的情况下才有用。
<br/>　　有限制，也稀有。
<br/>　　新兵的伤亡情况令人心情沉重，狼牙战团的老兵伤亡也堪称惨烈。
<br/>　　精英队的三千人死亡人数为两百还算能接受，警戒队的五千人却有七百人死亡，侦查队的游走能力最佳，伤亡人数只有十人。
<br/>　　三战队总计一千人不到的人数，则是有两百人死亡，每一名都是从狼牙精英队中选拔而出的最强者，其中甚至有好几名少将，都曾名震整个狼牙。
<br/>　　一百多万的高阶丧尸，哪怕有着高少辉的中途支援，也还是让三战队损失惨重。
<br/>　　射击队和防守队由于并非近战，没有和丧尸群有过正面战斗的他们，伤亡数为零。
<br/>　　战后统计的具体数据，何峰没有任何隐瞒的公布而出，他念着那些牺牲成员的名字，长长的名单念了很久，随着每一个名字的报出，都让空地上的不少人心头发紧。
<br/>　　对新兵来说，死去的人是他们朝夕相处两个月的同伴，是在这七天内最信任的战友。
<br/>　　对于狼牙老兵来说，越往后报出的名字，越让他们难以接受，不是一天两天更不是两个月的交情，而是长达一年甚至两年。
<br/>　　尤其是三战队，当何峰念到名单上最后几行的时候，那些名字后面跟着的一连串头衔及战绩，那些名字所代表的意义，顿时让所有狼牙老兵都红了眼眶。
<br/>　　新兵们也在听到最后阶段的时候，心中触动巨大，狼牙战团不仅将每一名死去的人都记录的清清楚楚，甚至还会不差一毫的将此人在狼牙的事迹都记录。
<br/>　　越是军衔高者，越是在狼牙时间长的人，跟在其名字后方的那些数据，就越是让人心下震惊。
<br/>　　新兵们听着那些恐怖的战绩，听着那一个个少将亦或是上尉的名字，全体都呆呆的站在原地震惊到无以复加。
<br/>　　这时候他们感受的不仅是这场创世纪大战后的荣耀，更有狼牙战团的强大如斯，以及战友间最为真切的情谊。
