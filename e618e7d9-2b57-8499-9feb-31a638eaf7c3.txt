　　叮铃铃……，叮铃铃……，叮铃铃……
<br/>　　巴坦伦秘密别墅的书房中，他办公桌上那部电话响个不停。
<br/>　　詹弗妮痴痴呆呆的坐在办公椅中，对办公桌上的电话铃声置若罔闻。
<br/>　　杀掉了警卫队长后，秘密别墅中那些警卫队员变得更加肆无忌惮起来。以前巴坦伦活着时候，这个书房是只有他和詹弗妮才能进入的地方，即便是警卫队长，也要得到巴坦伦的命令。也正是因为如此，很多值钱的东西都被巴坦伦藏在了这间书房中，洗劫着别墅中财物的警卫队员毫无顾忌的进出书房，把书房中的古董、名画、珍贵的装饰摆件往外搬。
<br/>　　保险箱的门不知道密码打不开？
<br/>　　几个警卫队员直接把保险箱从墙壁中撬了出来，把保险箱和保险箱中的财物一起带走，很快，书房中值钱的东西就被搬得差不多了。
<br/>　　一个警卫队员注意到了书桌上又一次响起的电话机，走了过来。
<br/>　　詹弗妮拉开抽屉，突然从抽屉中拿出一把手枪握在手中。看到詹弗妮突然拿出手枪，那个原本打算把桌子上电话机拔走的警卫队员愣了愣，随后抓起背在自己身后的AK自动步枪，眼神凶狠的盯着詹弗妮。
<br/>　　詹弗妮没有把手枪对准那个走向书桌的警卫队员，而是直接把手枪抵在了自己下巴上，拉开击锤，闭上眼睛的时候，两道眼泪从她眼睛中滚落下来在她那张精致的脸上留下两道泪痕。
<br/>　　她扣着扳机的手指不断的在颤抖着，第一次扣下扳机的时候，在压下二道火后，她又把扳机给松了开来，到了最后一刻，詹弗妮还是怕了；睁开眼睛，看了眼围在自己身边眼神不善的警卫队员，詹弗妮惨然一笑，深吸一口气，扣下了扳机。
<br/>　　砰！
<br/>　　子弹从詹弗妮下颚处穿入，弹头穿过了她的脑袋，掀开了她的头盖骨打穿了她身后的玻璃。
<br/>　　弹头从她脑袋中飞出的一刹那，一大蓬血浆喷射而出，溅落在她身后的玻璃墙上。
<br/>　　坐在办公椅中的詹弗妮身体一歪，窝在办公椅中彻底断了生机，握在她手中的手枪也是掉落在书房铺着手工地毯的地板上。
<br/>　　詹弗妮最终还是选择了自杀。
<br/>　　围在旁边的警卫队员各自散开，继续在书房中寻找着值钱的东西。那个警卫队员拔掉了办公桌上电话机的电话线，抱着电话机离开了书房。
<br/>　　……
<br/>　　布克尔特看了眼突然传来忙音的卫星电话，眼神闪烁了一下，当他第三次拨通巴坦伦别墅书房电话的时候，已经是无法接通的状态。
<br/>　　出事情了！
<br/>　　布克尔特意识到出事了。
<br/>　　为什么比洛多敢突然对自己的种植园下手，派出武装直升机和机动部队对种植园发起突袭？因为巴坦伦死了，联合政府军中没有人再能制衡住比洛多！
<br/>　　是谁杀了巴坦伦？不可能是比洛多，只能是来营救被自己绑架的那个女人的特种部队！
<br/>　　联合比洛多的机动部队，对巴坦伦实施斩首行动，让比洛多的行动不再受到制衡，所有的一切都串联到了一起。
<br/>　　布克尔特眼睛微眯，通过步话机试图跟派伯克取得联系，让他们放弃种植园撤离，但是他并没有联系上派伯克。
<br/>　　比洛多的机动部队应该已经让种植园那边彻底失守了，那边的守备武装也应该已经全军覆没。
<br/>　　“巴布沙，立刻把那个女人带来，我们准备撤退！”
<br/>　　布克尔特拿起放在桌子上的M4A1自动步枪，检查了一下弹药，对屋子里一个武装毒贩发出命令。布克尔特已经清楚的意识到了自己现在的处境，也明白佟千雪是他手中最大的底牌，想要活命，就必须控制住佟千雪！
<br/>　　为了歼灭掉龙隐部队，布克尔特可以说是做了充足的准备，他唯一没有料到的，就是龙隐部队会跟比洛多达成合作，而比洛多这个老奸巨猾的家伙居然还会答应跟他们合作！
<br/>　　丛林中，罗昊已经打开了手中紧凑型HK416自动步枪的保险，看到村子中突然陷入混乱，罗昊意识到布克尔特已经有了察觉。
<br/>　　“动手！”
<br/>　　咻！咻！
<br/>　　罗昊一声令下，李毅峰和慕容飞同时开火，两颗子弹解决掉了村子机枪哨塔上的机枪手，这两颗“大门牙”是战术中队进入村子最大的威胁。
<br/>　　机枪手中弹倒下，副机枪手还没来得及补上，两颗枪榴弹就飞了过来，落在机枪哨塔中，把机枪哨塔塔顶都炸得坍塌下来，彻底把这两颗“大门牙”给拔除。
<br/>　　轰！轰！轰！轰！
<br/>　　战术中队身后的丛林中，突然传来阵阵爆炸声，黑红色的火焰冲天而起，丛林中的动物四散奔逃。
<br/>　　丛林中负责埋雷的武装毒贩看到了被杀掉的同伴，触碰到他们尸体的时候触发了诡雷。
<br/>　　罗昊手中握着HK416，身体做着各种战术规避和战术动作，敏捷的移动着向村子里推进，安装在枪口的高性能消音器有效的抑制了枪口的上扬，钨芯穿甲弹精准的朝着村子中那些武装毒贩身上招呼。
<br/>　　一个武装毒贩手中抱着米尼米轻机枪躲藏在一个混凝土掩体背后，试图对罗昊进行一次偷袭；只是他的移动真的不算快，被罗昊准确的捕捉到。
<br/>　　枪口低压，对着拿出混凝土岩体就是一个短点射。
<br/>　　那块可以有效抵御覆铜铅芯弹的混凝土掩体在钨芯穿甲弹面前就跟豆腐一样，完全起不了任何防御作用，很轻易的就被穿透。
<br/>　　子弹在混泥土掩体表面留下三个弹孔，将混凝土掩体穿透后，贯入到掩体背后的武装毒贩身体中。身中三弹的武装毒贩扑倒在地，一颗子弹打入了他的肺部，导致他的口鼻中不断有血沫涌出，他双眼瞳孔开始涣散，很快就失去了生命的光泽。
<br/>　　屋子中，佟千雪听到从屋子外传来的枪声和爆炸声，立刻激动起来，眼眶忍不住发红，“他们来了，他们终于来了！”
<br/>　　佟千雪取出娜塔莉娅偷偷交给自己的钥匙，打开了自己手腕和脚腕上的镣铐。
