　　面对款款而来的独孤玉凤，文质彬彬的家伙明显愣了一下，但很快便反应过来，将手里的折扇‘唰’的一下合到一起：“黑夫人，难道你真的认为我会让您这么容易的离开么？”
<br/>　　“你有别的选择么？”独孤玉凤一边走，一边说道。
<br/>　　“我觉得有。”文质彬彬的家伙脸上的笑容一收，对着身后一挥手，立刻有四、五个人影窜了出来，站到他的身后。
<br/>　　“就凭这几个人？”独孤玉凤显然没有将他们放在眼中，脚下速度不变，依旧向他们一步一步逼了过去。
<br/>　　站在她对面的那个家伙瞳孔微微一缩，脸上闪过一丝狰狞：“好，既然黑夫人想成全我等，那就不要怕在下不客气，来人，给我……。”
<br/>　　此人话说了一半，便听到不远处的林子里面传出一种十分奇怪的声音‘砰’，而几乎是在瞬间，在无数人的注视下，他身后的某一个家伙脑袋就好像是被攻城凿砸了一样，‘啪’爆成了无数碎块，白色的脑浆混合着鲜血溅得身边几人满身都是。
<br/>　　这一切都发生在极短的一瞬间，如此惨烈的死法几乎吓退了所有人，就连正准备起身追向独孤玉凤的双胞胎兄弟都不由自主的停下了脚步。
<br/>　　只有独孤玉凤丝毫不为所动，甚至连看都没有看那个死掉的倒霉蛋儿一眼，只是看那着拦在自己面前的几人淡淡说道：“还有谁想成为第二个？有的话可以站出来！没有就滚开。”
<br/>　　到底要不要离开？所有人都有些犹豫，好好的一个人就这么不明不白的死了，脑袋直接被人从远距离直接爆开。
<br/>　　要知道，这可是黎明之前啊，一天中最黑暗的时候，尽管在凉亭的周围有火把照着，可是站在百步之外依旧是什么都看不清楚。
<br/>　　但就算是这样，独孤玉凤提前埋伏人的手依旧在极远的地方干掉了他们的人，而且看上去并没有费太大的力气。
<br/>　　“这就是新式步枪的威力么？黑夫人，能不能透露一下？”就在那文士犹豫着要不要让开的时候，鬼面在远处用其特有的声音问道。
<br/>　　“对，这就是新式步枪的威力，你可要试试？”独孤玉凤扭过头，看着鬼面说道。
<br/>　　“不必了，在下相信夫人。”鬼面飞快的摇了摇头，语气诚恳的说道。
<br/>　　“算你识相。”独孤玉凤哼了一声，转回头再次看向自己前面几人说道：“看来你们已经做出了决定，对吧？”
<br/>　　“没，我们……”拿着折扇装犊子的家伙刚刚摇头，耳中便再一次传来了刚刚的那种声音，接着就在他眼前，两个家伙的脑袋一如刚刚的那个家伙，直接爆成了碎片。
<br/>　　接着，又是两声枪响，又有两人倒下。
<br/>　　至此，那个拿着折扇的家伙再也不淡定了，兔子一样跳起来，大声叫道：“够了，够了，我让路，让路了！”不过他这话喊的晚了些，他身后出来的那几个家伙已经尽数倒在血泊之中。
<br/>　　太残暴了，今天晚上来到这里人几乎全都看的呆住了，他们还是第一次见到被新式步枪打死的人是什么样子，估计晚上回去少不得做上几晚的恶梦。
<br/>　　而且这还并不是最重要的，重要的是听枪声的距离，应该是在三百步之外。
<br/>　　那可是三百步啊，这怎么可能呢，在如此黑暗的环境中怎么可能有人会看到这么远的距离，而且还是枪枪打头。
<br/>　　难怪独孤玉凤敢独自一个人来这里赴约，在这样的手段之下，别说这里只有几百个各怀鬼胎的奸细，就算是有上千又如何。
<br/>　　独孤玉凤似乎并不意外折扇男的表现，面无表情的看的他一眼，然后随手将手里拿着的那份资料往地上一放便继续向林子外面走去，时间不大已经消失在黑暗之中，自始而终没有任何一人敢再拦她。
<br/>　　只不过在独孤玉凤离开之后，树森中却爆发了一场战斗，数方势力为了地上那几张纸打的不可开交，最后在死了近百人之后，资料被鬼面和那个折扇男各自抢走了一半。
<br/>　　至于说情报科……，因为被数个势力合击，情报科的损失是几方之中最大的，但最后却依旧没有能够将东西拿回来。
<br/>　　……
<br/>　　话分两头，却说这独孤玉凤在离开树林之后，在路边的一个小店里找回了自己寄养的战马，等了片刻之后程华安和舒天各自背着一把样式奇怪的步枪才从另外两个方向赶了过来。
<br/>　　而之所以说他们的步枪样式奇怪，那是因为他们两个的枪上都多了一个东西，一个类似于单通望远镜的东西，也正是因为这个东西才让他们能在数百步之外弹无虚发。
<br/>　　“嫂子，这东西你是从哪里搞到的？用来偷袭真是绝了！”程华安在重新上路之后的第一句话，就是称赞起那带有瞄准镜的步枪。
<br/>　　不过独孤玉凤似乎并没有听到程华安在说什么，只是骑在马上静静的想着什么，倒是舒天在一边接过话题说道：“这种东西除了科学院还有什么地方能有，没见识。”
<br/>　　“老子没见识？”程华安歪着脑袋，指着自己的鼻子说道。
<br/>　　“这不是明摆着的事儿么？”舒天摊了摊手说道。
<br/>　　“哎你……”程华安被舒天气的吹胡子瞪眼，憋了半晌才说道：“你有见识，你有见识怎么不说说这东西叫啥。”
<br/>　　“望远镜呗，还能叫啥。”舒天理直气壮的说道：“左右不过就是把望远镜装到了枪上罢了，也值得你大惊小怪。”
<br/>　　正在想事情的独孤玉凤被两人吵的有些头疼，回身瞪了他们一眼：“好了，你们两个别吵了，若是再想吵就滚回情报科去。”
<br/>　　听到独孤玉凤撵他们回去，程华安和舒天两人全都闭上了嘴，互相瞪了一眼之后，全都不说话了。
<br/>　　而独孤玉凤却在这个时候再次开口说道：“回头这两支枪先送回去，我们几个现在的情况并不算是安全，这两支枪放在我们身边一个不好就会落入敌人手里，所以还是送回去的好。”
