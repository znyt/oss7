　　第408章
<br/>　　程大雷和秦蛮的交手似缓实急，二人变招都很迅速，这是因为在对方出手之际，便立刻知道该如何应对。很多时候都是互相绕着转圈，紧紧盯着对方双肩。
<br/>　　这在普通人看来，过程是分外无聊的。但对于赵子龙这些人来说，却能看出更多的东西。
<br/>　　赵子龙、关鱼、张肥、高飞豹、秦蛮算是山寨的五虎上将。平日五人经常接触，每人率领一支队伍，互相之间也有竞争。手下的士兵也经常议论，五个人谁比较厉害。
<br/>　　若论枪法，当属赵子龙，但领兵打仗的统率能力则是关鱼最强，可二人碰上张肥，都不敢说必胜。张肥这个人又猛又狠，往往能发挥出常人难以想象的能力。至于秦蛮，属于比较踏实，稳扎稳打。总的来说，五人之间没有太大差距。
<br/>　　可今日大家发现，秦蛮从昏迷中醒来，实力突飞猛涨。其他四人心头琢磨，如果四人现在和秦蛮动手，怕谁也没有胜算。
<br/>　　此时，擂台上的切磋已经结束。双方点到为止，本也无所谓输赢。如果真的想分出输赢，必须是生死之争，死的人输，活着的人赢。
<br/>　　程大雷的目的，是对秦蛮的实力有个估计就足够了。系统没有诓自己，秦蛮果然升到了绝世，等他将绝世等级的力量全部消化，估计也用不了多久时间。
<br/>　　程大雷心里很是高兴，如今除自己外，山寨又增加一名绝世等级的高手。日后再碰见敌人，程大雷心中也有了底气。
<br/>　　程大雷独自回了将军府，而秦蛮刚走下擂台，就被人留住了。赵子龙、高飞豹一伙人围着他，目光像是在打量一头混入人群的大猩猩。
<br/>　　“秦大哥，你怎么突然变得这么厉害？”赵子龙。
<br/>　　“有么，有很厉害么。”秦蛮不是很能适应这些人的热情。
<br/>　　“秦大哥，咱们上去比划比划。”高飞豹已经摩拳擦掌，跃跃欲试。
<br/>　　“还是不要了，我怕伤了你。”秦蛮。
<br/>　　呃……老实人说气话来也是够噎人的。
<br/>　　高飞豹深吸一口气，平复着呼吸，认真问：“秦大哥，你这本事究竟是从哪里来的，不可能突然变得这么厉害吧。”
<br/>　　秦蛮想了想，压低声音道：“我告诉你们，你们可别往外说……”
<br/>　　众人都竖起了耳朵，谁也不想错过一个字，就听秦蛮认真道：
<br/>　　“仙人提起我啦……”
<br/>　　秦蛮虽然说了要保密，但这种事，能守住秘密才是有鬼。七口八舌，消息很快就传开了。对于一般人而言，这件事不算什么，可赵子龙几人则都格外上心。
<br/>　　本来大家都在同一水平线，谁也不差谁多少，凭什么仙人提了你一句，你蹭地就超过我们了。
<br/>　　张三爷是个爆脾气，道：“不行，我赶紧回去给仙人上香，让仙人也提我一句。”
<br/>　　“三爷等等我，同去同去。”高飞豹。
<br/>　　赵子龙和关鱼面面相觑，关鱼打了个哈哈，道：“子不语怪力乱神，神鬼之说只是虚妄。”
<br/>　　赵子龙重重的点点头：“男儿本事当一招一式学来，求神问卜，这是愚夫愚妇才会做的事情。”
<br/>　　“子龙明白就好。”关鱼道：“我还有些军务要处理，咱们晚上再聊。”
<br/>　　“是，是，我恰好也想起一些事。”
<br/>　　赵子龙在城里转了一圈，鬼鬼祟祟往身后看了一眼，见没人跟着，转向另外一个方向。
<br/>　　鬼神之说，不可尽信，但也不可不信。子虽然不曰怪力乱神，但子可没说世上没有怪力乱神。程大雷本人，就是个明显的怪胎么。
<br/>　　他转了一圈，去向奥黛琳娜的住处。本朝的仙人要拜，异域的神仙说不得会管用呢，在这一点上，赵子龙觉得自己不可有门户之见。
<br/>　　刚推开门，就见关鱼站在院子中，面前立着奥黛琳娜。
<br/>　　“圣女殿下，我突然对你们的教义有感，就是想问一下，你们的神仙会不会托梦？”
<br/>　　听到身后的脚步声，关鱼回过头来。他和赵子龙目光相对，两人都僵在那里。
<br/>　　气氛一时十分尴尬。
<br/>　　……
<br/>　　程大雷悠哉悠哉的走在返回城主府的路上。城池得到升级，原先的将军府也成了城主府，城门的牌子，也由琴川关改成了蛤蟆城。
<br/>　　现在程大雷也从山寨的贼头，变成了蛤蟆城城主。可无奈的是，大家对他的称呼一直改不过来，仍然以『大当家』相称。
<br/>　　程大雷还没走回将军府，就被急匆匆走来的刘悲拦住了。
<br/>　　“大当家，大当家……”
<br/>　　程大雷看他跑得气喘吁吁，也吓了一跳，山寨刚安稳几天，这是又出事了。
<br/>　　“怎么了？”程大雷问。
<br/>　　“福德勒走了。”
<br/>　　“走了？”程大雷怔了怔：“怎么走的？”
<br/>　　福德勒一直在医馆养伤，经过这些日子，身上的伤势已经好转。只不过瞎掉的一只眼睛却永远回不来了。所以程大雷仍将他留在医馆，只盼着他能早日从阴影中走出来。
<br/>　　然而今天福德勒独自离开了琴川关，不告而别。
<br/>　　“不是有人看护他么，怎么就让他走了？”程大雷想不明白，成了独眼龙的福德勒能往那里去。
<br/>　　“看守被他打晕了，当时大家都在演武场，所以没人看见他离开。城门的卫兵还以为他只是出城散散心，也没有细问。”
<br/>　　福德勒在山寨的人缘一直不好，一来是他本人沉默寡言，不喜与人交往，二来是因为他是个戎族，帝国人对戎族总有点芥蒂的。再加上他现在成了独眼龙，整个人看着阴沉，更加没人愿意和他多聊。
<br/>　　“这是要往哪儿去啊？”程大雷下意识道。福德勒该不会傻到去找宋伯康报仇吧，不，福德勒有些木，但绝不是张肥、高飞豹这样的愣货。
<br/>　　“大当家，他还给您留了一个字条。”刘悲。
<br/>　　程大雷拿过来看了一眼，见歪歪扭扭的字写在一块白布上，上面四个字联在一起：
<br/>　　『我要回家』
<br/>　　回家？程大雷愣了愣，突然发现，自己对福德勒一无所知。甚至李行哉对他也没什么了解。他似乎凭空而来，又凭空而去。
<br/>　　他的家究竟在哪里？
<br/>　　
<br/>(本章完)
