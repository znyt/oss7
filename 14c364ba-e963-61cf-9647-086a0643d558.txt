　　无穷无尽的玄力入体，苏莫不敢怠慢，立刻极力的炼化了起来。
<br/>　　他的身上气势越发的强大，伴随一声声沉闷的响声，气息不断的涌动。
<br/>　　时间缓缓流逝，苏莫不断的吞噬，不断的炼化，但他始宗没有突破。
<br/>　　足足过去了一整个天，消耗了大量的神丹，修为早已达到了武帝境四重的极限，但仍然无法跨出那一步。
<br/>　　呼！
<br/>　　苏莫停止了修炼，睁开了眼眸，长长的出了一口浊气。
<br/>　　他不禁暗叹一声，随着修为越来越高，再想突破已经不再那么容易了。
<br/>　　修为沉淀还是不够，需要在巩固一段时间。
<br/>　　低头思索了片刻，苏莫不禁莞尔失笑，他的修炼好像只剩下了提升修为，其它的基本上都舍弃了。
<br/>　　比如，一个正常的武帝境武者，基本上除了提升修为之外，将大量的时间用来练习武技，以及参悟奥义之上。
<br/>　　而他，武技已经很久没有修炼了，奥义也很少参悟，只是一味的提升修为。
<br/>　　没办法，相比于其它方面，提升修为对于他来说最简单。
<br/>　　“以后还是抽出一点时间，参悟奥义，修炼武技吧！”苏莫暗叹一声。
<br/>　　没有突破修为，苏莫也没有强行硬冲，准备再沉淀一段时间。
<br/>　　随后，他便离开了玉镯空间。
<br/>　　回到洞府以后，苏莫也没有再修炼，每日在山谷中练习剑法，或者参悟吞噬奥义。
<br/>　　吞噬奥义极为重要，能增加他吞噬战魂的威力，不过目前此奥义只有一阶，提升起来非常的缓慢。
<br/>　　就这样，苏莫又平静的度过了小半个月。
<br/>　　这一日，一名青年来到了火元峰，被苏莫请进了山洞之中。
<br/>　　这是一名消瘦的青年，正是皇祖圣朝派系弟子任云都。
<br/>　　“苏莫，过几日圣子殿下要组织派系中一些弟子，去圣朝历练，你也一起去吧！”任云都对苏莫说道。
<br/>　　“哦，可以！”苏莫点了点头，他也没有多问便答应了下来。
<br/>　　每天在火元峰待着，未免太过枯燥了，出去走走也好。
<br/>　　“那好，三日之后的清晨，我们在外府西苑广场集合！”任云都道。
<br/>　　“我必准时到达！”苏莫道。
<br/>　　随后，两人又闲聊了几句，任云都便告辞离去了。
<br/>　　三日时间转瞬即过，苏莫离开火元峰前，向聂乾和韩盈嘱咐了一番。
<br/>　　在他离开的这段时间里，两人尽量少离开火元峰，以免被太史世家派系的人报复。
<br/>　　朝阳初升，晨雾弥漫，苏莫便来到了西苑广场。
<br/>　　此时，广场上已经有二十余名外府弟子，有男有女。
<br/>　　这些外府弟子，均是武帝境的修为，修为从武帝境一重到武帝境六重不等。
<br/>　　不过，这些人身上，都有一个共同的特点，那就是煞气极重，显然个个都是杀过很多人的存在。
<br/>　　“见过苏兄！”
<br/>　　“苏莫！”
<br/>　　“苏大哥来了！”
<br/>　　苏莫刚一到达广场，广场上的众人便纷纷向他打招呼，个个客气有加。
<br/>　　目前，在整个神武学府之内，几乎没有人不认识苏莫了。
<br/>　　上次战界之内，与太史世家派系弟子一战，他是真正的名震神武学府。
<br/>　　现在，就算是那些修为高达武帝境八九重的高阶武帝，都不敢小觑苏莫了。
<br/>　　毕竟，苏莫所展露的天赋，实在是恐怖，不弱于那些圣子人物。
<br/>　　“各位有礼了！”苏莫向众人抱拳，微微拱手。
<br/>　　“哈哈！苏兄客气了，大家以后都是皇祖圣朝的人，以后多多亲近！”
<br/>　　“以后我们一起效忠圣子殿下！”
<br/>　　“是啊！效忠圣子殿下，干出一番大事业！”
<br/>　　众人纷纷开口，个个精神振奋，眸中斗志昂扬。
<br/>　　苏莫微笑点头，心中却是暗暗惊讶，这些人倒是对古天意死心塌地，估计都是完全被古天意收服了。
<br/>　　“诸位，这一次是什么历练？”苏莫沉声问道。
<br/>　　苏莫和众人一边闲聊一边等待，广场上聚集的弟子越来越多，不多时便达到了百余人。
<br/>　　闲聊之中，苏莫和众人也混了个脸熟。
<br/>　　通过闲聊，他得知这次所谓的历练，便是进入皇祖圣朝参战。
<br/>　　而带队之人，就是任云都。
<br/>　　皇祖圣朝所在的星辰，是一颗比古周星还要庞大的星辰，名为赤粱星。
<br/>　　赤粱星所在的位置距离古周星不远，此星与其它星辰不同，是一颗没有宗门势力的星辰，只有一个个古国和圣朝。
<br/>　　近年来赤粱星战乱不断，各种古国、圣朝、异族，每每兵戎相见。
<br/>　　因此，古天意时常组织人手，前去参战，获得军功。
<br/>　　有很多的人甚至因为获得大量军功，还是神武学府弟子的身份，便已经提前在皇祖圣朝谋的职位。
<br/>　　比如任云都，就在皇祖圣朝，有一个军中副尉的军职。
<br/>　　而这些弟子参战，不仅战利品全部归自己所有，还能得到圣朝的奖励，好处无穷。
<br/>　　当然，有得必有失，也有很多人因此丧命。
<br/>　　嗖！
<br/>　　破空声响起，任云都来到了广场。
<br/>　　“人都到齐了，我们出发！“任云都扫视了众人一眼，朗声喝道。
<br/>　　“是！”众人齐齐高喝。
<br/>　　苏莫见此有些无语，这哪里像是学府的弟子，分明就是一支小部队啊！
<br/>　　这些人不会是打仗打多了，都把自己当成皇祖圣朝的士兵了吧！
<br/>　　随即，任云都一马当先，率先离去，众人仅随其后。
<br/>　　一众人等，离开神武学府之后，便一路疾行，向最近的星空传送大阵之处飞去。
<br/>　　一路无话，众人花了月余的时间，辗转了三颗星辰，终于到达了赤粱星。
<br/>　　嗡嗡~~
<br/>　　阵法之光散去，苏莫等人出现在了一个无比辽阔的大殿之中。
<br/>　　“我们走！”
<br/>　　任云都招呼一声，率先向大殿外飞去，众人立刻跟上。
<br/>　　大殿之外，是一座城池，庞大无比，方圆数千里。
<br/>　　此城之大，比之天冥圣城都不遑多让，而且城中的建设更为恢弘大气，各种数百丈高的建筑，比比皆是。
<br/>　　“苏兄，这便是皇祖圣朝的圣皇城！”苏莫身边，一名高高瘦瘦的青年低声说道。
<br/>　　这名青年名叫谢彬，相貌清奇，修为和苏莫同阶，颇为健谈，一路上时常和苏莫闲谈。
<br/>　　“哦？居然是圣皇城？”苏莫闻言有些惊讶，这不就是皇祖圣朝的都城了吗？
