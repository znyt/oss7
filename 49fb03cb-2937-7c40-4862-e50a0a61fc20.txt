顾夜瑾有点明白陆寒霆为什么会喜欢上夏夕绾，这女孩冷静，聪慧，勇敢，耀眼，又不失20岁女孩的纤柔俏丽，这样的女孩很容易就激发男人的血性和征服欲。
<br/>
<br/>        等夏夕绾的俏影消失在了视线里，陆寒霆才收回了目光，他才不会告诉顾夜瑾，这世上有一类人，越坚硬的铠甲，越柔软的心房，她就是。
<br/>
<br/>        “行了，不跟你聊了，我要回去了。”陆寒霆道。
<br/>
<br/>        “回哪里，幽兰苑？”
<br/>
<br/>        “回公司。”
<br/>
<br/>        顾夜瑾觉得陆寒霆也变得神神秘秘的了，明明是担心夏夕绾在这场订婚party上被“群殴”，所以巴巴的赶了过来，但是看着夏夕绾全身而退了，他又自己回公司去了。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾先去了医院看林婶，林婶还没有苏醒，然后她回到幽兰苑就闷头睡觉了。
<br/>
<br/>        今天她虽然赢了一场仗，但是心里并没有半分的开心，因为她失去了很多，很多爱。
<br/>
<br/>        夏夕绾心里有些难过，想给叶翎发语音，但又怕她在剧组反而打扰了她，所以她趴在床上闭上眼。
<br/>
<br/>        可是怎么都睡不着。
<br/>
<br/>        这个时候突然有点怀念陆寒霆那温暖精硕的怀抱，如果他在的话，一定会将她搂在怀里，她枕着他强而有力的健臂，耳畔听着他搏动的心跳，整个人就像是漂浮的小船找到了休憩的港湾。
<br/>
<br/>        他的怀抱确实能给这天下所有的女人遮风挡雨。
<br/>
<br/>        等等，她在想什么？
<br/>
<br/>        夏夕绾迅速将脑袋里那张俊脸给甩掉了，他就是一个渣男，现在说不定就在公司里跟那个华总监深夜讨论文件呢。
<br/>
<br/>        她怎么突然开始迷恋他的怀抱了？
<br/>
<br/>        夏夕绾强迫自己进入了梦乡。
<br/>
<br/>        ……
<br/>
<br/>        翌日清晨，夏夕绾走出房门的时候就看见管家福伯在指挥佣人打扰卫生。
<br/>
<br/>        “这里擦干净一点，不要有一点灰尘。”
<br/>
<br/>        “厨房里的餐点都准备好了么，还有果汁。”
<br/>
<br/>        夏夕绾下了楼，“福伯，今天怎么大扫除了，是有什么特殊的日子还是有贵客要来？”
<br/>
<br/>        福伯的眼神有些躲躲闪闪的，“少…少奶奶，刚才少爷打电话回来，说是待会儿要带一位贵客回来，让我们准备接待。”
<br/>
<br/>        “贵客？”
<br/>
<br/>        “是的，少奶奶。”
<br/>
<br/>        夏夕绾不知道是什么贵客要来，陆寒霆也没有提前告诉她。
<br/>
<br/>        这时一辆加长版的商务豪车缓缓停在了草坪上，陆寒霆回来了。
<br/>
<br/>        福伯和女佣迅速迎了出去，“少爷回来了！”
<br/>
<br/>        夏夕绾站在门边抬眸看去，只见商务豪车的后车门被拉开了，陆寒霆那道高大英挺的身躯跃入视线。
<br/>
<br/>        他今天穿着手工版的白色衬衫黑西裤，精良的布料衬的他卓尔不凡，优雅倨贵。
<br/>
<br/>        这时另一道身影闯入了视线，是华容。
<br/>
<br/>        华容穿着红色连衣裙，那热辣凹凸的腰身被勾勒的一览无余，她化着精致的淡妆，烈焰红唇，格外的风情妩媚。
<br/>
<br/>        陆寒霆带着华容往这里走了过来。
<br/>
<br/>        夏夕绾僵在了原地，她没有想到陆寒霆会带一个女人回来。
