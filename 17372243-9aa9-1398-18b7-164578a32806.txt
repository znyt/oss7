　　在这二人的空戒之中，满满都是从九域之中所得到的无数宝物。
<br/>　　其中的许多东西，虽然对楚风眠这个级别的武者用处不多了，但是楚风眠也可以拿来，去可以给培养剑道门之用。
<br/>　　楚风眠要建立剑道门，自然不可能让楚风眠一个人去撑着，必须要发展起来。
<br/>　　就像是七大宗门，甚至是大罗天星域四大势力这样的存在，纵然是一尊在九劫古帝，甚至是半步天人境的强者陨落，也不至于会伤筋动骨。
<br/>　　这样的势力，才可以在万年岁月之中不断的传承下去。
<br/>　　“楚兄救了我徒儿一命，这也算是一个人情了，不过我从楚兄这里欠的多了，只能日后再还了。”
<br/>　　燕皇看了一眼楚风眠，眼神之中露出了几分笑容说道。
<br/>　　燕皇可以突破，踏入九劫古帝的最大原因，便是因为楚风眠通过青梦，交给燕皇的一枚原始圣丹。
<br/>　　这一枚原始圣丹，燕皇自然是清楚，肯定是楚风眠让青梦交给他的。
<br/>　　虽然说楚风眠是有着想要拉拢燕皇的心思，但现在燕皇还没有做什么，楚风眠就给了他一枚原始圣丹，也是天大的好处了。
<br/>　　这是一个大人情，也是让燕皇现在坚定的站在了楚风眠这一边，纵然是面对其他三大势力的古帝，他都没有顾忌，直接出手了。
<br/>　　当然最让燕皇下定决心的，还是因为楚风眠的实力。
<br/>　　楚风眠的实力提升的速度，超出了他的想象。
<br/>　　他当成见到楚风眠的时候，楚风眠的实力也就是足以跟八劫古帝抗衡，可现在短短不到一个月的时间。
<br/>　　楚风眠的实力就已经是突飞猛进，在面对九劫古帝的时候，都足以占据上风。
<br/>　　如此之快的实力提升，再给楚风眠一段时间，那未来楚风眠甚至可能会踏入半步天人境，甚至有极大的机会，踏入天人境。
<br/>　　在如今的时代之中，起码是有着数百年的时间，都没有听说过还有可以踏入天人境的强者了。
<br/>　　半步天人境级别的强者，的确还有，但是真正的天人境，却是就连一点点的消息都没有了。
<br/>　　“燕皇你这一次出手，你的燕州大陆，只怕是会危险无比啊。”
<br/>　　楚风眠看了一眼燕皇，突然开口道。
<br/>　　燕皇这一次，可是出手杀了葵海王的。
<br/>　　这个消息根本瞒不住，稍微推算一番，只怕就会被其他三大势力所知道。
<br/>　　倒时候这三大势力想找燕皇的麻烦不容易，但是燕皇所在的燕州大陆，也会遭到灭顶之灾。
<br/>　　楚风眠也是在提醒着燕皇。
<br/>　　“楚兄放心，我已经是派我那两名弟子，回到燕州大陆了。”
<br/>　　燕皇听到楚风眠的话，笑着开口道。
<br/>　　“听说楚兄在这九域之中，建立了剑道门，我燕州大陆的弟子，可否加入其中？”
<br/>　　“燕皇想要加入剑道门？”
<br/>　　楚风眠听到燕皇的话，心中一喜道。
<br/>　　“不错。”
<br/>　　燕皇点了点头道。
<br/>　　他现在已经是选择跟楚风眠站在一边了，也保持不了独立了。
<br/>　　剑道门现在是楚风眠新建的宗门，没有任何的底蕴，燕皇一旦是加入，便是绝对的高层，地位显赫。
<br/>　　日后如果等楚风眠真的踏入到了半步天人境，那个时候剑道门只怕也会水涨船高，不知道多少古帝，回来投靠。
<br/>　　燕皇现在前来投靠，也是有着先见之明，日后楚风眠称霸，他也是绝对的功臣。
<br/>　　“好，燕皇加入，我当然欢迎，只要是燕州大陆的弟子，加入剑道门之中，便就是我剑道门弟子。”
<br/>　　楚风眠立刻答应道。
<br/>　　燕皇跟燕州大陆，加入到剑道门之中，绝对是一件好事。
<br/>　　现在剑道门刚刚建立，虽然有楚风眠这样一个门主在，但是门下势力还是太薄弱了。
<br/>　　别说是跟九域之中的七大宗门相比，就算是九域之中的许多一线势力，许多古老的世家，都要比起现在的剑道门要强横。
<br/>　　燕州大陆，虽然只是六级大陆，但是也有着数十亿人口之多，其中的天才比起整个北域的天才加起来，都要多上不知道多少倍。
<br/>　　如此之多的天才，加入剑道门之中，在加上楚风眠留在剑道门之中的无数功法密集。
<br/>　　不用多久，剑道门就会兴起强盛，不亚于七大宗门，甚至是未来超越大罗天星域的四大势力。
<br/>　　“这一次在赤红大陆之中，必须要多得到几座大型宝藏才行。”
<br/>　　楚风眠脑海之中安排好了一切，脸上也露出了几分凝重。
<br/>　　剑道门的核心，是楚风眠，一旦是楚风眠陨落，剑道门便是会崩塌离析。

<br/>　　趁着赤红大陆之中无数宝藏开启，楚风眠必须要多得到其中几座大型宝藏，来提升他的实力才行。
<br/>　　从狮霸武祖的洞府之中得到的好处，也让楚风眠清晰的认识到了，这一座大型宝藏到底是蕴含多庞大的机缘。

<br/>　　楚风眠有着一件完整的极道仙兵，才是真的足以横扫一切。
<br/>　　“看来楚兄还不知道，现在赤红大陆，整个归墟天之中的强者，几乎都被一座巨形宝藏吸引了过去。”
<br/>　　楚风眠思考之际，燕皇再度开口了。
