　　　　涂小安掌控雷电属性，整个人都非常的亢奋，对于那些冲自己下跪的人，早就见怪不怪。
<br/>　　他现在的样子跟呈现的异象，比当初佘山更具备威慑力。
<br/>　　说他不是蛇王，他自己都快不信了。
<br/>　　一双蛇眸闪闪精光，掀开眼，看向宅院中那颗大槐树，眼睛浑然一眯。
<br/>　　夜空没有乌云密布，没有狂风呼啸，没有轰隆隆的雷声，一道刺耳的白光骤然而起，如一条粗大的银蛇，又像是一把雪白的利剑，劈向了不远处槐树的树根之上。
<br/>　　轰....
<br/>　　嚓...
<br/>　　大槐树骤然夭折，一股青烟冒出，渺渺炊烟的腾空而飘。
<br/>　　赵家人吓的三魂不见了七魄，他们院中的大槐树跟这栋古宅是一个年岁的，百年古树也，树木的枝梢交错着，伸展开来的繁盛的枝叶如碧绿的云，把蓝天遮了个严严实实，就算是来一个成年人，双手张开，也抱不过树身。
<br/>　　如此的百年老树，眨眼间就夭折了。
<br/>　　“蛇王生气了，是蛇王生气了”
<br/>　　赵老爷子惶恐之极，所谓平身不做亏心事，夜半不怕鬼敲门。
<br/>　　倏然，赵老爷子感觉到了一道锐利的目光投射到自己的身上，那是一双无法形容的蛇眸，像是可以洞穿一切。
<br/>　　蛇王在看他，像是要将他所有的秘密都给看穿。
<br/>　　“你该死”
<br/>　　一道诡异的声音骤然出现在了赵老爷子的脑海中，他蓦地，怔了一下，短促而痉挛地呼了一口气，浑身颤栗。
<br/>　　“嘭...嘭...嘭...”
<br/>　　“小人该死，小人该死”
<br/>　　赵老爷子发了疯似的在地面上磕头轻罪，一个个响头结结实实，没几下，地面就是一滩血迹。
<br/>　　赵家人呆滞的看着老爷子冲着蛇王边磕头边说自己有罪，样子凄惨悲凉，让人不禁鼻子一酸。
<br/>　　如此大的岁数，这样磕头，就算是一个年轻大小伙子都受不了。
<br/>　　可蛇王在前，无人敢动，敢乱说一个不字，生怕触犯了神灵。
<br/>　　蛇王高高在上，他们只能屏息仰望着，一点反抗心理都没有。
<br/>　　“嘭...嘭...嘭...”
<br/>　　赫然，赵家宅院变的安静如斯，鸦雀无声，唯有赵老爷子不断的磕头请罪，不敢有半点侥幸的心理。
<br/>　　不知磕了多少个，赵老爷子都快晕眩过去了，一位大孙子连忙冲上来制止，慌忙说：“爷爷，别磕了，蛇王走了”
<br/>　　“蛇王走了？”
<br/>　　赵老爷子一个愕然抬头，宅院的上空仿佛恢复了往日的平静，没有惊雷，没有蛇的影子，唯有黑沉沉的夜像要将他吞噬。
<br/>　　“不，蛇王还在，蛇王还在”
<br/>　　赵老爷子惶恐不安，呢喃自语，他总觉得那双洞察人心的蛇眸还烙印在自己的身上，如影随形。
<br/>　　那条家蛇被蛇王杀了，他一个老头子不用被逼着自尽，但赵老爷子心里依然还压着的巨石，一点都没落下。
<br/>　　反而犹如千斤之重。
<br/>　　这几十年，他并不好过，一直受到良心的谴责。
<br/>　　“快，扶我去前屋”
<br/>　　赵老爷子整个人变的有点疯癫，像是要去做一件半秒都不能耽误的事情。
<br/>　　“老爷你怎么了”
<br/>　　老管家将轮椅推过来，扶老爷子坐上去。
<br/>　　赵老爷子只是一个劲的催去前屋，很快一帮人随着老爷子的要求，带他来到了赵家前屋。
<br/>　　前屋内，赵老爷子抬头望向房梁之上，一片红布包裹着一块骨头状的东西，高高悬挂横梁上，光线暗淡，仿佛是女人眼角的怨泪，缭绕一种异样邪魅的气息。
<br/>　　“来人，将它取下来”
<br/>　　赵老爷子此话一出，所有人都呆若木鸡，管家更是诚惶诚恐的说：“老爷，你当年可交代过，上面那块红心木，谁都不能碰，碰了会影响赵家的风水，会引来血光之灾”
<br/>　　“快拿下来”
<br/>　　赵老爷子颤抖的手指着房梁，眼神充满复杂，额头上的血迹，将他那张满是皱纹的脸染色，显得狰狞可怖。
<br/>　　管家不敢在多说一句，只能命人将那块高高悬挂房梁整整有五十年历史的红心木给取了下来。
<br/>　　“拿来，拿来”
<br/>　　赵老爷子伸出双手，双手极度颤抖，仿佛是用尽全身力气才将这条红心木接到手中。
<br/>　　“欣兰，欣兰，是我对不起你，是我对不起”
<br/>　　赵老爷子双手捧着红心木，瞬间失控，嚎啕大哭，哭的让人不知所措。
<br/>　　“爷爷，爷爷，你怎么了，”
<br/>　　“老爷，你没事吧”
<br/>　　身边人被弄的不知所措，这一晚，事情已经够多了，好不容易一切都要过去，老爷子又来这一出。
<br/>　　“你们出去，全都出去”
<br/>　　老爷子朝着众人挥了挥手，大家你看看我，我看看你，只能顺从的退下，将老爷子一个人留在了屋中。
<br/>　　所有人都离开之后，赵老爷子小心翼翼的捧着红心木，就像捧着什么宝贝疙瘩，生怕弄坏了。
<br/>　　眼中满是畸恋之色。
<br/>　　“欣兰，五十年了，你还好吗，当年我只所以怎么做，只是想将你永远的留在我身边，留在我们赵家”
<br/>　　“但我错了，我对不起你，你生前我没有保护好你，你死后，我也没有厚葬你，我愧对你”
<br/>　　她何止是没有厚葬，而是将人快挫骨扬灰了。
<br/>　　赵老爷子撕心裂肺的说着，老脸哭着，那红心木里面是他生平最爱的女人。
<br/>　　记忆中那位淡雅无菊的女人，随着时间的流逝，让他几乎记不清她的模样，可却依然生生烙印在老爷子心底最深处。
<br/>　　他这辈子，唯一对不起的，就是她，还有她腹中没出世的孩子。
<br/>　　当家蛇来报复，当赵家开始死人，赵老爷子就知道，这段尘封的往事，要重新见光，那欠下的债，到了要还的时候。
<br/>　　忽然，红心木像是长了翅膀，猛然挣脱了他的双手，飞了出去。
<br/>　　“欣兰，欣兰”
<br/>　　赵老爷子直接从轮椅上滚了下来，看着悬浮在空中的红心木，满眼的惊骇。
<br/>　　“欣兰是你吗”
<br/>　　赵老爷子隐隐激动起来。
<br/>　　“你应该下去求她的原谅”
<br/>　　空荡荡的房内，当所有人都不在，唯有赵老爷子，可此时却出现一道诡异而冰冷的声音。
<br/>　　不知何时，赵老爷子的前方，昂首而立一条灿然生辉的金银蛇，那双蛇眸泛着冷然的望着他。
<br/>　　“蛇王大人”
<br/>　　赵老爷子顿时全身紧张得像一块石头，匍匐在地，诚惶诚恐。
