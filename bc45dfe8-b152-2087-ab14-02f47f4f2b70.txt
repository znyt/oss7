
<br/>　　<br/>　　    王惊蛰是有点呆萌，一根直肠子到底，但总归不是傻，他忽然发现当他和小草姑娘离开长安后，他一觉醒来就到了良水城外这件事，也许并不只是巧合？<br/>　　<br/>　　    鬼门开启，多道人影快速朝着良水河畔那处峭壁下赶去，最先抵达的是齐峥坤，常子敬和胡胡三人，他们直接从渔船上跳下来，到了峭壁下后，迅速攀登上去，来到那处被荆棘和杂草掩盖的洞口，一头就扎了进去，随即后面就是秋成子，姜冰啸和陈小瑜，跟着他们的脚步攀登而上，也钻了进去。<br/>　　<br/>　　    所谓的尝试，只要有人敢身先士卒，就有人会接着前仆后继了，一连几个人都毫不犹豫的冲了进去，也有人驻足在岸边迟疑起来，毕竟进入阴曹地府的话一个搞不好，出不来你可能就得永远留在阴间了。<br/>　　<br/>　　    王惊蛰和茅小草往过去的时候，也看见了陈三岁的身影，他几乎是一路疾驰而过，来到峭壁下方的时候，脚尖点了下地面身子就腾空而起，然后飞跃上去后一手扒着洞口身子就探了进去，没带一点犹豫，干脆利索。<br/>　　<br/>　　    “唰”这时，忽然有一道人影从王惊蛰身边一闪而过，对方在经过的时候回头留给了他一个意味深长的笑容：“我之前就觉得你有点古怪，现在看来还真是这么回事，我就料想今天晚上会碰见你，果然啊，待会里面见？”<br/>　　<br/>　　    王惊蛰笑着回了一句：“彼此，彼此了”<br/>　　<br/>　　    菜刀文伸手抓着荆棘藤蔓，快速的攀爬而上，扎进了洞口里，这时候岸边剩下的就是观望的人了，有人胆大包天，自然也有人谨小慎微“你身上还有伤······”茅小草回过头，叹了口气说道：“这个热闹其实你没必要跟着我一起凑的，阴间阴气太重了，你伤势未愈就进去的话是完全有可能加重的”<br/>　　<br/>　　    王惊蛰说道：“你自己进去我不放心”<br/>　　<br/>　　    茅小草懊恼的说道：“你跟着去了，我还得分心照顾你，你这不是给我添麻烦呢么？就不能让人省点心啊”<br/>　　<br/>　　    “你觉得这个理由能说服我么？”<br/>　　<br/>　　    茅小草：“·······”<br/>　　<br/>　　    自古以来，总有无数人猜测除了自己生活的阳间以外，到底有没有所谓的阴曹地府，阎王，判官，牛头马面，黑白无常还有所谓的地藏这些人物到底是不是杜撰出来的，这就像是一个流传千古的谜题一样，似乎从来都没有人真正的解开过。<br/>　　<br/>　　    但就像有句话说的那样，不在此山中，不知此山美。<br/>　　<br/>　　    常人猜测不出，但有些人却知道，在阳间之外确实也有阴间的，人死之后的亡魂总归需要一个安放之处的。<br/>　　<br/>　　    阴间是一个荒凉的世界，这里的色调似乎只有灰和黑，到处都充满了刺骨的阴风，一望无际贫瘠的土地上根本看不见其他的颜色。<br/>　　<br/>　　    当王惊蛰和茅小草进入到阴曹地府之后，首先感觉到的就是不适，这里的阴气太重了，喘息之间都会让人十分的不舒服，体内的三魂七魄都隐约有点不受控了，仿佛随时都要钻出来一样。<br/>　　<br/>　　    “你感觉怎么样了？”茅小草皱眉问道。<br/>　　<br/>　　    “还行，短时间内问题不大，阴曹地府啊果然不适合活人呆着，这就是规则，就像阴差去阳间拘魂也只是转一圈就走，而不敢耽搁太久一样，我们得在鬼门关上之前就得出去了，两个小时而已挺一挺还是可以的”王惊蛰眺望远处地平线，发现之前曾经进来的人大部分都消失，只能隐约看见远处有两三个人影。<br/>　　<br/>　　    “不是说，良水河下连接着的就是忘川河吗，河呢？”茅小草说道。<br/>　　<br/>　　    “传言而已，又没人真正的验证过，这一点明显就说不通啊，真要是连着忘川河的话，这处鬼门也就不难被发现，然后被疏漏了······”<br/>　　<br/>　　    人死以后，按照正常的程序首先就是要过鬼门关，这就是到了真正的阴曹地府了，然后要走上一段传说中的黄泉路，黄泉路上不回头，那意思是你一踏上黄泉路就是再世为人了，在黄泉路和阴间的冥府之间，有一条忘川河，河上奈何桥，喝了那碗孟婆汤，你就等着进入轮回投胎了。<br/>　　<br/>　　    还有另外一种可能，就是在阳间你犯过一些罪过，暂时不能入轮回就得在忘川河里飘着里，这里全都是不能投胎的孤魂野鬼，他们唯一能等待的就是自己身上的罪孽减轻，才可以投胎。<br/>　　<br/>　　    至于那些生前曾经罪孽滔天的人，就只能下十八层地狱了，被镇压不知道多少年了。<br/>　　<br/>　　    所以，人在做天在看，说的就是这么一回事了。<br/>　　<br/>　　    王惊蛰和茅小草一路前行，半个多小时后，忽闻水流还有阵阵鬼哭狼嚎的动静传来，体感也猛然间又凉了不少，两人愣了下，对视一眼后连忙往前快赶了几步，上了一座小山头上，从上往下眺望，发现不远处一条泛着阴气的河流里，挣扎着不知多少亡魂厉鬼。<br/>　　<br/>　　    忘川河，过了那条河，就是正儿八经的阴曹地府了，只是没有看见奈何桥和传说中的孟婆。<br/>　　<br/>　　    “他们也在那里·····”茅小草指着河边的两伙人，一伙是秋成子，姜冰啸和陈小瑜，离他们不远是孤身而立的陈三岁，两方人都站在河边眺望着。<br/>　　<br/>　　    “你来阴间，应该也是和他们一样吧？”王惊蛰瞥了那边一眼，忽然转头问道。<br/>　　<br/>　　    茅小草轻声“嗯”了一声，没有隐瞒，而是直白的说道：“你之前问我，是什么信念让我和你呆在良水城里，是爱心么还是助人为乐？当时我的回答是闲的······其实，并不是，我是为了那彼岸花来的，花开两朵，一朵彼岸，一朵在忘川河旁”
