　　在街边两侧的房顶上奔驰着，鬼修老人脸色有些阴沉，林九爆发出来的实力，让他感受到了一股威胁。

<br/>　　或者说，是不是应该联合其他的鬼修，一齐解决掉这个心腹大患。
<br/>　　心中正在思索着，鬼修老人身体稍稍一顿，低下了头，看见了正在流血的左手。
<br/>　　是手上的头颅，因为之前情形紧张，他拿的时候，是扣住了口腔倒提的，或许是动作过于激烈，居然不小心将手掌划破了。
<br/>　　“妈的，真晦气，这家伙死了都要咬我一口。”暗骂了一声，鬼修老人换了个姿势，揪着刘哔的头发继续前行着。
<br/>　　他可不敢过多停留，谁知道林九会不会从后面追上来。
<br/>　　但是没过多长时间，他就感觉就感觉有些不太对劲，像是身体生病了一般，一股强烈的虚弱感从体内涌来。
<br/>　　难道是刚才战斗的时候，林九趁他不注意下了毒？
<br/>　　可恶，这些所谓的正派弟子，手段也不比他们鬼修高尚多少！
<br/>　　暗骂一声，鬼修老人抹除了现场的痕迹，转身来到了附近一户人家的房屋中。
<br/>　　房中，一对夫妇正在熟睡，身边还躺着一个不大的孩童。
<br/>　　被开门的声音惊醒，床上赤裸着上身的男人坐起身来，看见提着一具尸首的鬼修老人，不由恐惧的张开了口。
<br/>　　但还来不及发出半点声音，鬼修老人身后的五颗头颅一动，扑向了床上三人。
<br/>　　只是一眨眼的功夫，床上多出了三具干枯的尸体，鬼修老人像是做了一件微不足道的事情一般，将刘哔的尸首扔下，拍了拍手，盘坐在床上。
<br/>　　他想抓紧时间，先搞明白身体究竟出了什么问题，这里并不安全，他要抓紧离开。
<br/>　　知道鬼修老人体内的丧尸病毒发作了，刘哔眼中精光一闪，指挥着自己的身体，将头颅重新安回了脖颈处。
<br/>　　盘坐在床上的鬼修老人猛地睁开双眼，一脸惊骇的看着刘哔，一时间，他有些没有反应过来。
<br/>　　“嗨，你好啊，老头。”虽然是将脑袋装在了脖子上面，但却没有任何愈合的迹象，刘哔无奈的苦笑一声，将脑袋重新拿了下来。
<br/>　　真是的，看来只能花积分修复了，但刚兑换完物品，他身上也没剩余的积分了。
<br/>　　“你……你没死？”有些慌乱的从床上站了起来，鬼修老人身体一阵摇晃，不得已只能用拐杖撑了一下床面，方才勉强站稳。
<br/>　　此时，他体内那股虚弱感更加剧烈了，连带着灵气都运转不起来，甚至他的意识，都开始有些模糊了。
<br/>　　看了一眼鬼修老人，刘哔撇了撇嘴，对方身体上，已经开始出现了一些丧尸化的特征，本来就鬼气森森的鬼修老头，变的更加不像人了。
<br/>　　或许是这老头的年龄问题，丧尸病毒的扩散，要迅速了许多，连带着变异的过程，都相对缩短了。
<br/>　　手中提着自己的头颅，怪异的视角让刘哔有些不适，但这并不影响他的动作。
<br/>　　一缕缕猩红色的丝线从刘哔手上浮现了出来，没有丝毫犹豫，刘哔大手一挥，猩红色的能量丝线缠上鬼修老人的脖子。
<br/>　　随手，刘哔手臂向上一抬，一颗头颅冲天而起，伴随着一股暗红色的血液，结束了鬼修老人的性命。
<br/>　　“叮~恭喜宿主击杀筑基期鬼修一名，获得积分一千，积分余额一千。”
<br/>　　系统的提示音从脑海中传来，轻呼了一口气，刘哔打开了商城面板，用八百积分修复了一下身体。
<br/>　　一阵乳白色的光芒，将刘哔的身体和头颅覆盖，十几分钟的功夫，他发现自己的视角恢复了正常。
<br/>　　有些不太适应的扭动了一下脖子，刘哔伸手在脖子上摸了摸，没摸到一点伤痕。
<br/>　　就连胸口的那个破洞，也随之一并修复了。
<br/>　　满意的点了点头，刘哔便离开了商城，对鬼修老人的尸体发动了吞噬功能。
<br/>　　之所以花费这么多的积分，主要是因为胸口的伤口太过于严重，而且完整度也受到了很大的损害，光是修复头颅的话，道是用不了这么多。
<br/>　　毕竟血肉气管什么的都没有太大丢失，系统所做的事情，只是将头颅和身子连在一起而已。
<br/>　　故此，就算头颅掉了对他产生的影响更大，但却是花费积分最好的一项。
<br/>　　先前不知道有这样的倒霉机制，刘哔心中有些后悔，早知道的话，之前那颗破损的心脏，就不要随手丢掉了。
<br/>　　心中有些不满的嘟囔着，他手上的动作却没有任何停顿，片刻的功夫，床上鬼修老人的尸体，就化为了一具白森森的骷髅。

<br/>　　听到吞噬完毕，刘哔方才抽出手来，意念沉入了技能面板，找到了新获得的这项技能。
<br/>　　名称：修罗场
<br/>　　级别：A级
<br/>　　分类：辅助技能
<br/>　　介绍：来源于《僵尸先生》位面，可消耗精神力，指定一处区域，抽取该区域尸体内的能量自己使用，持续时间以及范围由宿主等级决定。

<br/>　　只是让他有些疑惑的是，鬼修的那具骷髅上，却不存在半点能量，或许，是因为自己的吞噬效果？
<br/>　　舔了舔口中的獠牙，刘哔在鬼修老头身上摸索一番，转身离开了院落。
