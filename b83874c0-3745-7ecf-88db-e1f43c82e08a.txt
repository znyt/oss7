　　第220章
<br/>　　朱雀大街被清空，家家关门闭户，街道两边林立着持戈的皇城禁卫，在高处也有手持强弓劲弩的兵丁。
<br/>　　今天是明帝御驾游街的日子，一来是为了迎接帝国战神杨龙停，二来是今天武科开考，明帝需要在天下举子面前露个面。
<br/>　　御撵从皇城出发，净水泼街，黄土垫道，为了今天，长安城从昨天就开始准备。
<br/>　　禁卫军在前，明帝的御撵在后，其后是皇亲贵胄，在其后是文武官员，程大雷在考官的队伍中，他后面是举子中的优秀人物。
<br/>　　程大雷今天不拿刀，不背斧，而是佩了一柄剑，匹夫剑。从李婉儿口中，他已得知这把剑的来历，于是悄悄将剑柄上的『陆』字抹去，在长安城这种地方，若是被人认出这柄剑的来历，也就麻烦了。
<br/>　　他行走在队伍中，算是想起前生被各种大会支配的恐惧，今天天气又忒热了些，衣服里襟已经被汗水打湿。
<br/>　　而这还仅仅是个开始，明帝将会出城迎接杨龙停，之后还要祭祖，祭天，昭告天下：朕一心为民，愿老天降下几个人才来保护帝国呐。
<br/>　　程大雷已经在一天前拿到今天的兵力布防，即便是他，也心惊今日的防御，总共出动的兵力在五千人，而这五千人还是优先军饷，优先装备的禁卫军。
<br/>　　凭自己经营一年的江湖草莽，真的能在重兵保护下刺杀杨龙停么。
<br/>　　城门外，是程大雷动手的时候，拿到兵力布防图后，程大雷已经做了安排。到时候外围的徐神机以焰火为号，埋伏的人手制造混乱，而必杀一剑将由程大雷亲自刺出。
<br/>　　可程大雷此刻心里觉得有点悬。
<br/>　　“牛校尉……”
<br/>　　正琢磨着今天的行动计划，忽然耳边一个声音响起，程大雷心中一警，手已摁在剑柄上。
<br/>　　百里胜吓了一跳：“牛校尉，你的眼神……好吓人呐。”
<br/>　　原来是他，程大雷还以为被发现了，他将剑轻轻松开，擦擦额头的汗珠道：“紧张，紧张。”
<br/>　　“是呐，我也紧张。”百里胜道：“将要见到那战神大人，如何能不紧张呢。”
<br/>　　程大雷一听到『战神』两个字就恶心，的确，杨龙停算是一个合格的军人，但战神……那仗是老子打的好不好！
<br/>　　“不过战神大人可是相府的人喔。”
<br/>　　百里胜道了一句，言外之意就是：战神是相府的人，你身为军人，对他有些崇拜也正常，可是不要站错了阵营。
<br/>　　程大雷现在心里正想着如何杀人，杀机满满，哪有时间听百里胜闲扯。此刻他心中忽然一动，或许能利用将军府和相府的矛盾，扳倒杨龙停。
<br/>　　“我特意去制了一套新盔甲，今日能见到战神大人，一定要给大人留个好印象。”
<br/>　　百里胜兴奋道，虽然两家阵营不同，但身为一个军人，他自然崇拜英雄。而程大雷也发现，参与今天的举子，每个人也都打扮得焕然一新。
<br/>　　“牛校尉在西北也应该听说过战神大人的名声吧，我看你今天也穿了新衣服，一定也为这天特意准备了。”
<br/>　　可不是特意准备了，准备的场面之大，怕是会吓你一跳。
<br/>　　出得城门，地界便开阔许多，禁卫军持戈立在官道两侧，在他们外围挤着许多围观的民众。
<br/>　　杨龙停候在城门外，此时杨龙停带着几员大将快步过来，躬身拜倒。
<br/>　　“臣杨龙停见过陛下，陛下万岁万岁万岁。”
<br/>　　“我帝国战神何必如此拘礼，快起来说话。”明帝没有走出御撵，隔着帘子与杨龙停说话。
<br/>　　“臣不过一介武夫，只知精忠报国，为陛下分忧，帝国战神四个字实在敢当。”
<br/>　　程大雷的手已经握紧剑柄，一年不见，杨龙停活得很好，当然程大雷也活得很好。可是当杨龙停活得很好时，程大雷就觉得生活没那么好了。
<br/>　　隔着人群，徐神机以眼神冲他示意，程大雷轻轻摇了摇头。
<br/>　　稍安勿躁，现在时机未到，今日行动的难度比预估更加困难。
<br/>　　“坚决不与戎族和谈！”
<br/>　　“不和亲，不称臣，与戎族血战到底！”
<br/>　　正这个时候，一队书生打扮的人站在队伍外面高声喊着，他们约有百十人众，每个人都慷慨激昂，神情亢奋。
<br/>　　帝国和戎族的和谈早已经开始，最近戎族的谈判使节将来到长安城，商议合约的具体细节。关于这一点，民间早有怨言，我泱泱大国，怎么能和戎族蛮夷和谈呢。
<br/>　　而今天这些儒生，是程大雷怂恿的的。
<br/>　　他手中握着剑，等待合适的机会。
<br/>　　可是，令程大雷没有想到的是，明帝连一句话都没有说，杨龙停也弯着腰面带笑意，其他文武官员此刻耳朵都像聋了一般，自动把士子的声音屏蔽了。
<br/>　　而一些禁卫军已经开始抓人，一方其乐融融，一方暴力血腥，两种截然不同的景象却隔着一道人墙同时发生。
<br/>　　特么的！
<br/>　　程大雷冲徐神机摇摇头，今天的行动计划还是取消吧，自己低估了朝廷的脸皮厚度。
<br/>　　哒哒哒！
<br/>　　正在此时，两匹快马向这里奔来，马上的人脸上蒙着黑布，手中持着长枪，杀机赫赫冲向御撵。
<br/>　　谁动的手，谁动的手！
<br/>　　程大雷有些失态，事情出离自己控制了，没有自己的命令，是谁动手。今天这幅场面，动手不是找死么。
<br/>　　禁卫军立刻护在御撵前，有兵丁却拦截这两个刺客，可二人骑术惊人，枪法犀利，竟然冲进队伍中央。
<br/>　　“昏君，受死吧！”
<br/>　　程大雷一惊，这二人要杀的不是杨龙停，而是——帝国天子。
<br/>　　人群慌慌乱乱，队伍出现了乱象，纷纷向外逃散，程大雷逆流而上，摁着剑向前走。
<br/>　　“牛校尉，牛校尉！”百里胜道：“莫往前去，小心危险呐。”
<br/>　　程大雷摇摇头，一道锋利的笑容挂在他唇边。
<br/>　　乱，就是机会。
<br/>　　马上两个刺客行动很快，已经逼到御撵前，两柄枪同时刺出。
<br/>　　忽然，一道拂尘扫过，缠住一柄枪，另一柄被他手掌拍开。
<br/>　　刘阿吉。
<br/>　　程大雷一激灵：太监中果然有高手啊。
<br/>
