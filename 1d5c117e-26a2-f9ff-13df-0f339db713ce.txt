　　第634章
<br/>　　程大雷琢磨着这次收获的系统奖励。使用次数只有一次，限制田地面积是一千顷，至于加快作物生长的进度能加快多久，程大雷目前还不知道。只有使用过后才能明白具体效果，但估计不能达到朝种夕收。否则的话，就太过匪夷所思。
<br/>　　程大雷忽然觉得系统变得大方起来。比如这次的系统奖励就十分有用，等河渠挖好，种子入土，自己便可施展技能。有这千顷之田的丰收，朔方城就可以迈过第一道关。
<br/>　　万事开头难，第一关度过之后，之后的事情便会变得顺利。
<br/>　　程大雷不仅奇怪，莫非是系统转性了，还是说自己已经脱非入欧。
<br/>　　目前手中还握着一次抽奖次数，此时不用更待何时。
<br/>　　那扭蛋机一样的抽奖机器再次在脑海中出现，不多时程大雷便收到一条系统提示。
<br/>　　嘟，恭喜获得绝世谋士杨国忠。
<br/>　　程大雷手托额头，刚才的好心情荡然无存。杨国忠，乃是大唐宰相，能力方面是绝对不用怀疑的，可以相一国的人，治理小小一座城实在是轻而易举的事情。不过，此人也是个反面角色呐，果然那暴恶属性仍旧在发挥作用，系统仍然没有转性。
<br/>　　嘟，暴恶属性生效，随机出世绝世山贼卢俊义。
<br/>　　程大雷一阵无语，梁山泊第二把交椅，号称枪棒无双，这妥妥是正面角色的人设呐。
<br/>　　却也不知他会落到谁家，话说史文恭还在自己手中，假若二人撞在一起，不知会是如何情形。
<br/>　　不过话又说回来，那杨国忠的妹妹便是四大美女之一，那位祸国殃民的红颜祸水。却也不知道，她这次会不会一起出世。
<br/>　　程大雷心中忽有些期待起来，盼着杨国忠早些到来，却也不知他会在什么时候出现。
<br/>　　……
<br/>　　夜，狼山关灯火通明，与宋伯康交过几战之后，对方已经选择避而不战。而狼山关已经岌岌可危，距离断粮的最后关头越来越近。百里胜派人发动过一次夜袭，但也没能够成功突围，左臂还中了一箭。
<br/>　　“侯爷，不若我们向草原转移。那里防守的兵力少一些，我们转移草原，可以暂时度过眼前的危机。”孙鸣之道。
<br/>　　“向草原转移等于将狼山关的险地拱手让人，况进入草原后，我们的粮草依旧接济不上。到时候只能成为无家之犬，任敌人宰杀。”百里胜缓缓道。
<br/>　　“侯爷，那你说该怎么办，咱们的粮草最多只能支撑三天。”
<br/>　　“把粮食发下去，每人携带三天干粮。”百里胜抬起头道，开口缓缓道：“今夜决战。”
<br/>　　所有人被百里胜散发的杀气震慑住，此刻他没半点京城风流侯爷的样子，铁青着一张脸，面目狰狞如亡命徒。
<br/>　　狼山关灯火通明，三万将士列阵完毕，每人携带三天干粮，以做突围用。
<br/>　　百里胜一身戎装，骑在乌骓马上，他目光缓缓扫过众人，开口道：
<br/>　　“诸位弟兄，今日我们已是绝地，帝国之大，却已无我等兄弟容身之处。我们能做的，也只有舍了这条性命不要。今夜，会有很多兄弟死，包括我也会死。但如果有兄弟活着，就杀到长安城，替死去的弟兄讨个公道。”
<br/>　　“杀到长安，讨还公道。”
<br/>　　三万人齐声怒道。
<br/>　　百里胜点点头，口中吐出两个字：“出发。”
<br/>　　赵子龙这个时候也没有歇息，他虽然装病不出战，却密切关注这一战的发展。某些事是讲天赋的，赵子龙就是一个有天赋的人，他明白战场之上没有绝对的胜，也没有绝对的败，瞬息万变的战场影响胜负的东西太多了。
<br/>　　忽然，营外传来喊杀声，一名小校闯进大帐。
<br/>　　“报，狼山关袭营，这次似乎是全军出动。”
<br/>　　“算着也该来了。”赵子龙点点头：“传令下去，所有人上马，遵我号令行事，无我命令不可与敌人接触。”
<br/>　　“是。”
<br/>　　龙字军的反应算是快的，等他们集结完毕，撤到大军的侧翼时。宋伯康才着急忙慌的穿好衣服。
<br/>　　这次不同上一次的夜袭，百里胜是抱着决一死战的心思，因为大家都知道，如果再冲不出去，面前也就只剩下死路一条。
<br/>　　在这种形势下，自然人人拼命，各个争先，抱着一腔愤怒与敌人厮杀。宋伯康的大军没料到对方有如此决心，应对之上便显得慌张一些。
<br/>　　宋伯康举目四望，所见都是一团乱糟糟的厮杀。倒也不是完全的乱象，又一支军队就保持完整的阵型。
<br/>　　可惜，是赵子龙率领的龙字军，他们一早就远离战场，立在高坡上观战。
<br/>　　特么地，还真是看戏来的，出人不出力啊。
<br/>　　宋伯康气得怒火翻腾，但此时此刻也没时间想这些。他急忙下令组织军队，阻止百里胜突围。
<br/>　　“将军，假若百里胜突围成功，咱们要不要出手拦一拦？”手下副将问道。
<br/>　　“突围？”赵子龙骑在马上摇摇头：“我看难喔。”
<br/>　　夜里发生的乱战，互相厮杀在一起，但宋伯康出动的兵力是百里胜的数倍。百里胜虽然一腔破釜沉舟，可无奈敌众我寡，他挣脱不开这牢笼。
<br/>　　“凉州兵势大，虽然应对狼狈，但根基未乱。就算暂时失利，也能稳住局势。百里胜想要突围怕是九死一生，就算有人可以突出去，也不过是一帮残兵败将而已，做不得什么大事。”赵子龙一边观战一边道。
<br/>　　这时候，从东南方向突然杀过来一个人，他骑着一匹高头大马，手中一杆金枪，斜刺里冲杀过来。
<br/>　　几乎他刚出现，赵子龙的视线就落在他身上，他吸引了赵子龙所有注意力。
<br/>　　一匹快马闯过人群，马上人冲杀到百里胜面前，挥枪将四周兵丁杀退。
<br/>　　“百里侯爷，某家特来投靠。”
<br/>　　见此人如此武勇，百里胜一开始吓了一跳。却也没想到此人竟是来投靠自己的，不过这人是有病吧，现在自己这状况竟还有人投靠。
<br/>　　“那个……你是？”
<br/>　　“某家卢俊义，江湖上的弟兄都唤我……”他一枪横扫，周围一片惨叫：“玉麒麟。”
<br/>(本章完)
