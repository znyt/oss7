　　“首领，要进攻么？”良久的沉默之后，大祚荣身边白山部族长终于忍耐不住上前问道。
<br/>　　“这城早晚都是我们的，但现在却不是取它的时候，且等等吧。”大祚荣淡然说道，明显是另有的打算。
<br/>　　“为什么？这城池明明没有多少守军，加上里面又全都是些没用的书生，我们大可一鼓而下。”白山部族长有些不解的问道。他实在搞不清楚，为什么大祚荣明明想要进攻大唐都想疯了，但看到第一座城池却又不想进攻。
<br/>　　“大唐有句话，叫做不计一城一池之得失，我认为这句话并没有错。”大祚荣满是沧桑的脸上带着一抹谁也说不清楚含意的笑：“所以我们不争他们的城池，我们找他们的主力决一死战。”
<br/>　　“什，什么？决战？”白山部族长有些惊讶，他怎么也没想到大祚荣会如此有豪气。
<br/>　　“对，决战，只要我们把那些唐人从辽东赶走，这里的城池最后还不都是我们的。”大祚荣淡淡看了白山部族长一眼说道。
<br/>　　“可是……”
<br/>　　“没有什么可是的，我们有十万铁骑，而唐人只有三万骑兵，至于另外的那两万步军，不值一提。”大祚荣摆手指断白山部族长的话。
<br/>　　作为一个强者，首先要有的就是自信，大祚荣认为，以他可以一统靺鞨七部的能力，与对面唐军守将李绩的能力应该不相上下，这就是他敢于出兵的理由。
<br/>　　另外他曾经听那些逃入深山的高句丽人说过，唐人的武器装备十分犀利，而这正是靺鞨人所不具备的。所以在出征之前他便定下计划，这一次出征绝不与唐军争夺城池，只要把唐军赶出辽东便好。
<br/>　　至于说如何把将唐军赶出辽东，那就是与李绩交手之后再考虑的事情了。
<br/>　　……
<br/>　　看到这里有朋友会问，那靺鞨既然当初能够与高句丽达成合解，为什么现在却要与大唐作对呢？难道他们认为大唐的实力会不如高句丽？
<br/>　　其实不然，其实上大唐刚刚征服高句丽的时候，靺鞨人并没有想与大唐争锋，抱着得过且过的想法，他们最开始的打算是学习契丹那样归附大唐。
<br/>　　可事与愿违，那些将土地置换到辽东的世家贵族们实在太过贪婪了，他们并不满足于现有的土地，所以便一直在边境侵蚀着靺鞨人的领地，甚至一度将他们放牧的草场全部占为已有。
<br/>　　再有就是那些河流，为了扩大种植面积，河流也成了那些世家的必争之地，这使得一些以捕鱼为生的靺鞨人没了生计。
<br/>　　正是因为这样的情况，导致了靺鞨人不得不站出来，将那些入侵他们领地的唐人从自己的土地上赶走。
<br/>　　这也是为什么大祚荣会一呼百应，能够统一起靺鞨七部与大唐为敌的原因。
<br/>　　毕竟谁都不想自己的地盘被别人占据不是。
<br/>　　……
<br/>　　书归正传，靺鞨人入侵的消息很快便以飞鸽传书的方式送回了长安，送到了李二和李承乾的手中。
<br/>　　看着那潦草的字迹，李承乾不由得摇头苦笑。
<br/>　　这还真是想什么来什么，前几天才跟那个钦陵说让他去辽东处理靺鞨人的事情，今天就收到靺鞨人入侵边境的消息，也不知道自己是不是传说中的乌鸦嘴。
<br/>　　“你笑什么，这不是正合了你的意，别以为朕不知道你一直在惦记着东北的那块地。”李二陛下丢下手中的情报，没好气的说道。
<br/>　　“父皇，看您说的，儿臣这不也是为了将来考虑么。”李承乾讪讪的笑了笑。
<br/>　　“为将来考虑？考虑什么？”老头子问道。
<br/>　　“父皇，这次靺鞨人主动出来，我们完全可以将计就计，一边引他们深入，另一边派人出云州绕到他们后面直插黑水河，断其后路，然后……”说到这里，李承乾站身来走到墙上挂着的简易世界地图前，指着一个位置说道：“只要占领了这里，我们就可以从两面遏制倭岛，到时候那块飞地自然也就是我们的了。“
<br/>　　老头子看着李承乾兴奋的样子，也有些无语，他是真搞不懂什么这混小子一直对倭岛那么感兴趣，总是对其念念不忘。
<br/>　　不就是点银子和金子么，抓一些倭人去挖也就是了。
<br/>　　而且老李这段时间也听说了，倭岛那破地方总是地震，三天一小震，五天一大震，根本就不是人呆的地方，可以说要不要那破地方根本没多大意义。
<br/>　　李承乾也看出老头子似乎对此并不感兴趣，可奈何有些话他是真的没办法说，关于倭国的一切始终是他心里的一个疙瘩，如果不彻底把这个岛国给弄残了，他是真的不放心。
<br/>　　想到这里，李承乾觉得自己必须编一个合理的理由让老头子放心，只有这样才能不引起老头子的疑。
<br/>　　于是便清了清嗓子继续说道：“父皇您看，这里是倭国，这里是琉球，如果我们在这里划一条线，那么就明显可以看出来，大唐是被封闭在一个环里面。”
<br/>　　“那又如何呢？”老头子看着李承乾指的几个位置，皱了皱眉头。
<br/>　　“这就是说，如果有一天我们衰弱了，或者不在再强大了，很有可能被敌人封锁住海上的出入口，这样一来我们就会受制于人。”李承乾说道。
<br/>　　“你觉得大唐会和以前的那些皇朝一样？”老头子眉头皱的更紧了。
<br/>　　“父皇，智者千虑必有一失，愚者千虑必有一得，这个世界上没有什么事情是绝对的，后世子孙会如何我们完全料想不到，所以不得不防啊！”李承乾知道老头子并不想听自己说一些长他人志气灭自己威风的话。
<br/>　　但很多事情就摆在那里，不是你不去想它就不会发生的。
<br/>　　这个世界没有亘古永存在皇朝，这一点在后世已经无数次验证过了，就算是再开明白君主他也只能对自己这一生负责，儿孙的事情完全不是他可以决定的。
<br/>　　所以李承乾决定赌一把，把这个谁都不敢对老头子说起的事情挑明了说上一回，看看老头子到底是个什么反应，大不了就是被打一顿板子，拼了！
