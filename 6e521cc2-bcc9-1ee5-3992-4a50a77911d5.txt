　　“但是鸟兽昆虫，不是人形吧？”楚涵淡淡的说出这句话。
<br/>　　“废话，当然不算。”高少辉没能反应过来，随口回答。
<br/>　　“这些飞行丧尸太多，我的空军应付起来恐怕有些困难。”楚涵说着开场白，而后忽然话锋一转：“所以你来帮忙，就用这天照3000，这玩意儿应该有攻击功能才对。”
<br/>　　高少辉猛地一惊，乍然扭头看着楚涵：“你特么在逗我？！”
<br/>　　“飞行丧尸不算人形物种，不是么？”楚涵笑看着他，随后不知道从哪取出了两个能源，摆在了高少辉眼前：“这是能源，够不够？”
<br/>　　高少辉傻了眼，瞪着楚涵仿佛在看怪物：“你到底哪来的这么多能源？”
<br/>　　“我就问你能不能杀。”楚涵并不回答他的问题。
<br/>　　高少辉想了想：“理论上可以，而且如果用上天照3000作为中间介质，也不算我直接出手？”

<br/>　　那里南署基地已经出现在他们面前，那两圈一高一低的内外墙，也呈现而出。
<br/>　　同时在高墙上的路冰泽，也猛然睁开双目，眼底有着无尽的惊恐，大声开口道：“快！千万尸潮！二十分钟！飞行丧尸……数不清！”
<br/>　　说罢，路冰泽就猛然吐出一口血，显然是过度使用能力后精疲力竭。
<br/>　　徐锋第一时间朝着后方指挥台打出手势，接收到的蒋天庆瞳孔一缩，当即汇报。
<br/>　　屋内的几名参谋部成员全体傻了眼，他们大概能猜到尸潮很多，但是千万尸潮，几千万？
<br/>　　这比整个南都城的丧尸还要多！
<br/>　　可南署基地的面积才多大，恐怕连南都城的百分之一都没有，如何抵挡这千万尸潮？
<br/>　　何峰是唯一一个没有任何停顿的人，第一时间就打开了扬声器，顿时响亮的声音传递到了基地内的各个角落：“所有人准备战斗，空军出动，陈少爷准备击爆引诱剂分流！”
<br/>　　哗！
<br/>　　整个南署基地顿时活了过来，之前的一片安静好似只是错觉，所有人都第一时间放下了手中事物，以最快的速度进入战斗状态里。
<br/>　　迷宫地带的五万人战斗时间相对靠后，他们有条不紊的进行着各处站位调整。
<br/>　　两墙之间的一万五千人，则是紧张又刺激，城墙的通道没有门，他们甚至能够轻而易举的看到外面的场景。
<br/>　　基地内一百架战斗直升机腾空而起，朝着四面八方分散而去，每一架直升机内的军法处成员和射击队成员，都相对无言也表情严峻到了极致。
<br/>　　陈少爷所在的直升机，更是已经开始朝着南面飞行，前方的有很多个定点，都早已被放置好了丧尸引诱剂的一半部分，陈少爷所要做的就是在尸潮踏入第一个区域的时候，就依照顺序逐一击爆引诱剂，让这些丧尸分散到四面城墙的方向。
<br/>　　千万丧尸一下子涌·向同一面城墙的后果，就是整个基地全部被淹没，所以分流，是大战开始前的最重要一环，如有差错这场战役就整个完蛋。
<br/>　　所有人都紧张无比的待在自己的岗位上，等待着尸潮来临，而这时候感知力强的人，也已经感受到了大地在震动。
<br/>　　轰！
<br/>　　轰！！
<br/>　　轰！！！
<br/>　　一次比一次剧烈，一次比一次亮相，每一次的震动强度甚至在成倍增长。
<br/>　　仅仅三次震动之后，待在两墙之间的那些人，就全体感受到了这股震动，身旁还有细小的尘埃被震得浮起，没入他们的口鼻。
<br/>　　眼力好的人，则是已经能看到在基地外的最南方，已经有一条黑色的横线在越来越近，横线很长很宽，长到左右的视野宽度都被完全占据！
<br/>　　“来了。”徐锋开口，所有杀羽战队成员准备就绪。
<br/>　　“开始。”陈少爷开口，负责击爆引诱剂的直升机已经进入了第一个区域。
<br/>　　而就在此时，高空的一架黑色飞行物却眨眼即逝的从陈少爷所在的直升机旁擦肩而过，快的好似是错觉。
<br/>　　驾驶直升机的飞行员吓了一大跳，双手一抖差点没扶稳：“刚刚那是什么？飞行丧尸！？”
<br/>　　陈少爷也同样大惊，但很快就反应了过来，大喜道：“我们继续，那是我老大！”
<br/>　　陈少爷的老大，当然就是楚涵。
<br/>　　这话一出后，那名飞行员反而更震惊，不可思议道：“我靠长官的座驾这么帅？”
<br/>　　两人从基地起飞到此处一句话都没说，机舱内沉默就这么被天照3000的飞过而打破，而对话的语气更是显露出了两人此时难以言喻的喜悦。
<br/>　　楚涵果然来了！

<br/>　　“一号机发现天照3000，目标十二点方向，是否击毙？”
<br/>　　“三号机发现天照3000，目标九点方向，是否打招呼？”
<br/>　　“十二号机发现天照3000，目标两点钟方向，是否过去撒个娇？”

<br/>　　天照3000稳稳的停在了指挥台的空地上，在所有人激动的心情中，楚涵手握修罗战斧从机舱内踏出。
