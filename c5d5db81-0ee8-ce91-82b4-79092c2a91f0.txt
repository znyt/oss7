　　第521章
<br/>　　了不得就了不得……你喔那一声算几个意思。
<br/>　　秦蛮的脸色变得不太好看。他出身就是个猎户，有一身蛮力，一股脑冲锋是绝对没问题的，但让他领兵打仗，运筹帷幄委实有些强人所难。
<br/>　　也就是程大雷不断补课，又用系统帮他升到绝世，他的脑袋才勉强够用。
<br/>　　但还是和关鱼这种人有很大差距，关鱼未来可是传说潜质的，但秦蛮现在怕已经满级了。
<br/>　　“啧啧，蛮好，蛮好。”
<br/>　　关鱼负手在身后，脸上带着淡淡的微笑。看着面前的灰烬，与其说是欣赏火场，倒不如说是欣赏秦蛮脸上的表情。
<br/>　　“这场大火，扑灭怕是需要不少时间吧，有没有人不小心烧到。”关鱼。
<br/>　　“……”秦蛮，的确有三五个人灭火时伤到。
<br/>　　“重新盖起来可需要一些时间，这笔钱不能让大当家出吧。”关鱼。
<br/>　　“放心，蛮字军会重新把院子盖起来。”秦蛮虎着一张脸。
<br/>　　“当然，当然，你们蛮字军盖房是很内行的么。”关鱼由衷赞美。
<br/>　　“……”秦蛮脸上越来难看，什么意思，盖房子内行，打仗就不内行了么。呸，谁说我们盖房子内行的。
<br/>　　“那老尼姑死了么。死了啊，嗳，应该是生擒啊，万一她有什么同伙。”关鱼看了秦蛮一眼：“当然，秦将军能杀死就算不错了。”
<br/>　　“……”秦蛮险些暴走。
<br/>　　关鱼神清气爽，有些槽当真是不吐不快呐。不过，也不能将秦蛮刺激得太厉害了，要是动起手来，自己还真不是他的对手。
<br/>　　程大雷倒是不关心关鱼和秦蛮的斗嘴，当然，主要是关鱼在怼秦蛮。这种怼也没有太大关系，大家都是从青牛山来的，秦蛮是个猎户，关鱼是个逃犯，谁瞧不起谁呀。
<br/>　　他颠颠的跑回书房，小心翼翼将门关起来，免得有人会闯进来。
<br/>　　之所以如此小心，整个人像做贼一般，是因为刚才有件不了得的事情发生。
<br/>　　这白衣老尼肯定属于boss级别了，杀死一个薛丁山奖励就不小，更何况是一个绝世boss。
<br/>　　在刚才击杀白衣老尼后，程大雷获得了系统奖励。只不过这次奖励的东西，令程大雷有些懵。
<br/>　　系统的奖励是一张卡牌。
<br/>　　一张人物卡牌。
<br/>　　姓名：鱼洞真（隐世不出的绝世高手）
<br/>　　技能：洞玄功
<br/>　　隐藏属性：无
<br/>　　鱼洞真这个名字，程大雷没有听过。而现在程大雷也不是没见过世面的人，一个绝世高手也不会令他激动得上蹿下跳。问题的关键在于，卡牌上的人物模样，和那白衣老尼一模一样。
<br/>　　所以程大雷才会一脸懵。
<br/>　　他大概琢磨明白是怎么会事：很可能，是自己击杀白衣老尼后，便获得了白衣老尼的人物卡牌。
<br/>　　仔细想了想，以系统的尿性，很有可能这么搞喔。
<br/>　　但问题同样也来了，为何自己击杀聂隐娘和薛丁山时，没有出现这样的奖励。难道说这项奖励是随机的，或者说……他们死的不够惨。
<br/>　　程大雷琢磨不明白，但无论如何，把这张卡牌用了不就知道了么。
<br/>　　也是艺高人胆大，就算真出了幺蛾子，程大雷自觉也顶得住。如果是以前，程大雷却是不会这么冒险的。
<br/>　　他就在书房内使用了卡牌，就见眼前白光一闪，一道人影出现在程大雷面前。
<br/>　　程大雷倒是吓了一跳，系统越来越瞎搞了。这次连人设也不给，就在自己眼前冒出来了。
<br/>　　待看清这人影时，程大雷更加忐忑。关键是这相貌和白衣老尼太像了，什么叫像，根本就特么是一个人嘛。
<br/>　　唯一的区别是，白衣老尼的打扮是一身白，鱼洞真的打扮是一身黑，手中托着黑色拂尘，袖口还纹着蛤蟆。
<br/>　　程大雷还真担心，这是白衣老尼还魂要杀自己呢，想想她死得那么惨，也并不是没有这种可能啊。
<br/>　　不过，仔细看看两人还是有区别的，白衣老尼像清冷的冰棱，说不好什么时候会弄死自己。而这黑衣老尼，明显有些木纳，双目空洞宛若瞎子一般。
<br/>　　程大雷在她面前挥挥手，见她没有什么反应。他胆子大起来，挺直了腰板。
<br/>　　“知道我是谁么？”
<br/>　　“大当家。”鱼洞真的声音干冷。
<br/>　　“知道自己是干什么的嘛？”
<br/>　　“听大当家的。”
<br/>　　“很好。”
<br/>　　程大雷松了口气，大摇大摆在炕上坐了下来。很明显，这黑衣老尼没有脑子，只知道对自己的话言听计从。
<br/>　　但没有脑子，总是比脑子有问题好吧，自己说什么她就做什么，自己最需要的，不就是这种人么。
<br/>　　只是不知道动手的时候，会不会仍旧没有脑子，假若是那样，问题就大了。
<br/>　　咚咚咚……
<br/>　　房外响起剧烈的敲门声，响起秦蛮粗大的嗓门。
<br/>　　“大当家，大当家，出事了……”
<br/>　　程大雷将门打开，撇着嘴道：“干嘛呀，能不能让人休息一下？”
<br/>　　秦蛮看见了程大雷身后的黑衣老尼，呀地叫了一声，下意识握紧腰上的佩刀。
<br/>　　他想破脑袋也想不明白是怎么会事，怎么刚刚才把那老尼姑杀掉，她转眼又出现在程大雷屋里。
<br/>　　“别大喊大叫的，这位鱼先生，是我暗中寻找的高手，她以后就是自己人了。”
<br/>　　“那个……不是那个？”
<br/>　　“什么那个内个，说不是就不是了。对了，你来找我干嘛？出什么事了？”
<br/>　　“关鱼……”秦蛮脸上的表情颇堪玩味：“好像是中毒了。”
<br/>　　程大雷随着秦蛮来到火场，就见人围了一圈。刘悲已经急急忙忙赶到了，自家兄弟出事，他怎么能不担忧。
<br/>　　就见关鱼倒在地上，整条胳膊肿了一圈，一张脸也憋的黑紫，要多难看有多难看。
<br/>　　问过周围人，才知道是怎么会事。
<br/>　　程大雷走后，关鱼怼秦蛮怼得神清气爽，迈着步子走进火场中。随意捡起一只蛤蟆，道：“咦，这只蛤蟆闻起来好香。”
<br/>　　然后，他就倒下了。
<br/>　　现在倒在地上，整条胳膊又黑又紫。意识却还保持着清醒，心里悔得肠子都青了。
<br/>　　早知道，要积点口德呀。
<br/>(本章完)
