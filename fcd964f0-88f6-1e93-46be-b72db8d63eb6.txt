　　　　一个种族的强大，完全取决与种族的繁衍数量跟三阶妖灵级别的数量。
<br/>　　如今蛇潮推动，地面海洋般的群蛇密密麻麻，蜂拥而至，而天上也并不寂寞，十几条体形庞大，散发恐怖蛇类气息的巨蟒如神龙一样的腾空飞翔，目标都非常的明确，直指猴儿山领域。
<br/>　　猴儿山境内！
<br/>　　鼠潮已经包围而来，寸草不生，数不胜数的鼠兽如吞噬机器一样的，将前方任何事物全部吞噬干净。
<br/>　　你无法想象那种可怕的画面。
<br/>　　本来外围有很多种族势力观望，也因为鼠潮的降临，承受了灭顶之灾，能活下来的只有十之一二罢了。

<br/>　　鼠潮的降临席卷，或许对他这等深山大妖不会产生什么威胁，但是他的猴子猴孙怎么办，根本没办法抵挡，转眼就会被吞噬的一干二净。
<br/>　　普天之下，恐怕没有任何人能挡得住鼠潮，除非四阶圣灵从天而降。
<br/>　　绝望，猿猴王本来得到涂小安主人的命令，带着存活的猴族退到后方，还没等喘口气，这到好，转眼厄运就降临。
<br/>　　堂堂深山大妖只想说：草泥马。
<br/>　　现在真的是躲无可躲，他的身后无数只猴子惶恐不安起来，吱吱的发出哀嚎声。
<br/>　　因为它们感受到了死亡，这种死亡每分每秒都在逼近。
<br/>　　猿猴王不能帮自己的猴子猴子去抵挡鼠潮，不是他不愿意，而是他根本完全没有这种能力，若是去挡，他也会落的一个尸骨无存。
<br/>　　若在以前，猿猴王一定会跟赤阳巨猿一起破口大骂该死的蛇妖害人不浅，但是如今他成了别人的战宠，显然连骂的资格都失去了。
<br/>　　“大哥，我们猴族完了，彻底完了，不需要十分钟，猴族就要从妖灵山脉完全消失，完全被除名”
<br/>　　赤阳巨猿痛苦的用巨臂仰天捶胸，哀嚎连连，在之前没多久，它还在幸灾乐祸狼族的悲惨。
<br/>　　现在看来，狼族是幸福的，因为它们就是死了一个王，跟死了十几万的野狼，虽然元气大伤，但依然可以回到属于它们的狼山，修身养性，等待日后的崛起。
<br/>　　然，猴族现在面临的可是鼠潮，鼠潮一过，连一根猴毛都不会留下来。
<br/>　　“小弟，不要太悲观，还有一线生机，我们猴族还有一线生机的”
<br/>　　猿猴王将目光投放到青龙背上的那道睥睨的身影上，希望他可以创造奇迹。
<br/>　　“大哥，就算他是蛇王，就算可以号召整个蛇族蜂拥赶来，但远水解不了近渴，鼠潮已至啊”赤阳巨猿绝望的眼泪都下来了：“除非他能已一己之力，挡得住鼠潮的推进，给蛇族赶来争取时间”
<br/>　　猿猴王沉默了，鼠潮之可怕，不是一人之力可以抵挡的，那是碾压而来，群山都挡不住鼠潮，挡在前面的，都已经消失不见了。
<br/>　　..............
<br/>　　涂小安站在青龙的背上，他感受到了，感受到了无数强大蛇类的气息了，心中顿时豪气万丈，这次的召唤不同以往，绝对的大手笔。
<br/>　　他都无法预想会有多少强大的蛇种出现在自己的面前，听他的蛇王号令。
<br/>　　不过鼠潮来的太快，宛如龙卷风，说到就到，他的蛇族大军降临，还需要一点时间。
<br/>　　这一点时间，足以让鼠潮将整个猴儿山的所有夷为平地。
<br/>　　“三头邪蛇，黄金瞳给我回来”
<br/>　　涂小安意念一动，连忙将这两大战宠收回系统之内。
<br/>　　鼠潮太恐怖，它们又不会腾云驾雾，若是还继续留在地面，鼠潮一来顷刻间尸骨无存。
<br/>　　澎...！
<br/>　　之前被打入地层之中的肥鼠王冲了上来，大声咆哮：“蛇妖你的末日来临了，灰妖大王因为你发动一次鼠潮，你足矣骄傲了”
<br/>　　如今的肥鼠王已经呈现战斗本体状态，赫然一只无比巨大恐龙级别的巨鼠，浑身妖气沸腾，伪妖王层次不是盖的，显化本体的狰狞可以吓死人。
<br/>　　肥鼠王现在可是底气十足，鼠潮来临，一记强力的定心丸，可粉碎一切，鼠族天下无敌。
<br/>　　“我骄傲你玛”
<br/>　　涂小安看到这只悬浮在半空的巨鼠就来气，整个人骤然从青龙的背上消失，下一秒直接出现在肥妖王巨鼠的面前，气势雄浑凛冽，极尽刚猛霸道的一拳砸了过去。
<br/>　　这一拳砸过去，就是漫天火焰横流，伪妖王层次的巨鼠根本来不及反应的就惨叫一声，化为一颗陨石般的朝着鼠潮砸去。

<br/>　　涂小安蕴含天威火焰的一拳，绝对不是开玩笑的，够带劲。

<br/>　　“就让我涂小安来试一试这鼠潮到此有几分威力”
<br/>　　涂小安一拳砸飞了肥鼠王之后，直接就地降落，落到了地面后，一副气吞山河，横扫八荒的无敌气势。

<br/>　　别看青龙充满神兽气息，是个庞然大物，但是落到鼠潮中，不用一秒的时间就可以让它彻底从这个世界消失。

<br/>　　所以，不管出于什么原因，他都要全力的去挡住整个无比庞大规矩，鼠潮的推进。
