　　议事大殿之中。
<br/>　　帝释高坐于主座之上，他目光望着苏莫，面上露出了淡淡的笑容，道：“苏莫啊！你既然来提亲，直接来找本座便可以。”
<br/>　　呃~~
<br/>　　苏莫闻言一怔，顿时心中大汗，他此次可不是来提亲的！
<br/>　　不过，对方乍一开口，就认定他是来提亲的，他也不好反驳，总不能否认吧！
<br/>　　苏莫心中跟明镜似的，他知道帝释绝对是故意的，故意这般说，就算他不是来提亲的，也会变成是前来提亲的。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“那不知你带来了什么聘礼？”帝释微笑着问道。
<br/>　　“这个……！”苏莫迟疑了一番，心念急转，看来今日不提亲也不行了。
<br/>　　若是他说不是来提亲的，估计会被扫地出门吧，最主要的是，他若这样说估计会伤了夕儿的心。
<br/>　　沉吟一番，苏莫手臂一挥，三件宝物便突兀的出现，悬浮在了他的面前。
<br/>　　这三件宝物，分别是一枚玉简、一柄长刀、一块盾牌。
<br/>　　当这三件宝物一出现，整个大殿中瞬间便一片死寂，所有人的目光，均是第一时间落在了三件宝物之上，再也无法移动分毫。
<br/>　　就连帝释都是眼眸大亮，紧紧的盯着三件宝物。
<br/>　　因为这三件宝物，除了玉简的内容无法看到之外，长刀与盾牌，均是帝级下品神兵。
<br/>　　这可是两件帝级下品神兵，要知道整个帝玄宫，目前也只有一件帝级下品神兵而已。
<br/>　　“宫主，我的聘礼不多，只有三件，分别是一门帝级下品功法、一柄帝级下品长刀，一块帝级下品防御盾牌！”苏莫朗声说道。
<br/>　　三件帝级宝物作为聘礼，其实已经无比逆天了，堪称苍穹大陆上最珍贵的聘礼了。
<br/>　　苏莫本不想拿出来三件帝级宝物，但想到上次碧云湖一役，帝玄宫的帮助，他便没有吝啬。
<br/>　　宝物虽然珍贵，但帝玄宫的恩情，同样是非常的重。
<br/>　　“哈哈！好！难得你这么有心，这聘礼本座便手下了！”
<br/>　　帝释闻言大笑了起来，他丝毫没有拒绝，大手一探，立刻便将三件宝物抓了过去，而后郑重的收了起来。
<br/>　　帝玄宫其他的长老见此，也是个个面带微笑，虽然帝级神兵不可能分给他们，但帝级下品功法，他们还是有机会修炼的。
<br/>　　夕儿的脸上也是挂着欣喜的笑容，俏脸绯红，霎是美丽。
<br/>　　收好了三件宝物，帝释略作沉吟，问道：“苏莫啊！你和夕儿年纪也不小了，你准备什么时候成婚呢？”
<br/>　　“什么时候成婚？”苏莫闻言沉思了起来，既然提亲下聘了，成婚自然在所难免。
<br/>　　不过，还不待苏莫回答，帝释便再次开口了：“我看就选在两个月后的六月初六吧！是个好日子，非常适合成婚！就这么定了！”
<br/>　　帝释一言便直接敲定了日期，语气中蕴含不容反驳之意。
<br/>　　呃~~
<br/>　　苏莫彻底无语了，心中腹诽，你特么自己都决定了，还问我干嘛！
<br/>　　对方给他的感觉，就像是担心他反悔，又像是夕儿嫁不出去似的！
<br/>　　这也就是成亲的事，苏莫没有多大抵触，若是换了别的事，对方这么咄咄相逼，他都要发火了。
<br/>　　“那好吧！就两个月后，六月初六！”苏莫点了点头，既然定了亲，那早点晚点也就无所谓了。
<br/>　　“哈哈！如此甚好！”
<br/>　　帝释满面笑意，他看了看苏莫，又看了看自己的女儿，心中非常的满意。
<br/>　　现今的苍穹大陆，没有比苏莫更适合做女婿的人了。
<br/>　　不由的，帝释又想到了自己的弟子上官昊，心中无奈的叹了口气。
<br/>　　上官昊已经失踪了，仿佛人间蒸发了一般，就算是他，也是无法找到对方。
<br/>　　对于这个弟子，帝释还是非常看重的，即便现在有了苏莫这个女婿，他也是不想失去这个弟子。
<br/>　　苏莫和夕儿坐在一起，敲定了婚期之后，他便转头看向夕儿，轻声道：“夕儿，两个月后我们大婚！”
<br/>　　“嗯！”夕儿重重的点了点头，她并没有害羞的低下脑袋，而是目光直视苏莫，眸中温柔如水，仿佛有千般幸福、万般柔情！
<br/>　　“苏莫，我们再商议一下大婚的事宜吧！”少倾之后，帝释又开口了。
<br/>　　苏莫点头，随即便和帝释商议了起来。
<br/>　　对于大婚，苏莫不讲究什么排场，他的意思是随便小办便可，只是在苍穹神宫走个形式。
<br/>　　但帝释以及所有帝玄宫的长老，却是坚决不同意苏莫的意见，众人的意思是必须大办，并昭告天下，广邀天下所有大势力。
<br/>　　按照帝释所言，苏莫现在也算是苍穹世界的巅峰强者，又是苍穹神宫之主，而迎娶的还是他帝释的女儿，无论如何也不能弱了声势。
<br/>　　对于此，苏莫只能无奈接受，准备大办一场。
<br/>　　“苏莫啊！本座让你大办婚礼，其实还有一个其它的目的。”帝释沉声道。
<br/>　　“哦？什么目的？”苏莫疑惑的问道。
<br/>　　帝释闻言面色稍稍凝重看起来，他目光望向大殿之外，眸中璀璨的精光一闪而过。
<br/>　　“现在的苍穹世界，看似风平浪静，实则风起云涌！你和夕儿的大婚，就是剧变的开始！”帝释告诫道。
<br/>　　“还请宫主告知！”苏莫不太明白帝释的意思，抱拳问道。
<br/>　　“黄泉魔宗虎视眈眈，大有一统天下之势，他们也拉拢了一些大势力，欲掌控整个苍穹世界！”
<br/>　　帝释目光深邃，徐徐说道：“这次大婚我帝玄宫会和你苍穹神宫结盟，当然，也会将天涯海阁拉进来！”
<br/>　　“所以，以后的苍穹世界，就会出现两大阵营，分别是我们这一阵营，以及黄泉魔宗的阵营。”
<br/>　　“从这次大婚，便能看出其它势力的态度，让其它势力站队！”
<br/>　　“这也是我催促你和夕儿大婚的原因，宜早不宜迟！”
<br/>　　帝释面色郑重，说出了自己催促苏莫大婚的目的。
<br/>　　苏莫闻言心中讶然，这一点他倒是没有想到，果然姜还是老的辣，对方考虑的倒是非常全面。
<br/>　　的确，对方此言非常在理，他一个大婚就能看出天下各大势力的态度，让所有势力站队。
