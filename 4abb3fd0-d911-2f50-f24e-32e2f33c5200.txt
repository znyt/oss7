　　“注意安全。”
<br/>　　云熙从背后对着罗昊叮嘱了一声，罗昊对着云熙竖起一个大拇指，然后跳下了指挥车，带着司徒空、林晓阳、周柏等人消失在夜幕中。
<br/>　　司徒空取下自己背在身后的窄皮包，从里面取出一把折叠起来的复合弓打开，用内六角将几个支点紧固住，手轻轻拉动了一下弓弦，试了试弓弦的张力，将皮包中的箭杆跟箭矢旋紧在一起。
<br/>　　弓箭这种冷兵器，配合各种不同的箭矢，在特种作战中能有很广泛的应用。
<br/>　　在射程和威力上，弓箭或许无法跟自动武器相比，但是夜间作战中，弓箭却是有着热武器无法比拟的优势。尽管自动武器可以使用消音器，以及使用微声子弹做到无声杀敌，可是自动武器射击时所产生的火光，却是无法被掩盖的，火光就有可能会曝光行动队员的位置。
<br/>　　可是弓箭却能做到真正意义上的无声杀敌！
<br/>　　刘三看了眼手机上的时间，对李霄力说道：“力哥，时间快到了。”
<br/>　　“有些不对劲。”李霄力皱眉看着周围港口周围，说道：“这里太安静了。”
<br/>　　“港口工人都下班了，安静一点也没什么特别的。”
<br/>　　刘三觉得李霄力有些太过敏感，但是又不好直接明说，对着负责踩点的马仔招了招手，问道：“你们在这里盯着的时候，有什么异常吗？”
<br/>　　“没有。”马仔摇了摇头，回答道：“我们几个人在车里一直都在盯着呢，没有发现什么异常。”
<br/>　　“力哥，您就放心吧。”刘三对李霄力说道：“我们中间没有警察的卧底，不会走漏消息的，只要船一进港，我们验完货分散离开，没人会知道我们今天晚上在这里做了什么。”
<br/>　　李霄力周围扫了眼，注意到港口附近的监控探头，说道：“监控探头断线了没有？”
<br/>　　“对、对不起力哥，我们疏忽了！”
<br/>　　负责踩点的马仔立刻意识到自己做错了事情。
<br/>　　“怎么做事的！？”
<br/>　　刘三在那个马仔脑袋上狠狠地拍了一下，瞪着他骂道：“到底会不会做事，这么简单的事情都做不好，还愣着干嘛，去把线剪了！”
<br/>　　马仔立刻带着几个人去把附近监控探头的线剪断掉。
<br/>　　指挥车中，云熙看着突然黑屏的屏幕，拿起对讲机对罗昊说道：“他们剪断了监控探头的电线，我这边无法看到现场情况了。”
<br/>　　“我接入战场视频系统。”
<br/>　　罗昊打开了携带在身上的战场视频系统，指挥车中又重新看到了现场的画面。这些通过战场视频系统拍摄到的影像，包括一会儿行动时的画面全都会被保存下来，经过剪辑后，会被当做起诉李霄力的证据。
<br/>　　当然，即便是没有这些视频，就李霄力私藏枪械、管制刀具，以及那十公斤的高纯度cocaine，这些罪名加起来，枪毙他十次都足够了。
<br/>　　“三儿，我要的机票订好了吗？”李霄力对刘三问道。
<br/>　　“订好了，凌晨两点飞往美国的航班。”刘三回答道。
<br/>　　李霄力点了点头，对刘三的办事效率很满意。内心的不安，让李霄力如坐针毡，他相信自己的直觉，他能感觉到危险正在向自己靠近。
<br/>　　注册国籍为南美洲某国货轮缓缓驶入港口，这是今天晚上最后一艘入港的货轮，也是李霄力所等待货轮。
<br/>　　由市刑侦队副队长刘炯伪装而成的港口工作人员，手中拿着手电筒，嘴里故意嘀嘀咕咕抱怨着，朝着港口的方向走去；李霄力的视线在刘炯身上停留了一下，最终没有发现什么疑点。
<br/>　　这个行动中，如果说有什么风险的话，这就是唯一的风险点；如果留下港口的工作人员，可能会在行动中造成误伤；如果不留下工作人员，必然会引起李霄力的警觉。
<br/>　　所以讨论后决定让刘炯伪装成港口工作人员，在货轮进入港口后，跟货轮上的船长完成办理手续的工作。刘炯也是个二十多年的老警察，经验丰富，有其他年轻警察所没有的沉着冷静，不会轻易漏出马脚而引起李霄力的警觉。
<br/>　　甚至为了不让李霄力产生怀疑，刘炯主动提出不穿防弹衣，不携带武器。臃肿的防弹衣还有鼓起的枪套都有可能会暴露自己的身份，就这份胆量，让得罗昊对刘炯这个老警察心生敬佩。
<br/>　　“我们的人办理完手续，已经撤出来了。”
<br/>　　慕容飞透过狙击镜看着刘炯走下货轮，打着哈欠离开，刚松了口气，突然看到李霄力朝着刘炯走了过去，然后故意在刘炯身上撞了一下，把他拿在手里的文件夹给撞落在地上。
<br/>　　看到这一幕，慕容飞顿时又紧张了起来，把狙击枪瞄准了李霄力伸向腰后的右手，如果李霄力有什么异动，慕容飞会毫不犹豫的开枪。这一颗子弹下去，李霄力不会死，但是他的手一定会被打断。
<br/>　　刘炯弯腰捡起地上的文件夹，拍了拍上面的灰尘，看了眼李霄力，随后指了指那艘刚办完手续的货轮，说道：“龙门吊的工人都下班了，今天卸不了货！”
<br/>　　李霄力盯着刘炯看了一会儿，把伸向腰后的手又拿了回来，对刘炯说道：“这货轮上的船长是我朋友，我来接他的。”
<br/>　　李霄力从口袋里摸出一包玉溪递给刘炯，说道：“师傅，抽烟。”
<br/>　　“不会！”
<br/>　　刘炯对着李霄力摆了摆手，说道：“别在这里逗留的太久，大晚上在这海边太危险，要是不小心掉下去了，喊救命都没人能听见了。”
<br/>　　“我们接了人就走，不会逗留太长时间的。”李霄力对着刘炯抬了抬手，说道：“辛苦了，师傅。”
<br/>　　“孩子要读书，没办法。要不然谁大半夜不睡觉在这里候着。”刘炯又抱怨了几句，离开了。
<br/>　　通过罗昊身上的战场视频系统看到刘炯安全离开，指挥车中的周宋捷也是松了一口气，行动中最薄弱了一个环节还好没有出现纰漏，其实这个时候就算是李霄力发现了什么，他也不可能逃掉，唯一的顾虑就是刘炯可能会遇到危险。
