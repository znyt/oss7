　　“不可能！”
<br/>　　罗昊斩钉截铁的说道：“布克尔特这混蛋一定在这里！摩托车的轮胎印是到这里中断的，而且附近没有脚印，所以布克尔特一定藏在这里。”
<br/>　　“可是这间屋子就这么巴掌大一片地方，一眼过去都看到了，根本就没有人，难道他还能七十二变，变成一只小虫子？”司徒空不解。
<br/>　　罗昊在安全屋的墙壁上不断敲打着，罗昊现在已经可以肯定，这间安全屋中肯定藏着暗室，不然无法解释布克尔特和佟千雪为什么会消失在这间安全屋中。
<br/>　　把安全屋四周的墙壁都敲打了一遍，四周的墙壁都是实心的，除了那个竖在墙壁边的木柜没有检查。
<br/>　　罗昊走到木柜前，仔细的打量着眼前的木柜，伸手在木柜上轻轻一推，发现木柜底部的地面上竟然安装了滑动槽，可以让木柜自由移动。
<br/>　　“嗬，藏在这里呢。”
<br/>　　周立上前正打算一把将木柜推开，可是却被罗昊一把扣住他的手腕，“别动！”
<br/>　　罗昊指了指他们脚下的位置，一根在空气中闪烁着微弱金属光泽的细钢丝，赫然横在他们面前。
<br/>　　看到细钢丝，周立忍不住倒吸一口冷气，刚才自己如果把木柜再推开一点，极有可能触发跟眼前这根细钢丝相连的东西。如此狭小的空间中如果触发一颗诡雷，引发爆炸，后果简直不堪设想。
<br/>　　罗昊蹲下身，伸手在细钢丝上触摸了一下，顺着细钢丝找到了藏在木柜背后利用80mm迫击炮弹和手雷组合制作而成的诡雷，如果刚才被周立触发了这个陷阱，在场的人估计要被炸死一大半。
<br/>　　“妈的，这个布克尔特居然还有空设置这样的陷阱。”陈应天瞪了周立一眼，说道：“笨牛，刚才差点就被你给害死了！妈的，老子还没娶老婆呢，你是想要让我老陈家断后啊！”
<br/>　　罗昊取出斜口钳，将那根连接着诡雷的细钢丝剪断，仔细检查确认在木柜背后不再有陷阱存在后，才让陈应天和周立两个人把木柜推开。
<br/>　　那扇使用脱氧钢打造，厚度足有十公分的地下室门出现战术中队面前。
<br/>　　地下室的脱氧钢板门使用密码锁控制，要打开的方式只有两种，让人从里面打开或者正确输入密码进入。
<br/>　　“蜂鸟，能打开吗？”罗昊对林晓阳问道。
<br/>　　林晓阳走到钢板门的密码锁前，检查过后，摇头道：“这个密码系统是目前世界上最先进的密码系统，三次输入密码机会，如果尝试失败，就无法从外部再打开，只能由人从内部开启。跟我们当年‘暗之龙’中遇到的一样，这种系统定时更新，有效防止黑客破解，使用价格的确是要比其他密码系统昂贵，可是这玩意儿值得起它这个价格！”
<br/>　　“这怎么可能猜得到！”司徒空嘀咕道：“我们又不是布克尔特肚子里的蛔虫，怎么可能知道他在想些什么！”
<br/>　　“三次机会，那也就意味着我们有两次尝试的机会！”
<br/>　　罗昊盯着脱氧钢板门上的密码锁盘，说道：“当然还有一种办法，那就是用焊枪把这扇铁门切开，这样的话，耗费的时间会有点多。而且布克尔特既然带着佟医生藏在这里，肯定不可能只是龟缩在这里等着我们离开，他清楚这基本是不可能的事情，所以他必然是还有其他准备。”
<br/>　　“我介意这样，我们可以尝试着猜测密码，同时跟比洛多取得联系，让他调两组焊枪过来，做两手准备。”梅庆伟提议道：“我来跟比洛多进行联系，你们猜测密码。”
<br/>　　罗昊点了点头，觉得梅庆伟提出的这个办法有效可行，从比洛多那边调两组焊枪过来，这样即便是猜错了密码，也能使用焊枪切割开钢板门。
<br/>　　地下室中，布克尔特看着通过安装在安全屋中的监控探头传输来的画面，对佟千雪问道：“你觉得他们能猜出密码吗？”
<br/>　　还没有等佟千雪回答，布克尔特就直接说道：“我认为他们猜不出密码，这个世界上，除了我自己以外，没有人能知道这扇门的密码。”
<br/>　　安全屋中，罗昊盯着密码盘思考了一会儿，对司徒空问道：“司徒，如果是你，你会用什么来作为自己的密码？”
<br/>　　司徒空想了想，说道：“生日，简单好记；或者是家庭地址的数字缩写，这个也容易记忆。”
<br/>　　“布克尔特是在贫民窟里长大的，所以可以排除掉家庭地址的数字缩写这个选项。”罗昊打开自己手腕上的PDA，翻看着存储在里面布克尔特的资料，说道：“不过他的生日倒是可以尝试一下。”
<br/>　　罗昊在密码盘上输入了资料中布克尔特的生日，但是密码盘上方的屏幕中却是提示密码错误，并且提醒罗昊还有两次输入密码的机会。
<br/>　　“会不会是他农历的生日？”周立开口道。
<br/>　　“你是不是蠢？”陈天宇在周立后脑勺上拍了一下，说道：“外国人哪儿来的农历？”
<br/>　　“布克尔特在‘绿色贝雷帽’服役过，那是改变他人生轨迹的存在，可以试试他的服役号。”周柏提议道。
<br/>　　罗昊点了点头，输入了布克尔特在“绿色贝雷帽”服役时的服役号，但是依旧被提示密码错误。
<br/>　　一共三次输入密码的机会，现在已经输错了两次，如果第三次再输入错误，这个密码盘就会被系统锁死，只有从内部打开或者是使用暴力手段从外部打开。
<br/>　　这个时候梅庆伟回到了安全屋中，他的脸色有些难看，对罗昊说道：“狂龙，比洛多拒绝配合我们的行动，在村子里的机动部队也被撤回了，他说我们跟他之间的合作已经结束了！”
<br/>　　“他这是什么意思！？”
<br/>　　从进入安全屋后，一直没有开口说话的慕容飞这个时候开口道：“怎么突然就中断合作！？”
<br/>　　“原因很简单。”罗昊解释道：“因为有人给他施加了压力，让他不得不放弃跟我们之间的合作！”
