　　砰砰！
<br/>　　两声闷响，小杰和奇犽双双摔倒在地。大口喘气，几次挣扎却始终无法站起。
<br/>　　脸上肿胀出血，奇犽的一条胳膊，甚至扭成了诡异角度，直接被打折！
<br/>　　两人不远处，苏重静静的站在原地，脸上面无表情，好似一台冰冷机器。这是苏重制造的线傀儡。
<br/>　　接到奇犽求助信件，苏重立刻动身来到NGL边境。除了来看一看奇犽，更多的是想要跑出揍敌客，来找比杨德和帕里斯通的麻烦。
<br/>　　只是奇犽向他求助，隐晦提出让苏重训练的意思。苏重感觉有趣，就答应了下来。这才有了眼前这一幕。
<br/>　　不过苏重可没兴趣亲自下场。以他如今实力，就是站着不动让两人打，累死两人破不了他的防！
<br/>　　苏重也没那个闲心亲自出手训练。线傀儡正好派上用场。
<br/>　　线傀儡非常奇特，兼有自主性念兽和操控系的操纵属性。相当于从操纵系念线，变异出来的一种独立念兽。
<br/>　　受到苏重制约，也有着一定的机械智慧，好似分身一般。正好用来给奇犽小杰两人做陪练。
<br/>　　“你真敢下手！”比斯基双手握拳，咬牙启齿的盯着苏重。
<br/>　　苏重盘坐在树荫之下，闻言睁开眼睛，看了眼伤痕累累的小杰两人，淡然道：“想要走捷径，当然要付出代价。只是断手断脚，没让他们折寿，就已经很好了。”
<br/>　　手指勾动，不远处线傀儡身上浮现一条条念线。整个身体分解成一团线，最后消散在空气中。
<br/>　　苏重起身走到两人身旁，双手对着远处茂密森林一抓。两团绿色生命能量汇聚于掌心之间。
<br/>　　不等小杰两人作出反应，就被绿色能量拍入两人体内。
<br/>　　肉眼可见的，奇犽和小杰脸上的伤痕快速约合消失。肿胀处迅速消肿。几个呼吸之间，除了沾染的血迹，已经看不出任何伤势。
<br/>　　奇犽那条扭曲的手臂，也被他自己摆正，随着绿色能量聚集，很快就恢复如初。根本看不出曾经扭曲骨折。
<br/>　　这个半个月以来，两人对这种情况已经见过很多次。一天里，甚至有多次接受绿色能量治疗。
<br/>　　可即便如此，每次恢复如初之后，依旧感觉很惊奇。
<br/>　　“这大概就是糜稽当初敢疯狂训练的依仗了吧。”奇犽若有所思。他想到了当初苏重那堪称地狱式的训练。
<br/>　　到如今，枯枯戮山深处，仍然有一个巨大的露天矿场似得碎石坑。那里面的每个细小砂砾，可都是苏重一个人从山中挖出巨石，然后亲手打碎！
<br/>　　正式因为能快速回复，且不损伤潜力，糜稽才敢那么疯狂训练，实力才会急速提升。
<br/>　　心里想着，奇犽右手活动了几下，一点感觉不出刚才曾被拗断过。甚至感觉，比被拗断之前，还要强横几分。心中一动，奇犽不由对着空中猛击一拳。
<br/>　　啪！
<br/>　　一声闷响，拳头冲破空气的声音，异常清晰。
<br/>　　比斯基脸色一变，又变强了！
<br/>　　半个月以来。苏重用线傀儡，不断攻击奇犽和小杰两人。实力始终控制在搞出两人一线的程度，下手却格外重。每次都把两人打伤打残。然后就会用那种诡异能力帮两人恢复。
<br/>　　起初比斯基还没有发觉，可两次之后。当她用自己的念能力，帮两人去除疲劳时。她敏锐的发觉，奇犽两人的身体素质，竟在飞速提高，在不正常的提高！
<br/>　　比斯基是体术大家，对人类身体了若指掌。立刻就看出其中不对。之后仔细观察，很快就确定，那种绿色能量，不仅能治愈伤势，更能提升身体素质！
<br/>　　“怪不得实力那么强大，有这种掠夺植物生命，强化自身的能力，实力不强才怪！”比斯基对当初苏重一拳轰死申猴塞右的场面，记忆犹新！
<br/>　　只是这种能力……
<br/>　　比斯基看了看周围。
<br/>　　半个月之前，这里是一片绿色林海，到处都是郁郁葱葱的植被。
<br/>　　可现在地面是枯黄的杂草，四周布满枯死树木。从上方向下看，就好似绿色的大地上，打了一块黄色的补丁。而远处的植被也呈现出衰败迹象。宛如滴在宣纸上墨汁，不断向着四周扩散。
<br/>　　这种能力让她心中发寒。吞噬植物生命增强自身。只是想一想，比斯基就忍不住打了个寒颤。看着苏重从容淡定的模样，她越发觉得，这就是一个披着人皮的凶残妖怪。
<br/>　　不知是不是幻觉，每当她仔细观察苏重的时候，他都觉得自己是在看一个恐怖魔兽。看得久了，甚至会有种错觉，仿似看到了对方闪烁寒光的尖牙。
<br/>　　“好了，这是最后一次训练，接下来的训练，就看你们自己的了。”苏重收回手淡淡道。
<br/>　　“糜稽，你要走？”奇犽皱着眉头。
<br/>　　这段时间他和小杰的实力飞速提升，他不是傻子，自然知道这都是苏重绿色能量的功劳。没人不喜欢强大的力量。
<br/>　　“我来这里，可不是为了给你当教练，逗留了半个月，时间已经够长了。”苏重看了眼一脸惋惜的奇犽，知道这小子练上瘾了，不由好笑：“你这是吃着碗里的，看着锅里的，贪心的很那。你以为，我打入你们两人体内的生命能量，是那么好吸收的？”
<br/>　　“治疗伤势只消耗了一部分，其他的都已融入你们体内。且能量存储，已趋于饱和状态，只等你们去淬炼挖掘！即便继续给你灌生命能量，也没法吸收。除了疗伤，之后就会逸散出体外，白费功夫。”苏重眼珠转动，扫了一眼满脸傻笑的奇犽，没说什么转身就走。
<br/>　　“谢谢大哥哥帮忙，再见！”小杰在背后大声感谢。
<br/>　　奇犽面色复杂的看着苏重背影，自己这个二哥，已经完全变了样子。从以前那个总是找他麻烦，好似长不大似的胖子，变成了一个实力强大，足以给他提供庇护的二哥。
<br/>　　这么多年过去，在那个冰冷家族，也似乎只从这位二哥身上，才感觉到一丝家的味道。因为他的肆无忌惮，因为他不在乎揍敌客这三个字。
<br/>　　奇犽知道苏重去干什么，那些蚂蚁可不简单。一个个实力强大。
<br/>　　一定要保重啊。奇犽暗自祈祷。旋即哑然失笑，自己这想法真是有些好笑：“以糜稽的实力，如果有嵌合蚁对上他，该担心的应该是那些蚂蚁吧。”
