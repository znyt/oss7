        谁知，居然被季思妍反将一军。
<br/>
<br/>        进地利置业的第一天，季思妍就已经给身边的同事发了新人见面礼。
<br/>
<br/>        更何况，她季思妍因为救助孕妇的助人为乐的事情，在网上成了最美女司机的小网红，人气爆表。
<br/>
<br/>        别说男同事对她有好感，女同事们也对她格外亲近。
<br/>
<br/>        公认的心地善良的女人，自然是人心所向。
<br/>
<br/>        姚明娜这个时候冒出来，简直就是跳梁小丑，明着大家都帮着她“鼓励”季思妍，但是私底下，这种事情早就传到了季思妍的耳中。
<br/>
<br/>        季思妍有家世有背景的，还是傅大Boss傅夜沉介绍进入地利置业的，自然没童筱颖那种没家世没背景纯靠自己努力工作的女人好欺负。
<br/>
<br/>        姚明娜光顾着吃醋，智商完全不在线。
<br/>
<br/>        季思妍坐在茶水间里亲自给身边的同事们泡手磨咖啡，听到她们说姚明娜在针对自己的时候，只是云淡风轻地微笑道：“小司峻人长得那么帅，又是单身，对下属又好，招我们大家喜欢，难道不是一件好事吗？再说了，我们对小傅总的喜欢，是单纯地崇拜与好感，就好比我们是小傅总的粉丝一样。大家实在是没必要为了小傅总，去针对谁，对吗？我们应该团结起来，把手里的工作做好，把我们的业绩推上去，让公司越来越好。”
<br/>
<br/>        “就是啊！以前，那个财务部的姚副部长每次一看到公司里那个女同事和小傅总走得近，就故意刁难那个女同事，直到把那个女同事逼走！”坐在圆桌前的一个女人，一边小啜着咖啡，一边眉飞色舞地说。
<br/>
<br/>        另一个女同事也跟着开腔，说起了八卦的事情：“前段时间，我们设计部来了个在校未毕业的实习生，是小傅总介绍进来的。姚副部长就特意跟我们设计部的孙副部长打了声招呼，拿业务故意为难那个小实习生了！”
<br/>
<br/>        “后来呢？”季思妍忍不住好奇地问，有些难以置信，阿景会给一个在校生开后门。
<br/>
<br/>        女同事接着说道：“后来，那个实习生虽然应付了孙副部长出的难题。但是她抄袭了我们设计部另一个同事崔素妮的设计稿，孙副部长一怒之下，为了维护公司形象，就直接把她赶出了公司。”
<br/>
<br/>        “原来那个小实习生品德不行啊！”季思妍突然暗自窃喜，不经意地扬了扬嘴角，漫不经心地端起咖啡杯喝了起来。
<br/>
<br/>        “这件事情，最后小傅总不是帮那个小实习生证明了清白吗？最后，小傅总还让法律顾问部把崔素妮给告了！”在座的另一个女同事接着补充道。
<br/>
<br/>        “噢！对对对！是小傅总出面，还了那个小实习生的清白！小傅总人真好啊！”
<br/>
<br/>        “是啊！小傅总对下属真的好！”
<br/>
<br/>        大家七嘴八舌，都只是在夸傅昼景的好，唯独季思妍却在心底非常在意那个“小实习生”。
<br/>
<br/>        “那个小实习生叫什么名字啊？”季思妍装作若无其事地随口问道。
<br/>
<br/>        女同事想了一下，才回答道：“好像是叫梁……什么沫！”
<br/>
<br/>        “是叫诺筱颖啦！”另一个女同事笑着说。
<br/>
<br/>        “对对对！诺筱颖！”
<br/>
<br/>        “其实她人挺好的，乖巧又热心，工作努力又上进。”
<br/>
<br/>        “只可惜，她没啥背景啊！”
<br/>
<br/>        “不是听说她是小傅总的助理赵斌的女朋友吗？”
<br/>
<br/>        “都是一些八卦绯闻吧！赵助理怎么可能会看上她！”
<br/>
<br/>        “听说她家里很穷噢！”
<br/>
<br/>        季思妍淡淡地笑了笑，不再二话了。
<br/>
<br/>        梁、以、沫……
<br/>
<br/>        她也不知道自己为什么会一下子就记住了这个名字。
<br/>
<br/>        童筱颖去菜市场买了食材回来，刚出电梯门就打了个喷嚏，一边揉了揉鼻子的时候，抬眸看到家门口站着两个穿着西装革履的男人，心里莫名其妙地涌上一种不祥的预感。
<br/>
<br/>        其中一个男人看到童筱颖朝这边走过来，一脸严肃地问道：“请问，这是傅昼景先生家吗？”
<br/>
<br/>        童筱颖抿了抿唇，弱弱地问道：“你们是？”
<br/>
<br/>        “我们是……”
<br/>
<br/>        “叮——”地一声，另一个电梯的楼层抵达声响起。
<br/>
<br/>        童筱颖看到傅昼景从电梯里走出来，下意识地往傅昼景身边靠去。
<br/>
<br/>        傅昼景看到这两个男人的时候，不禁皱了下眉头，随即又若无其事地微笑着说道：“二位进屋坐坐。”
<br/>
<br/>        童筱颖见傅昼景似乎是认识这两个男人，这才敢掏出钥匙，按了指纹打开家门。
<br/>
<br/>        傅昼景邀请这两位进了家门后，便招呼他们去了客厅坐下。
<br/>
<br/>        童筱颖将食材放进冰箱后，非常识趣地去给那两个客人泡茶。
<br/>
<br/>        她刚端上茶，并将手机交给了傅昼景后，傅昼景便命令她去房间里待着。
<br/>
<br/>        童筱颖没有二话，点了下头，便去了自己的卧室。
<br/>
<br/>        其中一个男人情不自禁地当着傅昼景的面前，夸赞童筱颖乖巧懂事，难得他傅昼景单身这么多年，如今也能轻而易举地就找到这么一个既漂亮又乖巧的女孩当女友。
<br/>
<br/>        让筱颖被外人误会成是自己的女友，傅昼景此刻听着这两个男人的题外话，只是淡漠地笑了笑。
<br/>
<br/>        三个男人在客厅里究竟谈论着什么，童筱颖自然是非常地好奇。
<br/>
<br/>        所以，她匍匐在门边侧耳倾听。
<br/>
<br/>        可她只能听到他们三的说话声，却听不清他们三在说些什么。
<br/>
<br/>        中途，她还听到傅昼景大发雷霆地怒吼，似乎是遇到了什么令人暴跳如雷的糟糕事。
<br/>
<br/>        童筱颖有点儿替傅昼景赶到担忧，想出去，又怕影响到傅昼景，反倒给他添麻烦，只能急得独自在房间里徘徊。
<br/>
<br/>        直到她听到开门声和关门声，心里揣测着那两位客人应该是已经走了，便立即开了房门走了出去。
<br/>
<br/>        童筱颖走到客厅里，只见傅昼景拉松了领带，颓废不振地躺在沙发上，看着天花板发呆时，便挪不过去，在傅昼景的身边坐了下来。
<br/>
<br/>        “昼景，发生什么事了吗？”童筱颖心绪不宁地问。
<br/>
<br/>        傅昼景黑眸眯了眯，眸光深邃难测，忽然冷不丁地开口说道：“筱颖，我们离婚吧！”
