　　大概十几分钟的功夫，刘哔来到城门跟前，负责检查的士兵上前，接过了车窗递过来的晶核，挥手让刘哔将车开了进去。
<br/>　　基地并不限制车辆进入，只是需要缴纳额外的入城费用，在来时的路上，刘哔顺手收取了几枚晶核，早就做好了进城的准备。
<br/>　　因为生化危机的爆发，像末世前那种货币早就停止了使用，现在几乎所有地方，都是用晶核作为货币。
<br/>　　这种从丧尸脑袋里取出来的东西，用途十分广泛，不仅可以作为异能者的修炼材料，还可以用来制作很多东西，譬如城墙上那几座巨大的魔晶炮。
<br/>　　故此，晶核也就成了末世的硬通货，除却一些特殊的东西之外，寻常物品，用晶核都能买到。
<br/>　　在禁闭室里呆了小半天，一直到黄昏的时候，刘哔才被放了出来，这是基地的惯例，为了基地的安全着想，从城外回来的幸存者，都要在一个小房间里呆几个小时，看看会不会变成丧尸。
<br/>　　出了禁闭室，刘哔草草的将车停在停车场，心急火燎的往家中赶去，他居住的地方是贫民区，说是家，也不过是一个用泥瓦搭建起来的简易房屋罢了。
<br/>　　这里还是那副老样子，街道两旁都是残破狭小的砖房，大多都房门敞开着，一些面黄肌瘦的幸存者，有气无力的坐在门口。
<br/>　　这些人大多是一些妇孺老人，年轻力壮的幸存者这个时间都在外面和丧尸厮杀着，奢望能带回来一些生存资源。
<br/>　　虽然说基地每天都会提供免费的食物，但也只不过是连米粒都难见的清汤，光靠那种东西，连活下去都成问题。
<br/>　　可是，即便是每天都在拼命，他们也很难将生活维持下去。
<br/>　　基地附近的资源早已经被搜集的差不多了，想要找到点什么，又怎么会太过容易。
<br/>　　人命，在这种环境下，还不如一包过期了的饼干值钱。
<br/>　　“阿哔啊，你回来了，真是幸运呢，我们都以为你出事了呢。”走在坑坑洼洼的道路上，刘哔脚步匆匆，突然听到一道熟悉的声音。
<br/>　　说话的是一妇人，年已古稀，脸上布满了密密麻麻的皱纹，那是岁月留下的痕迹。
<br/>　　是大柱他娘，两家做了几年邻居，关系虽然算不上要好，但也能说上几句话。
<br/>　　倒是这个老太太，为人挺不错的，自己不在的时候，经常帮着照顾自己生活不方便的弟弟。
<br/>　　这样淳朴的老人，在末世前倒也常见，但现在这种年头，已经不多了。
<br/>　　反而他的儿子大柱，更像是在末世常见的普通幸存者一般，沉默寡言，大多时候都在盘算自己的事情。
<br/>　　像他们这种人，是没有朋友的，只是活着，就已经很费心劳神了，哪还有空闲的精力，去经营一段友情。
<br/>　　也没有人，那么轻易就能交出自己的信任。
<br/>　　“大娘，小莽怎么样了，家里没什么事情吧？”停住脚步，刘哔脸上勾起一抹笑意，心情也变的开朗了起来。
<br/>　　看来是家里没出什么事情，否则的话，大娘早就和自己说了。
<br/>　　“唉，回来就好，家里能有什么事情，就是你走了一周多，也没留下什么东西，小莽这孩子，都快给饿坏了。”老人挪了挪椅子，有些费劲的想要站起来。
<br/>　　刘哔赶忙伸手，将老人搀扶了起来，一边将老人搀进屋，一边问道：“大娘，您是说，我离开了一周？”
<br/>　　转头看了看刘哔，老人点头应了一声，脸上的表情有些奇怪。
<br/>　　知道是自己失态了，刘哔挠了挠头，开口解释道：“我被困在一个地下室里，都记不清时间了，没想到已经过去一周了啊。”
<br/>　　恍然大悟的点了点头，老人笑了一下，脸上的皱纹都挤成了一团：“那可真不容易呢，没受什么伤吧，赶紧去回家看看把，小莽这几天老是念叨你呢。”
<br/>　　‘哎’了一声，刘哔将老人扶到床边，脚步匆匆的出了门口，走进了一旁自家矮小的房屋。
<br/>　　房间里的空间很是狭小，没有什么家具，只有一张粗糙的土炕，上面铺着一些打着补丁的被褥。
<br/>　　虽然简陋，但东西收拾的却十分整齐，床上的有些破旧的被褥，倒也干净。
<br/>　　此时，屋内有一个留着短发的青年，正用仅剩的左手，清理着房间内的地面。
<br/>　　听到门口处的动静，青年转过身来，脸上的表情却猛然凝固，随即，一抹狂喜从其目光中透露了出来。
<br/>　　“哥！”
<br/>　　激动的叫喊了一声，青年扔下手中的拖布，满脸惊喜的跑了过来，用仅剩的左手，拉着刘哔转了一圈。
<br/>　　“哥，你回来了，你终于回来了，我还以为你出什么事来了呢，你干什么去了，受伤了没，担心死我了都要！”足足经过了十几分钟，青年才平复下来了激动的心情，却还是拉着刘哔不肯放手，口中喃喃的说道。
<br/>　　身后摸了摸眼前青年的头发，刘哔脸上满是笑意，任由他拉着自己的胳膊，带着青年来到了床边坐下。
<br/>　　这是他弟弟刘莽，也是他在这个世界上最亲近的人。
<br/>　　“莽，这段时间怎么样，我没回来，你是怎么生活的，王大头有没有欺负你？”拉着刘莽的手臂，刘哔目光很是柔和。
<br/>　　王大头就是这一片的保安队长，专门负责贫民区的治安和救济粮派放，也没少从中克扣粮食。
<br/>　　要不基地高层给拨出的粮食，也不至于让人连吃都吃不饱。
<br/>　　“没事，哥，你能回来就好，这段时间，多亏秦老太太了，要不是她老人家时不时给些口粮，估计我还很真得饿死。”坐在刘哔身边，刘莽的语气有些唏嘘。
<br/>　　现在可不是盛世，断掉一根胳膊也不至于饿死，在末世这种世道，本来异能等级都不高，还断掉一根胳膊，是根本不可能从外面搜索到什么资源的。
<br/>　　想活下去，自然是变得无比困难。
<br/>　　闻言，刘哔的目光有些闪动，秦老太太，就是刚才的老人，她的状况自己也了解一些，都不是什么富裕的人，估计为了给小莽送口粮，也没少挨自己儿子的骂。
