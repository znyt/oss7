　　不周山两大先天灵根——先天葫芦藤与先天芭蕉树！
<br/>　　先天芭蕉树，传说之中芭蕉树上生有四片芭蕉叶，分别主地水火风，其中，前两片虽然默默无闻，但，后两片却是闻名天下。
<br/>　　芭蕉风扇，为后世冥河教主所得，传到一位名为罗刹公主的手上，一扇就可以将敌人扇出八万四千里，端是厉害非常。
<br/>　　芭蕉火扇，则为元始兄弟太清老子所得，后期化身太上老君以此扇煽火炼丹，西游之中，其座下金角、银角两个童子曾经用此芭蕉火扇差点搞定齐天大圣孙悟空，简直是神器啊！
<br/>　　至于前两扇地扇及水扇，虽然默默无名，但由后两扇就可以看出前两扇也必然不弱，毕竟它们都是同一株芭蕉树上所生之芭蕉扇。
<br/>　　先天芭蕉四扇，带有地水火风四象本源，对于元始的《玄黄不灭》第三重四象炼四极尤为重要，元始也志在必得。不过，现在它还只是幼苗，并没有大用。
<br/>　　至于先天葫芦藤，比起先天芭蕉扇还要出名。提起先天葫芦藤之上的葫芦，元始最先想到的就是太上老君手中的紫金葫芦，既可以装丹，又可以杀人。任你是大罗金仙，只要叫你一声，你敢答应，便会被吸入其中，化为脓水。
<br/>　　第二个则是封神之战中，西昆仑散仙陆压的斩仙飞刀。想到斩仙飞刀，元始不得不承认，这斩仙飞刀比起太上老君的紫金葫芦还要厉害，葫芦之上有三尺毫光，只要对着葫芦一拜，唤一声“宝贝请转身”，便可占下敌人六阳之首，也就是看项上人头。
<br/>　　第三个葫芦则是洪荒大神通者红云老祖的紫红色九九散魄葫芦，可以散人魂魄，端是厉害。
<br/>　　第四个则是洪荒继鸿钧老祖后第一位圣人女娲娘娘的炼妖壶，内藏天地，炼尽天地凶煞之物，取其本源，为先天大神器。
<br/>　　第五个为元始的紫黑混元葫芦，内藏混沌空间；第六个为通天教主的紫青水火葫芦，内含水火之力，第七个元始则是不知为何物。尽管后三个没有前三个出名，但元始并不认为它们没有比前四个厉害，只是它们的主人低调而不显名声而已。
<br/>　　可惜，此时，无论是葫芦藤，还是芭蕉树，都还只是一株幼苗罢了，距离开花结果，还不知道要多少岁月，元始也不能一直在此守候，他只好简单的设置一个阵法，在此守护，留下元神印记，随即就准备离去，毕竟，他知道自己在不周山待的时间已经够长了，是该到别处转转了。
<br/>　　不过，就在他准备离去之时，突然感觉好似有什么东西在召唤自己一般。
<br/>　　“难道是自己的机缘来了？有灵宝与我有缘？！哈哈！”
<br/>　　元始认为这是与自己有缘的灵宝，相互之间的吸引。他顿时激动起来，连忙向感应到的方向赶去。
<br/>　　仔细感应，转过一座山峰，来到一处山谷，元始发现谷中有先天大阵守护，竟然还有不少混沌气流泄露出来。
<br/>　　“轰隆隆！”
<br/>　　只是破除大阵，元始便用了百年时间，让元始不得不感叹，自身的实力，还是有些弱啊。当然，这是因为元始并没有用暴力破阵，而是一步一步，先熟悉阵法，学会阵法，再行破阵。
<br/>　　元始知道，自己阵法知识不够，这是弱点，需要不断学习，此处的先天大阵已然不弱，学会此套阵法，自己就可以知道更多阵法知识。若是以后自己再次遇到比这低级的阵法，自己就可以随手而破。当然，自己以后也可以布置出此等类似的阵法。
<br/>　　破阵，不一定要用暴力破，需要将阵法的作用利用到极致，这才对得起这先天阵法的存在意义。
<br/>　　“嗯？”
<br/>　　进入阵中，元始顿时发现了他熟悉无比的东西——混沌之气！他开始有些疑惑了，因为按理说，此处不应该有这等东西啊。
<br/>　　环境一变，入眼所见，元始好似回到了混沌之中，此处混沌之气纵横，地水火风演变。混沌中心，映入眼帘的是一颗巨大无比的擎天巨树，这棵树正缓缓吸收着混沌之气，无数枝干如同插入虚空深处一般，释放出强大威压，就像擎立于苍茫之巅，仿佛亘古以来便常存于世一般。
<br/>　　只不过，这树主干有一道巨大裂痕，并且，断了最大的一支分支，不知那分支去了何处？此擎天巨树看上去已经快不行了，树叶有些萎靡，好似长久没有得到营养，变得营养不良一般。
<br/>　　“这……这是……混沌灵根？！”元始目光震惊的盯着眼前的巨树，双目之中带有震撼，混沌灵根，这是只能生存在混沌之中的灵根啊。此处怎么会有？元始疑惑不已。
<br/>　　但这些疑惑并不妨碍元始对混沌灵根的渴望，混沌灵根，能够生长在混沌之中的，没有一个是简单的，就像之前元始在混沌之中得到的混沌悟道古茶树一样。
<br/>　　“混沌灵根，竟然能够承受住盘古开天时的巨大破坏而不灭，而且不在混沌之中也能存活到如今，这说明其本身就十分非凡了。不管它是什么，就此来说，这绝对是逆天的机缘。”元始眼中震惊之色犹在，只是不再如之前那般震撼了。
<br/>　　元始可以肯定，这绝非先天之物，而是远超先天灵根的存在，只不过这阵中世界，混沌之气所剩无几，就快要干枯了，使得这混沌灵根开始萎靡，仿佛要死亡一般。
<br/>　　“收！”元始大叫一声。
<br/>　　“嗖！”
<br/>　　顾不得那么多，元始大手一抓，便将这混沌灵根连带着这整片阵中世界泥土、灵水、神材等具皆吸入因果莲台中的混沌空间，与那混沌悟道古茶树种在一起。
<br/>　　接着身形一动，元始自己也闪身进入混沌空间之中。一片灰茫茫的混沌之中，巨树已不复萎靡，好似重新焕发生机一般。无穷的混沌之气鲸吞般的涌入此巨树根系、主干、枝干，好久才平息过来。
<br/>　　元始的因果莲台内的混沌空间本来就不是很大，虽然它能够自行转化混沌之气，但经过这么长时间积累的混沌之气被巨树如此吸收，也快要经不住了。况且，里面还中着混沌悟道古茶树，要是混沌之气没了，又该如何是好。
<br/>　　好在，此混沌灵根好似知道元始的心思一般，吸收到一半就停了下来，留下一半的混沌之气继续供养混沌灵根。
<br/>　　元始虽然心疼，但也没太过在意，混沌空间没了以后可以再找，只要再去混沌一趟就行了。他在意的是这混沌灵根到底是什么品种，根据如今这番表现，元始估计自己此番得到的混沌灵根怕是个了不起的东西啊。
<br/>　　能够如今鲸吞混沌之气的混沌灵根，本身就说明了许多问题。况且，单单就是混沌灵根这一名头，就已是不凡了。
