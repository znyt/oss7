　　叶少阳微微点头，心想这个张道长倒也不是神棍：被鬼纠缠，如果实在解决不了，其实远走他乡是一个办法，不过这里面也是有讲究的，首先一定要是白天，从佛堂、道观，或者衙门这样的地方走，从前门进，一般的鬼魂邪灵，还是不敢进这些地方的，会在门外等待宿主归来。
<br/>　　假如这时候宿主焚香沐浴，或者用灵符、法器之类护身，暂时压制自身的气息，让邪物无法察觉到自己的存在，然后远走他乡，用不着走出千里，只要走出几百里，就算身上的灵符或法器失效，邪物一般也没法再找到宿主的存在。
<br/>　　不过，这种方法治标不治本，毕竟害人的邪物还是存在的，宿主走了，还有可能去找另外的宿主，尤其对叶少阳这样的天师来说，遇到邪物害人，只有两个结果：要么杀了，要么超度，根本不存在这种在他看来很窝囊的办法来躲避邪物。
<br/>　　其次，并非所有邪物都可以用这种方法来逃避，有的邪物修为深厚，或者与宿主之间有某种特殊的连接方式，就算用这种法子也是无法驱逐的。
<br/>　　在叶少阳浮想联翩的时候，刘老头接着说，隔壁的这对夫妻，为了儿子的性命，已经打算离开这里，到南方避难去了，本来打算这两天收拾好了就启程的，之前张道长给了几张灵符，让夫妻俩贴在门上，说是能保证几天太平，没想到今天晚上又成了这样。
<br/>　　说起这些的时候，刘老头唉声叹气，也是希望隔壁家的孩子能够能够躲过这一劫。
<br/>　　叶少阳听完刘老头的讲述，沉吟半晌，请求刘老头天亮之后带自己去隔壁家看看，说不定自己能够搞定这件事。
<br/>　　刘老头一开始觉得不妥，毕竟张道长已经接下了这件事，让他们夫妻带着孩子逃亡，八成能躲过这一劫，如果让他去插手，万一耽误了逃亡，那孩子有什么三长两短，可就什么都来不及了。
<br/>　　“我不耽误他们逃命。”叶少阳十分认真地望着刘老头，说道，“我只是看看，如果能用逃亡这种法子摆脱害人的邪物，我绝不阻拦，如果不行，那就不要白费力气了，还是想办法除了那邪物再说。”
<br/>　　刘老头听他这么说，眼珠子转了转，压低声音说道：“我们这么讨论对付它，不会被它听到吧。你知道……我说的是谁？”
<br/>　　叶少阳笑道：“鬼又不是小偷，还来偷听你说什么，人家注意力都在隔壁，谁来管你。”
<br/>　　刘老头听他这么说，略微放心，说道：“你能看出逃亡有没有用？”
<br/>　　见叶少阳点头，他眉头更加皱起来，“你能看出的事，张道长难道看不出来？不是我说，张道长是我们县里有名的半仙，很有神通的，你看上去不过二十来岁……”
<br/>　　叶少阳知道他是什么意思，摊了摊手说道：“我不知道你说的那个张半仙多大年纪，不过捉鬼降妖这东西，绝不是看年纪大小的。”
<br/>　　刘老头张了张嘴，刚要开口，叶少阳抓着他的胳膊，十分郑重地说道：“大爷，我是认真的，我不是为了钱，也不指望能赚到钱，我只是尽一个法师的责任，我不可拿人家孩子的性命开玩笑。麻烦大爷明天去说一声，我只先看看那孩子再说。”
<br/>　　刘老头盯着他看了半天，最后被他诚恳的眼神打动，点了点头。
<br/>　　之后两人又躺下了，刘老头问起叶少阳的师承来历，叶少阳只说自己是茅山天师，其余不愿多说，刘老头也没再问下去，躺了一会又睡着了。
<br/>　　叶少阳到后半夜才睡着，醒来的时候，天已经亮了，身边刘老头已经不在了，起床出去，一个人没看到，只听见厨房有动静，走过去一看，翠云和刘老太正在一起做饭。
<br/>　　翠云看到叶少阳，让他自己去刷牙洗脸，等着吃饭。叶少阳从行李包袱里找到牙粉鬃刷，去门外水井边洗漱，完事后一转身，看到刘老头站在自己身后不远处，看样子是在等着自己，心中一动，探寻地看过去。
<br/>　　“我去把你的情况跟刘四夫妻说了。”刘老头不等他问，自己先说道，“我说你是我一个远房的亲戚，是茅山名门出身的道士……好说歹说，他们夫妻才同意让你们去看看，待会儿我们吃完饭就过去。”
<br/>　　早饭吃的很朴素：馒头配白粥，就着一些咸菜，叶少阳却是好久没吃到过这么家常的早饭，吃了个大饱。
<br/>　　席间，刘老头跟老伴说了叶少阳要去隔壁刘四家给孩子“看外科”的说，刘老太起初也很担心，翠云虽然吃惊，但他是了解叶少阳在这方面的能力的，当下对刘老太一通劝说，甚至把之前他为秋镇长家中成功驱鬼的经过讲了一遍，刘老太半信半疑，一再嘱咐叶少阳，看看可以，但是千万不要逞能，鬼怪都不是好惹的。
<br/>　　叶少阳唯唯答应，吃完了饭，让翠云留下，陪着老太太干点家务，聊聊天，自己让刘老头带着前往隔壁家。
<br/>　　开门的是个四十多岁的男子，就是出事那个孩子的父亲刘四了，一脸的憔悴，无精打采的，冲刘老头点了点头，目光落在叶少阳身上，上下打量起来。
<br/>　　刘老头立刻拍着叶少阳的肩膀，说道：“这位就是我之前提到的，我家内侄，从小就在茅山出家当道士，这次是去南方办事，顺便来看看我，正好听说了你家儿子的事情，坚持要来看上一眼，没准能有救呢。”
<br/>　　刘四打量着叶少阳，微微皱起眉头，说道：“看上去很年轻啊。”
<br/>　　不等刘老头解释，刘四接着对刘老头说道：“阿公，你也知道，这件事张道长已经帮我处理了，一事不烦二主，我是看你面子才让答应让他来看一眼的哦，先说好，我被这孩子折腾的，可是一分钱都拿不出了。”
<br/>　　叶少阳立刻说道：“你放心吧，我不收钱。”
<br/>　　刘四听他说的这么干脆，脸色缓和下来。
