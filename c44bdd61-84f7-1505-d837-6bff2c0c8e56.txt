　　无论他是哪一种，打过就知道，如今的林凡合体，已他的实力跟力量，对方的元素力量已经无法将他冰封了。

<br/>　　眼看这黄金瞳要被打死，余子童想冲上去阻止，却没有勇气，这林凡成妖人，高三米，浑身怪异，让人心生恐怖。
<br/>　　可要是黄金瞳被打死了，那么他这个一品捉妖师，也没资格在白家混了。

<br/>　　余子童气的浑身发抖，有点恨铁不成钢。
<br/>　　本来是林凡跟涂小安的战斗，硬生生被这只猫搅合了，此时妖人林凡暴虐黄金瞳，往死了打。
<br/>　　恐怕这只猫在劫难逃。
<br/>　　涂小安没出手制止，都说猫有九条命，生命力极为的顽强，很不容易死，这只黄金瞳就更理会了，超强的变异猫种。
<br/>　　性格凶残，暴戾，桀骜不驯，认定的事情，很难改，可谓不死不休。
<br/>　　别看林凡如此暴虐，但涂小安隐晦的发现黄金瞳的生命气息并没有极速的下降。
<br/>　　也就是说，它离死还远着呢。
<br/>　　此时，林凡尾巴抓着黄金瞳吊于自己的面前，就是一顿拳打脚踢，拳拳到肉，丝毫的不留情。
<br/>　　“别打了，在打要死了，还我的猫，我带它立刻离开工地”
<br/>　　余子童在远处喊话求饶，这可是他的命根子呀。
<br/>　　妖人林凡根本不理会余子童，他这种鼠类掌控者，对猫这种动物，天生的反感。

<br/>　　林凡停滞，显然他更为直观的察觉到了怪异之处。
<br/>　　而涂小安眼睛一亮，这猫的气息在暴涨，看样子是要突破呀。
<br/>　　本身19级，在突破，岂不是？
<br/>　　妖人林凡蹙眉，工地开始震动，那只猫已经从他的尾巴上挣脱下来，落到了地面。

<br/>　　好猫呀，神猫呀。
<br/>　　大家都撑着惊骇的目光，看着这只猫的变化。
