　　门琪还想在问些什么，却突然闭上了嘴，反而饶有兴趣的看向门口。
<br/>　　苏重已经慢悠悠的走进了第二场测试的地点，四周颇为空旷，摆着很多简易灶台，看来是临时建造。这场测试是要……做饭？
<br/>　　抬头看着萨次身旁的两个考官，一个苗条纤细，一个肥胖臃肿，这都是魔鬼身材啊。
<br/>　　不理会盯着自己看的三个考官，苏重随意找了棵大树，背靠大树躲在阴凉之下闭目养神。
<br/>　　门琪盯着苏重看了半晌，见苏重呼吸匀称，根本就不理会自己的关注，顿时有些无趣。
<br/>　　左看右看，也没发现有什么奇怪的地方。狐疑的看了萨次一眼，心中满是疑惑。
<br/>　　“萨次，这就是你说的那个特别的学生？我看也没什么特殊的吗？”门琪没有放低声音，甚至有意提高声音。
<br/>　　“切，真无趣。”见苏重一副事不关己专心睡觉的模样，门琪顿时没了兴致。
<br/>　　萨次沉默不语，苏重的外表很有欺骗性。如果不是见过苏重该出手瞬间，他也会被苏重骗过去。
<br/>　　不一会儿，参加考核的人陆陆续续的走进了考场。门琪彻底放弃观察苏重。每年猎人考试，都有几个好苗子，不差苏重一个。
<br/>　　正午时分，考生大部分已经到达，没有到的，大多已经陷在了来路上。好的只是迷路，没及时赶到。坏的，很可能已经丢了命。
<br/>　　“时间到，成功来到这里，你们第一场考核就算过关了。”门琪面无表情，心里有些不耐烦。
<br/>　　“第二场考试内容，料理。做出我们满意的料理，就算你们通过！”门琪冷着脸道，丝毫不拖延。
<br/>　　“什么！料理？”
<br/>　　“有没有搞错？”
<br/>　　“我们是来这里参加考核，是要成为猎人，不是为了当厨子！”
<br/>　　好不容易通过第一场考试的众人，顿时就炸毛了！
<br/>　　嗡！
<br/>　　一股凶悍气势陡然从门琪身上爆发出来，好似滔天骇浪一般，铺天盖地席卷而去！
<br/>　　咔嚓！
<br/>　　离得最近的一个简易灶台，顿时裂成两半。刚才几个叫嚣不已的家伙，首当其冲，顿时双眼一白昏死过去。
<br/>　　“还有谁有意见？”门琪眯着眼睛，周身一片冷然。
<br/>　　“这就是职业猎人的威力吗？”
<br/>　　“只要通过猎人考试，我们也能这么强大吗？”
<br/>　　“不就是做饭吗？有什么大不了的！”
<br/>　　有一部分人眼中带着胆怯，但更多的人，眼中却充满火热。每一个猎人，都是心智强大之辈。一个纤细苗条的美女，仅仅是气势，就让在场众人毫无反抗力。
<br/>　　这种实力，不就是他们所热切盼望的吗？
<br/>　　“说吧，题目是什么！”
<br/>　　扫视着众人一脸热切的样子，门琪眼睛动了动，脸上露出淡淡笑意。这才像点儿样子。
<br/>　　下意识看向苏重，门琪顿时一滞，这是……还在睡觉？！
<br/>　　“题目烤全猪！”门琪咬牙启齿道。
<br/>　　众人浑身打了个颤，这个考官不会是神经病吧，脸怎么说变就变！注意到门琪森冷目光，急忙跑了出去。他们可不想触霉头。
<br/>　　门琪死死的盯着苏重，还睡？这也太不尊重考官了吧，门琪眼睛里几乎要喷出火来。她也不理会其他人，就那么盯着苏重，几乎要忍不住放出身上的念了！
<br/>　　苏重睁开眼，慢悠悠租出考场，实在是被门琪盯的没办法。
<br/>　　如果没记错，这场考试，开始的时候，可没有一个人及格。不然也不会引来尼特罗。
<br/>　　走进森林，找了个开阔处，吹着清风背靠大树，苏重再次闭目开始休息。在和尼特罗交手之前，他要调整好状态。
<br/>　　“这家伙，真不讨人喜，是不是卜哈拉！”门琪阴着脸不忿道。
<br/>　　“是，是，你说是什么就是什么。”卜哈拉一脸无奈，他也不想惹这姑奶奶。
<br/>　　“你说，他不会跑到外面躲起来了吧？”门琪一脸狐疑。
<br/>　　就刚才那个没有干劲的模样，这种可能性很大啊！
<br/>　　心念一动，念力散发而出，圆迅速蔓延向森林。没往外探查多远，就看到了背靠大树休息的苏重。
<br/>　　“这家伙是怎么通过第一场考核的，难道是萨次放水啦！”门琪气的要炸了！
<br/>　　既然参加了考试，就要认真一点儿吗。这种消极模样，是来消遣我的吗？！
<br/>　　“不行，得给他个教训！”门琪一脸阴笑。
<br/>　　“喂喂！门琪，我们是考官，可不能随便出手！”卜哈拉一看门琪诡异表情，就知道门琪要搞事情！
<br/>　　“少啰嗦，我是考官，怎么考核当然我说了算！”门琪打定心思要给苏重个教训。
<br/>　　体内念力凝聚，快速冲出体外，无形气势直奔苏重而去。
<br/>　　嗡！
<br/>　　空气一阵颤抖，四周一片死寂，刚才还虫鸣鸟叫一派热闹场景，此刻却鸦雀无声。
<br/>　　“被我这么一吓，我看你还怎么睡觉！不会做噩梦吧，呦呵呵……”门琪一脸贼兮兮的笑。
<br/>　　然后……就没有然后了，苏重依然背靠大树，呼吸匀称。
<br/>　　门琪一脸懵然，使劲催动力量几下念：“不对呀，念已经发出去了呀？怎么没反应？不行，再试试！”
<br/>　　一股更加庞大的念朝着树林中的苏重压了过去。
<br/>　　咔嚓嚓，苏重背后的树干顿时出现一道裂纹。
<br/>　　“真是麻烦的小姑娘。”苏重睁开眼睛，猩红色宛若岩浆的竖瞳，遥遥瞥了一眼门琪。
<br/>　　轰！
<br/>　　门琪顿时僵硬在原地。冷汗顺着额头不住的往下流。
<br/>　　就在刚才，他似乎看到了一双比人还高的巨大眼珠。猩红瞳孔，漆黑的眼白，只是扫了她一眼，就让门琪如堕冰窟！
<br/>　　冷汗根本就不受控制的流出，一股绝大的恐惧占据心头。可笑的是，她甚至不知道恐惧的源头！
<br/>　　“门琪，你怎么了？”卜哈拉疑惑的看着浑身僵硬的门琪：“那个家伙有问题吗？”
<br/>　　呼！长出一口气，门琪回过神来。她也不确定到底有没有问题。刚才那种恐怖来的太突然，他不确定是不是来自苏重。
<br/>　　不管是不是，这场猎人测试，都不简单。
<br/>　　“没什么，等着他们来吧。”门琪心里乱糟糟，无精打采道。
<br/>　　“是啊，快点儿来吧，我都饿了。”卜哈拉大咧咧，肚子还配合着发出咕咕叫声。
<br/>　　门琪翻了个白眼，相比于她对美味的追求。卜哈拉则是来着不拒。她突然有些羡慕卜哈拉的没心没肺。不然她也就不用疑神疑鬼的担心了。
