　　第502章
<br/>　　程大雷使得是以快打快的法子，连续刺出七剑，一剑更比一剑快，招招都是聂隐娘身上要害。
<br/>　　便是以聂隐娘的本事，此刻也只有招架之力，而无还手之功。不过二人都明白，程大雷这种打法，撑不了长久。一口气泄了，便是聂隐娘攻击的机会。
<br/>　　聂隐娘能挡得住程大雷的剑，程大雷却未必能挡住这姑娘杀人的匕首。
<br/>　　交手也有过几次，程大雷渐渐也摸清楚聂隐娘的套路。这姑娘不知从何处学来的本事，招式并不繁复，只是简单直接，每次出手，都是调动全身力气，将速度和力量发挥到最大。
<br/>　　这种打法，威力自然很大，自古对体力消耗太大。若是以往，倒不算什么问题，因为敌人往往在一招之内，便被割断了喉咙。
<br/>　　可今天的事情出了点意外，聂隐娘被程大雷用龌龊手段拖了这么久。她的速度和力量自然下降许多，而程大雷气力虽然也消耗很大，但却比聂隐娘略强一些。
<br/>　　高手相争，只在方寸之间，程大雷强这一些，或许就能决定最后的胜败。
<br/>　　程大雷硬撑着挡过聂隐娘一轮反击，抽身后退，与聂隐娘拉开距离。
<br/>　　他双目盯着聂隐娘，渐渐已有了自信。聂隐娘的体力已经耗尽，此刻不过是靠一口力气撑着，自己再攻击，胜得便必然是自己，而不是她。
<br/>　　心中有了底气，程大雷对聂隐娘的畏惧大减，整个人也变得从容许多。
<br/>　　“好姑娘，何必动刀动枪，不如从了本大家，做一个压寨夫人，岂不快活。”
<br/>　　聂隐娘趁这空隙恢复体力，倒是没有被程大雷这话影响。她杀过许多恶人，论卑鄙无耻程大雷实在有些不入流。她不过是想多恢复些体力，然后找机会杀了程大雷。
<br/>　　此刻自然不能说必胜无疑，但现在胜负未能决出，输赢可能就在一息之间，最后谁生谁死，现在还说不好。
<br/>　　聂隐娘本不怕死。
<br/>　　程大雷当然不会给她这机会，呵呵冷笑一声，便提剑杀了过来。聂隐娘也握紧匕首，力从地起，气走四肢，最后全部递到手上，要一刃斩杀程大雷。
<br/>　　正在此刻，一阵腥风扑面，程大雷吃了一惊，只见草丛中，一头吊睛白额虎扑了过来。
<br/>　　若是往日，程大雷仗着鬼面斧在手，打杀一头斑斓猛虎也不算什么。然而今天力气也泄了，速度也慢了，手中的兵器也不趁手，一剑刺出去，能不能刺穿虎皮都是个问题。
<br/>　　这猛虎一现身，便向程大雷扑来，程大雷顿时惊出一身白毛汗，那敢有片刻停留，立刻纵身跃起，抓着树枝借力，跃到一棵大树的枝杈上。
<br/>　　那聂隐娘也是吃了一惊，她同样身子跃起，攀在大树上，几下便纵到大树上。
<br/>　　这大虫在林中寻食，被程大雷和聂隐娘的气味吸引，本想要两只人兽饱餐一顿，谁曾想一转眼二人都不见了，围绕着大树转圈，却不曾想二人就在它头顶。
<br/>　　程大雷和聂隐娘长松口气，待看向彼此时，忽地都楞住了。
<br/>　　刚才躲得急，几乎同时跃向大树，却不曾看好位置，此刻才发觉，二人竟躲在大树的同一根树干上。
<br/>　　程大雷立在树梢，身子一起一伏，而聂隐娘则攀在树干位置。
<br/>　　聂隐娘一楞，随即嘴角抿起一丝笑容，开口道：“这不怪我，是天要灭你这恶人，阎王爷面前莫告错了状。”
<br/>　　说着，她亮起掌心的匕首，铁栏杆都能砍断，何况一根树杈。只要一刀砍下，程大雷就得掉下去和老虎玩命。
<br/>　　以他眼下的力气，肯定搏杀不过这斑斓猛虎。
<br/>　　一路千辛万苦，终于能完成任务，杀了程大雷这恶人。不免心中也有些得意，目光挑衅的看了程大雷一眼。
<br/>　　却见程大雷眼中并无多少慌张，眼神中反而流露出一二悲悯，伸出手指了指聂隐娘头顶。
<br/>　　聂隐娘也觉得事情有些不对，下意识抬头往上看了一眼。便是这一眼，吓的她亡魂皆冒，扒开八块顶门骨，一桶冰水往里浇，瞬间浇了个冰透。
<br/>　　只见头顶树冠上盘着一条大蛇，水桶粗细，三四丈长短，通体黑色，张开血盆大口，满口腥气喷得聂隐娘头晕眼花。
<br/>　　聂隐娘啊呀怪叫一声，几乎是下意识沿着树杈逃。
<br/>　　这样的深山老林，几千年都未必有一个人出现，说不得会长出怎样的怪兽。便是聂隐娘从小在山中长大，也没见过这样的场面，一颗心突突直跳，好险要从胸腔内跃出。
<br/>　　幸运的是，那怪蛇似乎没有注意到她，或者不屑她的存在，沿着树干盘旋直下，扑向了树下的斑斓猛虎。
<br/>　　聂隐娘三魂七魄，此刻还不知存了几魂几魄，从鬼门关逃回一条命来，三魂七魄却丢在了地狱黄泉。
<br/>　　好不容易回过神来，却看到面前杵着一张丑脸。聂隐娘险些又叫出声，刚才她慌不择路，正逃向程大雷，此刻二人站在一根树枝上，相距不过三步远。
<br/>　　也就是说二人同时伸出手臂，能拉到手的距离。
<br/>　　“姑娘，你可不能再往前了，不然咱们都得掉下去喂老虎。”程大雷低声道。
<br/>　　程大雷身子沉重，这树杈容他一人便已经勉强，只是靠轻身功夫撑着。现在聂隐娘又跑过来，一根细弱的树干摇摇欲坠，耳边似乎能听见咔咔擦擦声响。
<br/>　　程大雷握住头顶一截树枝，聂隐娘抓住两侧的树枝，勉强能稳住身形。二人如今距离颇近，可是谁也不敢轻举妄动，在这样的树枝上，稍微动作大些，脚下的树枝折断，二人都得丧命。
<br/>　　大树下，那怪蛇和大虫已经斗在一起，两头猛兽真不知在山中活了多久，各自称霸一方，今天是好不容易碰上，都要灭掉对方，独霸大山。
<br/>　　树顶上，聂隐娘也不知为何会落到如此局面，他恨不得立刻杀了程大雷，但却又无可奈何要和程大雷和平共处。
<br/>　　程大雷心情倒是平静的很，视线肆无忌惮的打量着聂隐娘，一副无赖模样。
<br/>　　“好姑娘，不若咱们好好聊聊。”
<br/>(本章完)
