        &lt;h3&gt;第三卷第二百九十三章  姊妹情&lt;/h3&gt;
<br/>
<br/>        陈子昂坐在车上，看着聊天记录，深深的叹了口气，刚刚从陈子悦的家里出来，事情是一件又一件，赶着她逼着她不得放松。
<br/>
<br/>        陈子悦哪里是想让她去见蔷悦，她到了陈子悦的家里，和蔷悦没有说上三句话，陈子悦就将蔷悦和楠悦赶跑了。
<br/>
<br/>        “你想和我说什么？”陈子昂知道五姐有话给对她说。
<br/>
<br/>        “闫欢去甸城了。”陈子悦忧虑的说。
<br/>
<br/>        “我知道啊，你们给我说过。”
<br/>
<br/>        “我感觉闫欢有些反常，你不能不能不要去甸城了？”陈子悦现在也不知道闫欢是什么情况，她就是觉得闫欢不正常。
<br/>
<br/>        “闫欢怎么不正常了？”陈子昂调查过闫欢，知道了闫欢的过往，闫欢往甸城跑她不反对也不支持，但是她却不能告诉陈子悦半个字。
<br/>
<br/>        “我有一天晚上给她打电话，发现她好像在室外，她的声音很低，但是她却告诉我，她在家里。我听到了风声。”陈子悦很担心，毕竟她和闫欢在一起共事了那么久。
<br/>
<br/>        “你在担心什么？也许她的窗户开着啊，也许她刚刚从外面回家啊。闫欢比你聪明了很多，能力也比你强了很多，她找项目投资，前期肯定要调研市场，一个女人干事业是很辛苦的，她累了，不想大声说话也很正常，再加上晚上了，谁还大声的喊叫？”陈子昂听了陈子悦的话，便找理由来安慰陈子悦。
<br/>
<br/>        但是陈子昂在心里开始担忧闫欢的安危了，闫欢这是开始行动了？她一个人如何行动？利用秦少卿和闫芈的关系进行报复？
<br/>
<br/>        陈子昂查到了闫欢和张函以及陆玉的关系，查到了陆玉将闫欢扔到了甸城，查到了张函将闫欢带回来了，却没有查到闫欢在甸城的那一段时间都干了些什么，那几年的闫欢不叫闫欢，她也没有查到闫欢和谷强曾经相识。
<br/>
<br/>        在她的潜意识里，闫欢是想报复陆玉。
<br/>
<br/>        “你这样的解释也对。”陈子悦自动过滤掉妹妹说她不聪明的话，心里的担忧减轻了一点。
<br/>
<br/>        “你现在无需担心什么，好好的将生意做好，别离了闫欢你什么都干不了，闫欢在甸城投资是需要这边给她足够的资本的，没有资金她前期所做的一切都是白搭，与其你在这里白白的浪费精力担惊受怕不如安心的将生意做大做强，好给她足够的支持，好朋友就是在需要的时候出手，支持她也是支持你自己。”陈子昂开始给陈子悦讲道理了。
<br/>
<br/>        陈子悦什么都好，就是优柔寡断，爱胡思乱想，但是只要你给她一线希望或者是信念她会毫不犹豫的往前冲。
<br/>
<br/>        陈子昂并不希望她的亲人们的生活受到任何影响，她希望他们都平平安安，开开心心的生活，尤其是希望五姐更能够此生安宁，她以前过得太苦了，无可否认，闫欢对陈子悦的影响是巨大的，陈子悦现在的成就有一半的功劳都是闫欢的，闫欢也算是一个商业奇才，有明锐的市场嗅觉和判断力，她投资的地方都没有失败过，而且超市都是人流集中的地方，尤其是西城现在这个商圈，更是一个传说，将西城之前几个老的，比喻某井某元商场都给比下去了，最可怕的是到了节假日人流量能达到十万次。
<br/>
<br/>        陈子昂心里对闫欢有一种怜惜，这样的一个人才就埋没了，似乎她和闫欢有着不同的遭遇却有着相同的命运。
<br/>
<br/>        成功的劝解了陈子悦，她还想去见见陈子乔，问问四姐这几年的情况，然后再去看看三姐，这一次离开西城，回北京后她便要踏上去甸城的路途，前路漫漫，她也不知道后期会是什么样的状态，她想走一步算百步，可是后期的事情也不会都按照她的意愿来进行，谁也不知道会不会出现意外。
<br/>
<br/>        刚刚又约了江俞轩和凌辰，她这时间排的满满的。
<br/>
<br/>        陈子昂驱车到了陈子乔的家里，陈子乔也在西城新区，独栋的别墅，常萧林和孩子都在家里，陈子昂敲响陈子乔的家门。
<br/>
<br/>        “子昂？”陈子乔因为父亲手术请假了，几个人在医院刚刚见面，陈子乔没有想到陈子昂会来她家。
<br/>
<br/>        陈家姊妹都比较聪明，陈子乔知道陈子昂找她有事情，她们姊妹的相处关系很奇怪，陈子昂到处跑，回来就是和大家一起聚聚，很少单独去谁的家里，就连电话打得也不多，因为忙。
<br/>
<br/>        常萧林见陈子昂来了，陪着陈子昂坐了一会，然后对陈子昂说：“子昂，我带着孩子去趟超市，你和你姐聊，晚上就在家里吃饭。”
<br/>
<br/>        “姐夫，晚上我已经约了人，我坐一会就走，你忙你的去吧。”陈子昂微笑着对常萧林说。
<br/>
<br/>        常萧林走了，她们姊妹才好说话。
<br/>
<br/>        “子昂，你是不是有事情要问我？”陈子乔等常萧林一走便直接问陈子昂。
<br/>
<br/>        “嗯，这么多年，我只想知道你陷得有多深。”陈子昂看着她的四姐，让陈子乔无可遁形。
<br/>
<br/>        “我和陆二有关系，和老马也有关系，不过你姐夫都不知道。”陈子乔满脸羞愧。
<br/>
<br/>        “通过你手中输出去了多少资源？”陈子昂紧追不舍。
<br/>
<br/>        陈子乔听到陈子昂的话，手中正在削水果的刀不小心将手指割破了，鲜血滴答滴答的落下来，陈子昂并没有动，她只是紧紧的盯着陈子乔。
<br/>
<br/>        陈子乔任鲜血滴落，她忍着疼将手中的苹果削完，并没有递给陈子昂，而是自己狠狠的咬了一口，慢慢的嚼着，过了很久，她说：“很多，但是我并没有拿一分一毫。你姐夫的生意只是沾了我们的光，也没有实质性的来往。”
<br/>
<br/>        陈子昂松了一口气：“我有孩子的事情陆家人知道了，陆家的陆宇霆曾经是我的朋友，他带话给我，让我注意陆家，他的妹妹和父母亲都不会让我好过的，所以四姐，为了我自己，我和陆家是对立的，至于你，你自己选择。有一句话我想告诉你，得道多助失道寡助，陆家并不是你看到的那么光鲜亮丽。”
<br/>
<br/>        “子昂，我知道了，我们是血浓于水的亲人，只有你们才是我的想要守护的人，别人再好与我何干，无非就是利用而已。”
<br/>
<br/>        “想办法从你现在的位置调离吧，你呆的时间够长的了。”
<br/>
<br/>        “我怕离开不了。”
<br/>
<br/>        “找机会，过一段时间就有机会，你自己看好机会。”陈子昂并没有告诉陈子乔后期该怎么做。
<br/>
<br/>        “子昂，西城公司是不是进展缓慢？不好开展业务？我都帮不了你什么。”陈子乔有些难过，洛城的项目她以为她给马总说好了，她以为万无一失，可是马总还是心向夏颜。
<br/>
<br/>        “做生意那能一帆风顺，磕磕碰碰是常有的事情，别想着你能帮我，你只要好好的我也就无忧了。”陈子昂能理解陈子乔此时的心里。
<br/>
<br/>        “子昂.......”
<br/>
<br/>        “我该走了，回来一直都没有去看三姐，我要去见见三姐，我有可能后天就走了，我说的话你要记得。”陈子昂站起来告辞。
<br/>
<br/>        前路漫漫，我们都是血浓如水的亲人，于陈子昂来说，她的心既大也小，大，她能考虑到大义，小，她容不得她的亲人们有任何闪失，只要他们平平安安就好。
