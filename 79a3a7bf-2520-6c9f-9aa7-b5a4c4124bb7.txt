夏夕绾听说了夏小蝶暴打孔真儿的事情，这对塑料姐妹花还真是经不起一点的考验，不管是她们中的任何一个都没有让她失望。
<br/>
<br/>        夏夕绾对孔真儿没有丝毫的同情，孔真儿原本可以将自己活得有尊严些，但是她内心贪慕虚荣，心眼又坏。
<br/>
<br/>        夏夕绾打算去看一下孔真儿，但是这时苏希走了过来，“绾绾，我终于找到你了。”
<br/>
<br/>        “苏希，你找我干什么？”
<br/>
<br/>        苏希去拉夏夕绾的小手，“走，我带你去看一个人。”
<br/>
<br/>        夏夕绾敏捷的避开了他的拉扯，往后退了一步，“苏希，看你这么迫不及待的样子好像是等着看我什么笑话，你不要碰我，我跟你去看。”
<br/>
<br/>        苏希今天心情很好，就没有跟夏夕绾计较，“那就走吧。”
<br/>
<br/>        夏夕绾跟了上去。
<br/>
<br/>        两个人来到了一个豪华包厢门口，苏希勾唇，“绾绾，你自己看吧。”
<br/>
<br/>        夏夕绾看向包厢里面，里面一股烟味，都是海城的显贵们，几个男人在打牌，其中有一道熟悉的英挺身影，是陆寒霆。
<br/>
<br/>        他两天没有回去了，在这个酒吧里打牌。
<br/>
<br/>        豪华包厢里有几个穿着清凉的美女，一看就是1949的头牌，十分的漂亮。
<br/>
<br/>        陆寒霆身边也有两个美人，可谓左拥右抱。
<br/>
<br/>        夏夕绾感觉一根针刺进了她的心里，疼的不是太明显的，但是密密麻麻的，让她眼眶泛红。
<br/>
<br/>        夏夕绾跑远了几步，不想再去看。
<br/>
<br/>        这时苏希跟了上来，他就是想看夏夕绾伤心失落的样子，可是这一刻来临了他却感觉无比的愤怒，“绾绾，你喜欢上他了？”
<br/>
<br/>        夏夕绾扭头看着苏希，“是，我就是喜欢上他了，这跟你有什么关系？”
<br/>
<br/>        “绾绾，你也看到了，他不过就是玩玩你的，像他这样出身的男人，根本就不缺女人，他现在觉得你新鲜，等把你追到手了他就觉得这股新鲜劲过了，到时就将你抛之脑后了，这种男人有什么值得你喜欢的？”
<br/>
<br/>        夏夕绾白皙的眼眶泛着红，她知道他在外面有女人，可是她就是喜欢上他了能怎么办？
<br/>
<br/>        “苏希，你看到我的笑话了，这下你满意了，满意了你就可以滚了，我就是喜欢他不喜欢你！”
<br/>
<br/>        苏希一张俊脸迅速阴沉了下去，他冷笑一声，“夏夕绾，我真不该将自己送上门让你羞辱，你就继续执迷不悟下去吧，过两天是我跟小蝶的订婚party，你一定要来！”
<br/>
<br/>        苏希愤然离开。
<br/>
<br/>        夏夕绾贴着墙壁，纤长的羽捷垂落下来，原来这就是心痛的感觉，陆寒霆那个男人让她心很痛。
<br/>
<br/>        这时酒吧经理跑了过来，不由分说的将一杯好酒塞到了夏夕绾的怀里，“你还愣着干什么，豪华包厢里等着你送酒呢，这个包厢里可都是海城的风云人物，你给我机灵点，小心伺候。”
<br/>
<br/>        酒吧经理直接将夏夕绾推进了豪华包厢里。
<br/>
<br/>        夏夕绾懵了，她不是送酒的，她想开口解释的，但是来不及了，人已经被推进去了。
<br/>
<br/>        门口闹出的声响吸引了那一桌打牌的男人，其中霍西泽抬头看到夏夕绾时双眼都亮了，还开口提醒对面的男人，“二哥，你看谁来了？”
