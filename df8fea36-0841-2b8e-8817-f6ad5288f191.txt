
<br/>　　<br/>　　    “行了，让你的人都上来吧”王冬至跟工头说道：“麻烦您跟他们交代一声，出去以后嘴严一些，这里的事不要说出去，也最好不要乱打听”<br/>　　<br/>　　    工头笑道：“冯老板交代了，一切听你们的指挥，这一点您可以放心，我们就是拿钱干活，钱给了其他的问题一概不会多管”<br/>　　<br/>　　    “好，谢谢了”王冬至从身上拿出一包纸递了过去，工头打开一看，发现里面是一捧纸灰。<br/>　　<br/>　　    “这些纸灰你惨到水里，给刚才下去的那些人喝了”<br/>　　<br/>　　    工头顿时愣了，感觉头皮有点发麻，本来他们就在坟地这种地方干活，就怪吓人的了。<br/>　　<br/>　　    王冬至解释道：“我家这片坟地阴气较重，呆的时间久了可能会伤身，这纸灰以前是我从一道士手中求来的，可以辟邪，你也不用担心，就是以防万一罢了”<br/>　　<br/>　　    “好叻，好叻，我懂”干工程的人，对牛鬼蛇神这种事什么的还是有些见地的，长期施工的人都会接触到工地布置风水，和干活的时候碰见邪事的可能，所以比较有抵抗力。<br/>　　<br/>　　    坑下的工人全都开始上来，工头组织人马上离开后山，当人全都离去之后，王冬至利索的把头发扎上，翻身就跳入了坑中，毫不忌讳的蹲下身子用手扒拉着坑底的泥土，片刻后，一块青色厚重的大石从底部漏了出来，没过多久整块石头的形状全都被王冬至给扒了出来，石头上的一整面都刻着繁杂古朴的纹路，纵横交错着。<br/>　　<br/>　　    一股股的阴风透过石头渗透而出，王冬至感觉触手就有一种刺骨的凉意，凉的人从头到脚都掉了一地的鸡皮疙瘩。<br/>　　<br/>　　    “嗷······”突然，隐约间从青石的背后，似乎有阵阵鬼哭狼嚎呼啸着的声音传了过来。<br/>　　<br/>　　    仿佛，石头的背后存在着另外一个世界。<br/>　　<br/>　　    王冬至哀声说道：“家里养着个讨债冤家，真是让人头疼呢！”<br/>　　<br/>　　    一夜过去之后，隔天就是七月十四，这天也被称为中元鬼节，传说此日夜间子时直到黎明前夕，各地鬼门全会打开，世间到处都是游荡的孤魂野鬼，家中老人经常告诉小辈，这天晚上老实在家呆着不要出去乱逛否则容易碰到脏东西。<br/>　　<br/>　　    老辈人的这种说话并不是吓唬人和危言耸听的，因为全年里就这一天的阴气是最重的，阴间通往阳间的几道门户全都大开，铺天盖地的阴气会蜂拥而出，所以这时候阳间的阴气比往常都要重得多了。<br/>　　<br/>　　    特别是医院，坟地，殡仪馆这种本来就聚阴的地方，在这一天有多远就得离多远了。<br/>　　<br/>　　    鬼节那一天，这时候哪怕就是阳气重的人如果在外，也会被压的低一头，很多人就容易被脏东西给缠上了，明智的选择就是老老实实的在家呆着，或者留在人多的地方。<br/>　　<br/>　　    有人可能会问，既然鬼节那天的阴气重，那外出或者在家的状况不应该是一样的么？<br/>　　<br/>　　    其实则不然，有句话叫家国天下，家这个字排到最前面是为啥呢，那是因为这是生你养你和你生活的地方，家中已经被你们一家人的磁场全都给覆盖住了，一般清情况下孤魂野鬼是不会擅自闯到你家中的。<br/>　　<br/>　　    就像北方或者南方，都会供奉保家仙其实也是这个意思。<br/>　　<br/>　　    经常听说，谁买了房子后放在那里三五年或者更长的时间都没去住，就这种房子通常比较容易招东西，说白了就是家中没有了人气，自然很轻易的就能被东西给占了。<br/>　　<br/>　　    所以，去常年没人住的房子，要么带条狗要么抱着小孩子过去，听见狗吠或者小孩哭闹，那就得想想办法了。<br/>　　<br/>　　    如果有狗吠或者小孩哭闹的话，还有个很邪门的测验方式，就是准备一碗白米，米上插一根香然后点燃，如果香烧没了后白米变黑了，那你就得找人来做法驱邪了，家中的东西不好对付，不是一般的孤魂野会，这是会要你命的。<br/>　　<br/>　　    这种方式叫当面上香，米中插香就跟给死人上坟是一个道理，小时候老人总说那些顽皮的小孩子，不要把筷子竖在饭碗里。<br/>　　<br/>　　    你当面上香的话，米变黑了，就只有一种可能，家里的东西不想走，你若是留下来它就会要了你命的。<br/>　　<br/>　　    所以，中原鬼节的时候，不要乱跑，你别不信邪，沾上了就不只是倒霉了。<br/>　　<br/>　　    良水城，良水河上，今日的情形有点怪，很多船家白天的时候就被人把船给定了，说是要夜游良水河，给的佣金基本上等于平时半个多月的收入了，这还不是一条两条而是至少有十几条船全都被定走了。<br/>　　<br/>　　    晚上八九点钟的时候，良水河两岸就零散的出现了不少人，有的上了渔船，有的漫步在岸边，他们在都在等待着子时的到来。<br/>　　<br/>　　    子时，是一天中阴气最重的时候，也是鬼节门户大开的时间。<br/>　　<br/>　　    茅小草和王惊蛰今晚都没睡，沙发两头一边坐了一个，百无聊赖的看着电视。<br/>　　<br/>　　    隔壁，陈三岁坐在床边，伸手抚摸着床上女尸的脸颊，轻声说道：“老婆，今晚我暂时不能陪着你了，不过过了今天我晚上我们就能团聚了，我要去一趟阴间将你的三魂七魄带回来……无论如何我都要带回来，谁都不能拦着”<br/>　　<br/>　　    陈三岁说这番话的时候，语气相当笃定毋庸置疑了。<br/>　　<br/>　　    “嘎吱”推开房门，穿着一身黑衣的陈三岁快速走出家门，然后消失在了黑夜里，直奔良水河而去。<br/>　　<br/>　　    同时，看着电视的王惊蛰抻着脖子看向窗外的夜空，说道：“这两年的中元节已经太平不少了，自从阴间的门户被关上，妖魔鬼怪什么的都少见多了”<br/>　　<br/>　　    “还有半个小时，就是子时了”茅小草抬起胳膊看了下表说道。<br/>　　<br/>　　    王惊蛰叹了口气，说道：“咱俩啊，都不是安分的······”
