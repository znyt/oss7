　　楚风眠的实力，已经是凌驾于大部分半步天人境强者之上。
<br/>　　虽然楚风眠的境界还只是九劫古帝的境界，但是楚风眠的实力，却是比起寻常的半步天人境强者要强横的多。
<br/>　　对付寻常的半步天人境强者，像是法理古帝，凌山这个级别的人物，楚风眠足以轻易将他们击败，甚至将他们击杀。
<br/>　　也只有少帝，荒天王那等，得到过惊天奇遇的半步天人境强者，才可以跟楚风眠平起平坐。
<br/>　　在凝炼了不朽真身之后，楚风眠的一身实力也强横了不少。
<br/>　　更何况，现在这些半步天人境的老古董，都是落入到了天星河的掌控之中。
<br/>　　以天星河的力量，足以是将这些半步天人境老古董的力量死死压制住，楚风眠才可以挥手随意斩杀他们。
<br/>　　不过这主要还是天星河的功劳。
<br/>　　如果是在外面的话，也许楚风眠还要付出一些手段，击败容易，但是击杀可难。
<br/>　　毕竟所有半步天人境强者，几乎都有过惊天的奇遇，逃命的手段不少，楚风眠也未必可以轰杀他们。
<br/>　　楚风眠的实力，还做不到碾压一切的级别。
<br/>　　不过现在因为这些圣龙秘境老古董，都已经是被困在了天星河之中，在多的手段也难以从其中逃出生天了，楚风眠杀起他们来，自然轻松无比。
<br/>　　“你们三个，是愿意跟他们一样被陨落，还是臣服于我，自己选吧，愿意臣服的人，现在就跪下，敞开道心。”
<br/>　　楚风眠目光又看向了仅剩下的三名半步天人境强者。
<br/>　　万神龙，霸斗元，炎乾龙。
<br/>　　楚风眠最大的仇敌，还是圣院宗，六绝宗，现在这两宗的强者，已经是尽数死在了楚风眠的手中。
<br/>　　剩下这三人，楚风眠倒是可以给他们一个机会，毕竟未来圣龙秘境跟剑道门，最好都要有着半步天人境强者镇守才行。
<br/>　　“我们身为半步天人境强者！已经不是凡人，绝不可能臣服，楚风眠，我可以为你效力，不过却并不是当你的奴隶，而是只为你做事，帮你管理圣龙秘境，如何？”
<br/>　　迟疑了一会，那霸斗元先开口道。
<br/>　　他毕竟可是一尊半步天人境强者，地位超然，一旦是踏入了半步天人境，便不是凡人了，而是天人，虽然是半步天人，可也是接近成仙的人物了。
<br/>　　让他现在臣服在楚风眠的麾下，跪在一名九劫古帝的面前，更是让他无法接受的。
<br/>　　“不错，楚风眠，你只要愿意平等对待我们，我们可以为你效力，让你成为圣龙秘境真正的主人。”
<br/>　　听到霸斗元开口，那万神龙也开口道。
<br/>　　“而且我要那你参悟出的始祖九式的奥秘，你只要将这始祖九式的奥秘告诉我，我就可以全心全意为你做事。”
<br/>　　“想要始祖九式的奥秘？平等对待？”
<br/>　　楚风眠看向这开口道二人，似笑非笑的开口道，眼神之中似乎是在考虑。
<br/>　　“不错，这就是我们的条件，楚风眠，你也应该知道，圣龙秘境在大罗天星域之中，本就是被虎视眈眈，三大势力，一直想要灭了圣龙秘境。”
<br/>　　那霸斗元，看着楚风眠似乎是在犹豫，更是继续开口道。
<br/>　　“你的实力虽然强横，可就单单你一个人，根本不可能是三大势力的对手，你杀了我们，未来圣龙秘境也会毁灭。”
<br/>　　“继续说。”
<br/>　　楚风眠沉声说道。
<br/>　　看到楚风眠的表情，那霸斗元心中一喜，觉得已经是抓住到了楚风眠的弱点，继续说道。
<br/>　　“你现在只要是放了我们三人，跟我们平起平坐，日后三大势力来攻打圣龙秘境，我们也会出力，不过有一点，你杀了凌山，韩圣他们，他们身上的财富，你必须要平分给我们，这才公平。”
<br/>　　说道这里，霸斗元眼神之中更是露出了几分贪婪之色。
<br/>　　“平分给你们财富？还要平起平坐，看来你们还不明白你们自己的身份。”
<br/>　　楚风眠听完这霸斗元的话，冷笑连连说道。
<br/>　　“我说过了，不臣服便是死，你们还敢在这里跟我提条件？三大势力攻打圣龙秘境又如何？他们不来攻打圣龙秘境，我也打算灭了这三方势力，根本不需要你们几个废物相助！”
<br/>　　“你们两个，死！”
<br/>　　楚风眠开口的同时，指尖一点，又是两道剑气，向着霸斗元，万神龙斩杀而去。
<br/>　　“你……！”
<br/>　　那霸斗元，万神龙都没想到，楚风眠居然是丝毫不理他们提出的条件。
<br/>　　他们可是半步天人境强者啊，不管是放在哪一方势力之中，都是要受到礼待的强者。
<br/>　　哪怕是他们现在立刻去投身其他三大势力，其他三大势力都会将他们直接奉为上宾，甚至直接让他们都成为高层。
<br/>　　可现在在楚风眠的面前，半步天人境的强者，也没有太大的意义。
<br/>　　乖乖臣服，楚风眠还可以绕过他们一命，这已经是对他们莫大的恩赐了。
<br/>　　至于还敢来跟楚风眠提条件，那就是只有死路一条。
<br/>　　砰！
<br/>　　剑光闪过，这万神龙，霸斗元也被无情的斩杀。
<br/>　　眼前在天星河之中，只剩下了五灵宗的炎乾龙。
<br/>　　“你刚刚为什么没有说话？”
<br/>　　楚风眠看向炎乾龙，饶有兴趣的说道。
<br/>　　刚刚万神龙，霸斗元可都跟楚风眠来提条件，但是在炎乾龙，却是根本没有开口，而是安静的站在了一旁。
<br/>　　“我知道，我没有跟你谈条件的资本，我愿意臣服。”
<br/>　　炎乾龙看向楚风眠，一下子跪倒在了楚风眠的面前，主动敞开道心。
<br/>　　一旦是敞开道心，这炎乾龙在楚风眠的面前，便是再也没有任何的秘密了。
<br/>　　而且一旦是楚风眠在他的道心之中留下禁制，那这炎乾龙日后的生死，都会由楚风眠所控制。
<br/>　　不过比起死，炎乾龙更想要活下去。
<br/>　　“很好，看来圣龙秘境里还是有聪明人的，五灵宗的人，果然都识时务。”
<br/>　　楚风眠看到炎乾龙的表现，点了点头。
<br/>　　在圣龙秘境之中，跟楚风眠关系还算好的，也就是五灵宗了。
<br/>　　这炎乾龙也是很识时务，主动跪下敞开了道心。
