　　在某一个瞬间，大祚荣几乎产生了投降的冲动，但考虑到后果，他还是没有这样做。
<br/>　　而是转过头，看着那些跟着他出来的靺鞨骑兵：“靺鞨的英雄们，告诉我，你们怕不怕？”
<br/>　　“呼哈！呼哈！”身处绝境，靺鞨骑兵的凶性也被激发出来，双目尽赤的盯着前方高地影影绰绰的那些人影，发出类似于咆哮的战吼！
<br/>　　“好样的，勇士们，让我们杀出一条路来，给那些只知道玩心机的唐人一点颜色看看。”大祚荣心中热血也被手下士兵的战吼引燃，举起了手里的弯刀。
<br/>　　“呼哈！”战吼声在这一刻达到顶点。
<br/>　　“杀……！”
<br/>　　“嗷……！”
<br/>　　洼地中的靺鞨骑兵终于拿出了属于靺鞨人的无谓，催赶着战马迎向他们的敌人。
<br/>　　……
<br/>　　副将此时的心已经彻底放下了，围住了五万靺鞨骑兵，足以把整个辽东战局扳向另一个方向，只要把这五万人打散了，余下的五万人根本不足为虑，有很多办法可以慢慢拖死他们。
<br/>　　“床弩准备、连弩准备、火箭弹、铁菠萝、洧水准备。”看着冲上来的靺鞨骑兵，副将的嘴角微微挑起。
<br/>　　都是打老了仗的人，有岂会惧怕所谓的决死一击。副将别看昨天在黑齿常之面前棋差一着，但如果没有几分本事，又怎么可能混到副将的位置。
<br/>　　靺鞨骑兵悍不畏死，可是大唐骑兵的骄傲也让他们同样的无所畏惧。
<br/>　　迎着冲上来的靺鞨骑兵，大唐骑兵开始缓缓向前，一架架重型武器投入战斗准备。
<br/>　　“放！放！放！”“嘭，嘭，嘭”
<br/>　　简易的小型投石车将一罐又一罐洧水砸向敌人。
<br/>　　“火箭弹，发射！”“咻，咻，咻”
<br/>　　一发又一发闪着火光的火箭弹飞向敌人。
<br/>　　“床弩，攻击！”“咔，咔，咔”
<br/>　　机括声连响，手指粗细，三尺余长的三棱破甲箭以一个呼吸三发的速度凌空扑向敌人。
<br/>　　“钢弩准备，攻击”“嗡，嗡，嗡”
<br/>　　弩弦连响，密密麻麻，黑压压一片三棱矢破空而至。
<br/>　　短短的一瞬间，大唐骑兵所有全程武器全部暴发出最强大的攻击，将靺鞨骑兵的攻势一下子压了下去。
<br/>　　一道火墙在战场上燃烧，大火中无数人在痛苦的挣扎，发出撕心裂肺的惨叫声，但快声音就沉寂了下去，大火中的人影也倒在了水中。
<br/>　　战马因为受到了惊吓，发出一阵阵的嘶鸣，变的不受控制。
<br/>　　最要命的还是那些破空而来的三棱矢，这些弩矢连续不断的射过来，让靺鞨骑兵来喘口气的功夫都没有。
<br/>　　类似于盾牌的重装备都在来时的路上丢掉了，现在就算是想找都找不到。
<br/>　　而且因为地面积水的关系，战马的速度根本提不起来，又或者速度刚刚提起来，立刻就会掉进前面不知什么时候就会出一的大坑里面。
<br/>　　……
<br/>　　短短片刻时间，靺鞨骑兵损失惨重，就连大祚荣也被一支三棱矢射穿了胳膊。
<br/>　　“首领，我们冲不上去的，撤兵吧！”伯咄部族长蹲在一匹倒地的战马后面，措着大祚荣包扎伤口的时间喊道。
<br/>　　“前有阻拦，后有追兵，你想往什么地方撤！跳河自杀么！”大祚荣怒不可遏的吼道。
<br/>　　他并不是不想撤，而是真的撤不出去。
<br/>　　“马会游泳，我们可以留下一批人断后，其他人骑在马上过河，其他不重要的东西都不要了。”伯咄部族长喊道，显然这一切都是他刚刚已经算计好了的。
<br/>　　“那谁来断后，你吗？！”大祚荣吼了回去。
<br/>　　……
<br/>　　战斗还在继续，前面五千大唐骑兵的意志很坚决，那就是绝对不让靺鞨骑兵越过自己的防线，不管冲上来多少人，都会被他们的以手中的连弩击退。
<br/>　　靺鞨骑兵的身后，已经可以看到另一支截断他们后路的大唐骑兵的影子，很显然，他们是在两面包夹，打算将这五万人彻底留下。
<br/>　　就在这个关键的时刻，一个声音突然说道：“我黑水部原意留下断后！”
<br/>　　“什么？”大祚荣愣了一下。
<br/>　　眼下这个情况谁都知道，留下就是死，离开就是生，为什么黑水部会选择留下？
<br/>　　“首领，别犹豫了，撤吧！靺鞨不能断送在我们手里，撤回玄菟城，撤回大山里面，外面的世界不是我们能够接触的，大唐已经强大到不可力敌的程度，回去之后递交降书，归附了吧！”黑水部族长已经有了必死的决心，一些以前想说而不敢说的话也被他说了出来。
<br/>　　大祚荣没有说话，只是静静的看着黑水部族长，仿佛要看穿他的内心。
<br/>　　而就在此时，黑水部族长又说话了：“首领，黑水是我们靺鞨的发源地，所以黑水部绝不会背叛靺鞨，请你相信我！”
<br/>　　话说到这个份上，大祚荣还能说什么，除了深吸了一口洧水燃烧之后有些浑浊空气，重重在黑水部族长的肩膀上拍了一下：“保重，兄弟，大军渡河之后，如果有机会就过来，如果没有机会……那就投降吧，留得青山在，不怕没柴烧，靺鞨七部乃是一体，若是少了一部还叫什么靺鞨！”
<br/>　　“我明白，首领，时间紧急，快走吧！”黑水部族长不置可否的点点头，对着大祚荣施了一礼算是告别，随后起身一声大吼：“黑水部的勇士们，跟着我，杀……！”
<br/>　　“杀！”部族的人都是单纯的，没有那么多的心思，族长的命令就是一切，有族长带着他们，他们便无所畏惧。
<br/>　　……
<br/>　　“撤吧！”大祚荣看着黑水部族长远去的背影消失在浓烟之中，声音有些沉重。
<br/>　　“撤……伯咄部、白山部、粟末部、安车骨部……，丢掉所有的一切，向河边撤，快，快点！”伯咄部族长见大祚荣答应撤兵，立刻开始组织手人撤出战斗。
<br/>　　于是大批大批的靺鞨骑兵开始脱离战场，向着已经恢复平静的河边涌了过去。
<br/>　　一面是战马嘶鸣，杀声震天；一面是惶惶如丧家之犬的万人竞渡。
<br/>　　原本一场针对唐军的埋伏打成这样，大大出乎大祚荣的意料之外。
