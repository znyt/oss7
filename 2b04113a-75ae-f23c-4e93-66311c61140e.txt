　　“人皇罐碎片，每得到一块，人皇罐的力量都会成倍增长！”
<br/>　　楚风眠心中大喜，这人皇罐碎片，他如今手中已经得到三块了。
<br/>　　若是可以在得到这一块，这人皇罐碎片的数量，将会达到四块，接近半数。
<br/>　　除非是有人，将剩下的五块人皇罐碎片全部得到，才有可能从楚风眠的手中，得到人皇罐。
<br/>　　不然的话，这人皇罐只怕是除非楚风眠陨落，才会被其他人夺走。
<br/>　　这人皇罐碎片，想象得到一片都是极难的事情，剩下的五块若是被一个人得到，几乎是不可能是一件事。
<br/>　　“这一枚人皇罐碎片，我必须要得到！”
<br/>　　楚风眠心中想道，他现在财富无限，不管要花多少代价，也要得到这人皇罐碎片。
<br/>　　“楚公子看来对着极道仙兵残片，似乎有着兴趣？”
<br/>　　楚风眠脸上闪过的惊讶，李长老自然看到眼中。
<br/>　　不过李长老也并不意外楚风眠的惊讶，极道仙兵，这可是传说中的东西。
<br/>　　就算是一个碎片，也足足万年没有在九域出现过了，一块碎片，都足以引起无数轰动。
<br/>　　“自然，极道仙兵的残片，楚某当然有兴趣。”
<br/>　　楚风眠开口道。
<br/>　　“虽然只是一枚残片，但是其中蕴含的力量，也不会弱吧。”
<br/>　　“不错，这枚碎片中的力量，据说在传闻之中，已经不再天级灵器之下了。”
<br/>　　李长老也不由的感叹了一声道。
<br/>　　“极道仙兵，毕竟是传说中的东西，一块碎片的力量，都足以媲美天级灵器。”
<br/>　　“以楚公子身上的财富，也可以去争夺一下，不过也并不一定可以拍到手，这极道仙兵的残片，只怕是会拍出天价。”
<br/>　　李长老笑着开口道。
<br/>　　“这极道仙兵残片出世的消息，早早已经是传遍了整个九域，据说是中域，西域的一些势力都派人前来争夺了，所以想要到手，也并不容易。”
<br/>　　“恩，这极道仙兵的残片，我也只是有些想法罢了，毕竟这种东西，可遇而不可求。”
<br/>　　楚风眠随意笑了笑道。
<br/>　　中域，西域的大势力？
<br/>　　现在楚风眠身上的财富，足以比得上寻常的庞然大物了，跟北邙学院相比的话，虽然说还不如。
<br/>　　但是楚风眠也不相信，北邙学院可以拿出灵泉来买这样一枚残片，而楚风眠拥有整个血脉的财富，足以抵得上半个灵泉。
<br/>　　在财富上，楚风眠不会逊色于任何势力。
<br/>　　在楚风眠跟李长老交谈之际，下方的拍卖会场中，已经是坐满了人。
<br/>　　这拍卖会场，巨大无比，楚风眠一眼望去，在下方足足是有着上万张王座。
<br/>　　如今这一万王座上，便是已经坐满了人，没有任何的空缺。
<br/>　　周围的包厢中，楚风眠也已经感觉的到，其中已经都是满了。
<br/>　　“一万多名巅峰圣者实力之上的武者，这虚空海中，居然有这么多人。”
<br/>　　楚风眠也是感叹了一声。
<br/>　　一名巅峰圣者，都足以在一个小国度中称王称霸了，是绝对的巨头。
<br/>　　眼前这拍卖会场中，居然是所有位置都坐满了。
<br/>　　“平日里是没有这么多人的，往届拍卖会，一般能够坐满五成，便是不易了。”
<br/>　　李长老在一旁笑着解释道。
<br/>　　“不过这一届拍卖会，因为那六座古洞府出世的消息，将九域中的无数强者都给引来了，所以才会如此爆满。”
<br/>　　“的确。”
<br/>　　楚风眠点了点头道。
<br/>　　他灵识扫过下方，自然看得出，这其中的无数武者中，不少身上的气息，所修炼的功法，都不是北域的。
<br/>　　而是来自与中域，西域等地。
<br/>　　“拍卖会开始了。”
<br/>　　楚风眠交谈之际，下方突然传来一声声音。
<br/>　　只见从上方台上，一尊老者缓缓走出。
<br/>　　一件件宝物，都被人抬上了台上，这其中的许多宝物，都是北域难寻的，甚至是一些都是远古时代就已经绝迹之物。
<br/>　　如今因为这古洞府，再度现世。
<br/>　　“这老者，不简单啊。”
<br/>　　楚风眠看了一眼这老者，便是看出，这老者绝对是一尊高手。
<br/>　　在他的身上，已经是觉察不出多少灵力了，看起来甚至就连御风境的实力都没有。
<br/>　　但是楚风眠清楚，这一切都是这老者的隐藏，这老者的实力，已经是达到了返璞归真的境地，才会看起来如此平静。
<br/>　　“这老者，可是通天阁的一尊大人物，名为灵尊真人，据说他的境界，已经是达到了生死境八重，在生死境高阶中，都是佼佼者的人物。”
<br/>　　李长老看到这老者，脸上也是极为恭敬。
<br/>　　“这届拍卖会，看来通天阁也是极为重视，居然是请他出来了。”
<br/>　　生死境八重。
<br/>　　这灵尊真人的实力，比起不死魔尊，都还要强大几分，楚风眠感觉到，以他现在的实力，跟着灵尊真人交手的话，只怕也只有五五之数。
<br/>　　不过，这还只是现在楚风眠的实力，若是等到楚风眠将身躯精炼到极致的话，在跟着灵尊真人交手，他则就足足有着九成把握，可以击败这灵尊真人，甚至是斩杀。
<br/>　　“灵尊真人！居然是他来主持。”
<br/>　　下方人群中，一些人认出这灵尊真人，都是惊叹一声道。
<br/>　　“这老者是什么来历？一个实力就连御风境都没有的老者，也敢来主持如此拍卖会？”
<br/>　　也有着一些武者，是第一次前来，迷惑不已。
<br/>　　“御风境实力都没有？这灵尊真人可是生死境高阶中的佼佼者，真正的大人物，像是你这样的人，他一个手指都足以捏死。”
<br/>　　令一尊武者，不屑的开口道。
<br/>　　这样的交谈，还在继续。
<br/>　　“安静。”
<br/>　　一直到灵尊真人，一声厉喝下，在场才彻底安静下来。
<br/>　　“这一届通天拍卖会，如今开始，拍卖会的规矩，诸位也应该清楚了，可以以灵石付款，也可以拿奇物相抵，只要是价值相当就足以。”
<br/>　　灵尊真人开口道。
<br/>　　“当然，通天拍卖会也有着通天拍卖会的规矩，若是喊价拿不出来足够财富的，就别怪我们通天阁不客气了。”
<br/>　　说道这里的时候，灵尊真人的言语中，也蕴含了其中几分杀意。
<br/>　　这杀意，令人听见，都是不寒而栗，虽然灵尊真人没有说清楚，但是人人都清楚。
<br/>　　敢扰乱这通天拍卖会的下场，只有一个，那便是死。
