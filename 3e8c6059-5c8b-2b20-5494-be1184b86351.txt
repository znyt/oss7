　　第538章
<br/>　　李行哉对孙德龙的投奔感到欣喜，随之便是困惑。
<br/>　　明帝身边有许多太监，李行哉认得的也就刘阿吉一个，更何况，孙德龙跟随明帝时，李行哉还没有出生，更是不知道对方乃何许人也。
<br/>　　说来说去，李行哉才知道孙德龙说的人是程大雷。心中骂了一声，倒也没有对孙德龙挑明。
<br/>　　“老奴一直想在陛下面前做事，奈何陛下已经归天。现在老奴一条命，便是殿下的。殿下有什么要老奴做的，尽管吩咐。”孙德龙忠心耿耿，指着身后几人道：“他们是我这些年培养的娃儿，殿下若看得上，就让他们留在身边。”
<br/>　　李行哉目光扫过去，眼见这些人都挺着笔直腰杆，一脸江湖匪气。
<br/>　　“兔崽子，还不快给殿下行礼，愣着干什么，没见识的玩意。”
<br/>　　孙德龙对李氏皇族是一片忠心，可他的手下却是未必。之所以肯来落凤城，一来是不敢违逆孙德龙的命令，二来便是想着在乱世中某个前程。
<br/>　　可到了落凤城一看，这李行哉算什么逍遥王，江南的土财主都比不上。跟在他手下做事，哪里如在江湖作威作福快活。
<br/>　　也就是孙德龙撂下这句话，他们才给李行哉下跪，脸上不情愿的表情，根本没有去掩饰。
<br/>　　李行哉什么事没见过，尤其是别人的白眼，他这辈子最不会生气的，就是别人瞧不起他。
<br/>　　“好啦，好啦，诸位英雄都起来吧。”李行哉摆了摆手。
<br/>　　孙德龙气不打一处来，尴尬冲李行哉道：“殿下恕罪，这帮兔崽子我没训好，回头让我好好收拾收拾。”
<br/>　　就是这轻飘飘一句话，可以看到漕帮几员大将机灵灵打个冷颤。由此，可见孙德龙平日的恐怖。
<br/>　　李行哉也明白，靠自己的本事，让漕帮彻底臣服有些困难。现在有孙德龙在，还可以辖制住他们。但看孙德龙的模样，怕已没有几年好活。
<br/>　　不过，现在李行哉缺人又缺钱，任何来投靠他的人，不管三教九流，他都来者不拒。
<br/>　　反正落到自己手中，自己慢慢调教他们就是。
<br/>　　“漕帮现在有多少人？”李行哉问。
<br/>　　“各个码头的弟兄加起来，有三千儿郎，钱粮的话，殿下想要，我都能送来。”孙德龙。
<br/>　　“好，我要一千人，另外派大船送些粮米过来。”李行哉算盘打得啪啪响：“其他需要的一些兵器弓箭，具体数目可以问戚将军。”
<br/>　　孙德龙的到来，解决了李行哉的燃眉之急，也加快了落凤城的发展速度。现在有了人有了钱，东西一到位，自己就可以展开扫荡海盗的计划。
<br/>　　万事开头难，只要第一步完成，那么接下来的事情就顺了。
<br/>　　李行哉想的，不由得意气风发。跌跌撞撞这么多年，他第一次拥有了自己的兵。
<br/>　　雄姿英发之余，心底还悬着另外一件事：
<br/>　　程大雷那混蛋，究竟来东海干什么。
<br/>　　……
<br/>　　刘发财这些天过得很不顺心。
<br/>　　他被程大雷掳在身边，没有一刻想的不是逃跑。可程大雷这个人，真特么地是个妖孽。哪怕是睡觉的时候，自己想悄悄溜走，都会被他发现。
<br/>　　想想以前自己过的是什么日子，招摇撞骗，风流快活，一提起自己的名字，多少人纳头便拜。当然，他提的名字都不叫刘发财。
<br/>　　可现在呢，每日提心吊胆，生活在程大雷笼罩的恐怖下。时不时还要经历杀手的刺杀，都是全天下叫得上名字的人物，如今却成了自己的敌人。
<br/>　　他多少次高声呼喊，我和他不熟啊，求求诸位大侠杀了他吧，我和你们是一起的。
<br/>　　但这样的话，自然不会有人相信。
<br/>　　“呔那泼贼，看你的相貌也是一丘之貉，还不速速受死。”
<br/>　　刘发财就这样被迫成了程大雷的同伙。
<br/>　　这一日，二人穿过一片密松林，继续向前跋涉。
<br/>　　“老大，我想咱们是不是把牛放下，步行的速度或许更快些。”刘发财骑在牛背上，大着胆子问。这些日子他也发现了，有时候程大雷还是蛮好说话的。
<br/>　　也仅仅是有时候。
<br/>　　“为什么？”程大雷。
<br/>　　“唉，这还用问么。现在东海一代，谁不知道您老人家骑着牛来了。传说您要杀够一万人才肯收手，咱们骑着牛，多么显眼的目标。这不等着人来杀么？”刘发财说完又补充一句：“您这次出山，当真不是为了杀够一万人吧？”
<br/>　　“什么话，我这头墨玉乌月牛价值连城，怎么能随便丢了。”程大雷道：“江湖传言也能相信，谁说我要杀够一万人。”
<br/>　　“那您这次出山具体为什么？”
<br/>　　“杀十万人。”
<br/>　　“呃……”
<br/>　　刘发财打量着程大雷的黑牛，一路坎坷，它自然没有任何神骏可言。毛发打结，满身泥泞，但看着有些凶，似乎是打盹儿的兽王。
<br/>　　“程大雷，你哪里走，我等你多日了。”
<br/>　　刚转过一个山坡，就看前方官道上冒出三个人。刘发财认真打量着这三个人，两男一女，手里拿着剑。
<br/>　　刘发财第一眼就认出了这三人是谁，江湖上赫赫有名的杀胚：杀人夫妻。
<br/>　　这三人能在江湖上扬名，本事自然不弱。但他们更出名的是另外一件事。
<br/>　　三人是一个师父教出的徒弟，大师兄剑恨南，小师妹苏离，小师弟李越英。
<br/>　　三人同门学艺，经常切磋武艺，打来打去，便打出了感情。剑恨南和李越英同时恋上小师妹苏离。这苏离却是难以抉择，最后折腾来折腾去，竟然把两人都选了。
<br/>　　于是，三个人就幸福快乐的生活在一起。
<br/>　　在帝国，男人三妻四妾根本算不上什么事，但一凤双龙的事情还是有些……江湖人提起这件事，难免就啧啧出声，似乎在脑海中推测三人是怎样生活的。
<br/>　　在如此状态下，三人的行事越来越偏激，一旦有人敢嘲笑他们，即便相隔千里，也要去杀了。
<br/>　　三人的行事虽然诡异些，但本事却是不弱。尤其是三人合练的一套三才剑法，一旦三人合体施展开，实力陡增，多少成名高手都折在他们手中。
<br/>　　三人合体……刘发财摇摇头：哎呀，我太邪恶了。
<br/>(本章完)
