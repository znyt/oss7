　　楚涵将国师打断了手脚然后绑起来，跟大将军绑在了一起，整个过程中大将军一个劲的嚎，国师则是连个反抗的念头都没有，口中不断说着‘杀了我吧’，对比这种被绑起来游街，他宁愿一死。
<br/>　　但楚涵始终认为这是天运国的家务事，要死也是应当由国王动手，他手上沾染的血太多，杀太多人不好。
<br/>　　于是接下来的日子，楚涵就这么在天运国的王城中呆了下来，没事看几眼被捆绑关押的大将军和国师，晚上去河边上找闲来无事的老人下棋。
<br/>　　战报每天都有，不是国王又占领了一座城池，就是国王杀敌多少多少。
<br/>　　总之，每一天的天运国国民都生活在偌大的振奋和喜悦之中，天运国的各大主城更是不断供应，一车车的物资送往了前线。
<br/>　　明成国并不大，国王的战略也很明确，根本没想过打下整个国家，而是一条线直逼明成国的王城。
<br/>　　国王杀过去的速度极快，带着一万精兵打头阵，一条直线杀过去，竟然一路都是大胜，杀的对面措手不及！
<br/>　　明成国也根本没想到天运国会忽然杀过来，他们准备好的全部都是进攻，防守上尤其疏忽，兵力全部聚集在边境上的各处，还没来得及追赶上国王带领的那一万精兵，就被后面留在边境的大批天运国士兵拦截。
<br/>　　兵分两路，国王带领的那一批主负责攻占侵略，后面的大批士兵则专门对付明成国的兵。
<br/>　　这一战略出乎意料的好，明成国的境内没有兵力把守，让国王的一路杀到了明成国的王城城下！
<br/>　　明成国的国王完全没料到会有这种突变，整个王城都傻眼了！
<br/>　　兵临城下之中，天运国的女国王很霸气，一句废话没有，直接让一万精兵攻城！
<br/>　　于是当开始攻城的第三天，消息就传回了天运国王城。
<br/>　　“国王攻城了！明成国要被灭了！”
<br/>　　大将军和国师听到这一消息，双双晕了过去，任他们怎么想，都想不明白这是为什么。
<br/>　　楚涵算着日子，距离他离开这里相当快了。
<br/>　　攻打主城花了好几天的时间，与当初天运国的王城沦陷一样，国王让对面的人也感受了一次弹尽粮绝。
<br/>　　而她所率领的一万精兵，完全是一路掠夺过来的，吃的喝的用的，全部都是一路杀戮占领的各大城池用品，根本不缺东西。
<br/>　　孤立无援的明成国王城就这么一天天之中兵力所剩无几，对面的国王想坚持下去，但这时候的天运国女国王，让弓箭手的箭羽上穿·插着纸团，万箭齐发射向城内的各个建筑。
<br/>　　很多人都以为是带着火焰的箭羽，以为要烧城！
<br/>　　发现是纸团后打开一看，王城的居民纷纷躁动起来。
<br/>　　这一招很简单，天运国的国王在以德服人，纸团上的内容也很简单，无非就是两国交战，不伤平民。
<br/>　　也是在劝降！
<br/>　　很快，明成国的王城居民舆论成灾，尤其是他们也知道这战争的一开始，便是明成国的国君挑起。
<br/>　　可如今不但没有占领天运国的一毫国土，还被对面的女国王杀到了城下。
<br/>　　饥饿，是导致民心散乱的重要因素！
<br/>　　于是在巨大的暴动和舆论之中，明成国的国王顶不住压力，在王宫中自杀。
<br/>　　很快明成国的国门大开，天运国的一万精兵呼啸而入，直接占领了明成国的王宫，速度快的令人咂舌。
<br/>　　天运国的女国王说到做到，带着从各处城池掠夺而来的物资，分散给王城中的居民，不仅受到了全民爱戴，也让那些被活捉的各个明成国大臣无话可说。
<br/>　　做完这一切，后方的大批天运国军队也到了，开始一步步蔓延到明成国的每一个城池，占领一个国家要做的事可不简单，少至三年多达十年，很多东西都需要一步步来。
<br/>　　做完这一切的女国王有些恍惚，她回首看着天运国扩大至一倍的国土，至今还有些回不过神。
<br/>　　才不到半个月，她竟然就灭了明成国！
<br/>　　天运国丢失的王冠也在明成国的国库中找到，女国王带着王冠，领着不到一千的精兵先一步回到天运国。
<br/>　　一路上享受了全民的热情迎接，也让天运国长久以来的爱国之情到达高峰。
<br/>　　人性就是这样，打了胜仗便是英雄！
<br/>　　当国王回到王宫的时候，楚涵正百无聊赖的坐在台阶上等她。
<br/>　　身后是百官大臣，王宫前的空地上是一干精英铁骑，归来的国王也不再是那个爱哭的小女孩，她带着一股身经百战的强大气势，身染血色王袍带着王冠走向楚涵。
<br/>　　楚涵一瞬间握紧了修罗战斧，这种气势很像开打前夕，他是不是养虎为患了？
<br/>　　哪知国王在走到楚涵面前三米处就停下，她将王冠摘下来，想了片刻后才开口道：“如果是你领兵，这一仗会打多久？”
<br/>　　楚涵想了想，道：“那要看是领的哪一批兵。”
<br/>　　国王又问：“狼牙呢？”
<br/>　　楚涵噗嗤一笑，笑容满是戏谑：“出动狼牙神隐五十人，三天就行。”
<br/>　　这话一点没错，在本就比天运国明成国更加复杂和更高战力的末世纪元，神隐就是用五十人灭过一个基地。
<br/>　　国王瞳孔一缩，继续道：“那如果你是明成国的国君，你如何应对这一仗？”
<br/>　　楚涵仔细想了想，道：“杀了你，天运国的军队不值一提。”
<br/>　　国王的气势是楚涵提上来的，天运国的军队也是全体将她当成了主心骨，国王一死群龙无首，什么都不复存在。
<br/>　　国王叹了口气，缓缓将王冠递上：“主教大人，这是承诺你的。”
<br/>　　这一幕让下方的一千多人躁动起来，窃窃私语不断，但也有很多人看出了倪端。
<br/>　　国王的变化跟变了一个人一样，再看楚涵坐在台阶上的悠闲样子，对比国王对其的敬重，不难想象这个男人就是国王的老师。
<br/>　　顿时很多人看楚涵的眼神都变了，变得尤其复杂。
<br/>　　楚涵接过王冠，果不其然在王冠上，一枚蓝色如同宝石一般的回炉碎片，就镶嵌其上！
<br/>　　最后一枚回炉碎片，到手！
