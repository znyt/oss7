
<br/>　　<br/>　　    王惊蛰烦躁的搓了搓脸，他确实挺烦的，因为沈君明显是在看人下菜碟，他给的东西既不是钱也不是珠宝，就一块鬼谷子用的阴沉木牌子，这东西给别人可能都觉得是累赘放在家里供着也不合适，但给了他的话，就跟你送给了一个要上战场上的人一件防弹衣，太严丝合缝的合适了。<br/>　　<br/>　　    而沈君给了这东西，虽然说是酬谢，但王惊蛰真的就能这么平白无故的收下么？<br/>　　<br/>　　    “如果你和沈君之间没有爱情的话，那他送你这么贵重的东西，就明显是有所图谋了，今晚吃饭的时候，是不是聊了什么？”菜刀文一针见血的问道。<br/>　　<br/>　　    “确实是有别的事，这还得从沈沉鱼的问题上说起，沈君顺藤摸瓜找到了给她下灵媒的人，背后那人叫胡胡，是玄门山字脉的山主……”<br/>　　<br/>　　    “啊？”茅小草惊诧一愣。<br/>　　<br/>　　    “你怎么了，认识他啊？”王惊蛰问道。<br/>　　<br/>　　    “不认识，你接着往下说”茅小草平淡的摆了摆手。<br/>　　<br/>　　    王惊蛰说道：“沈君的意思是，他不适合对付胡胡，就问我能不能答应帮个忙，不过他一开口，我当场就给推了，接下来沈君也没有再提，后来他的司机送我过来下车的时候，把这个东西给了我，说是沈君对我表示的谢意，在车上我也没看，以为就是什么寻常的礼物呢，谁知道他送给我的时刻核弹啊”<br/>　　<br/>　　    “他虽然是没再提，不过你肯定不好意思白收，毕竟咱都是要脸的人，无功不受禄啊，之前的帮忙是人情，人情过去了，沈君再找你就是交易了，所以他把这块木头疙瘩甩了过来”<br/>　　<br/>　　    王惊蛰有点反感的说道：“这帮人太利益性了，这不是把我直接给架起来了么”<br/>　　<br/>　　    “咋办，送回去啊？”菜刀文问道。<br/>　　<br/>　　    “送回去，就不太好看了，东西都收到手里了，就这再给送回去的话那也太打脸了，收下了就得为他办事，这不是让人两难了么？”王惊蛰拿起一瓶酒仰头就干了一大口，抹着嘴巴子说道：“我算是长记性了，以后和陈重，沈君这种人那就是点头之交，往深了绝对不能接触，不然真说不准他们什么时候，就挖了个坑等着你呢”<br/>　　<br/>　　    “啪”茅小草拍了下王惊蛰，瞅着桌子上放着的木牌说道：“闲着也是闲着，要不你就试试？”<br/>　　<br/>　　    王惊蛰“咦”了一声，诧异的问道：“你不是等着让我陪你去骊山呢么，我这一试试说不定就耽误多长时间了，你不还得等着么”<br/>　　<br/>　　    小草手指摩挲着阴沉木，眼珠子转了转说道：“这个……现在也不急啊，再说了，这东西不是对你很有用么，既然不急，你又把它收了，那干脆就借坡下驴得了”<br/>　　<br/>　　    “你们女人，真是太善变了，东边日出西边雨的……”<br/>　　<br/>　　    茅小草是觉得，王惊蛰本来就跟山字脉的胡胡不对付，人家处心积虑的要对付他呢，真有个反制他的机会，不用白不用啊，再一个就是，当初王仙芝从她这里要走引魂灯的时候，曾经隐约提过一点要干什么，小草下意识的觉得，这块阴沉木搞不好以后可能会帮上大忙，所以就鼓动着王惊蛰答应了沈君。<br/>　　<br/>　　    王惊蛰一看，这前有说情的后面还有送礼的，也就真没办法拒绝了，于是当场就给沈君去了电话，那边就好像知道他今晚会回话一样，铃声刚想就被接了起来。<br/>　　<br/>　　    “沈先生，您这份礼真是太大了，我拿在手里的时候都差点掉在脚面上砸出个包来”王惊蛰用一种调侃和略带不满的语气说道。<br/>　　<br/>　　    电话那头，沈君明显是长吐了口气，然后才说道：“老弟啊，你知道么，从你走了以后我就坐在饭桌旁没动，手机就放在我面前的桌子上，茶水也喝了两壶了，我就等着你的电话呢”<br/>　　<br/>　　    王惊蛰皱了皱眉，瞬间无语了，对方姿态拉的这么低，他还真不好意思在挤兑沈君了。<br/>　　<br/>　　    沈君接着说道：“我和你说句实话，这个世上有太多人，在交往人的时候都是利益至上的了，但你君哥肯定不是这样的人，我指着头顶的灯跟你发誓，今晚你要是没给我打电话，下次再见面我仍然叫你一声老弟，并且语气和态度还是一如既往的，我的理念就是，不是办不了事就成不了朋友了，我和你，咱们的关系都会一直往下走，而不会因为你的拒绝产生疏远，明白么？”<br/>　　<br/>　　    王惊蛰沉默了一下，点头说道：“呵呵，我理解你了”<br/>　　<br/>　　    “唉，这就对，理解万岁吧……”沈君畅快的笑了笑，说道：“这样，你等我电话，我在安排安排，可能这两天就会联系你了，然后你那里注意一下”<br/>　　<br/>　　    “好的，我等你消息”挂了电话，王惊蛰把手机扔在桌子上，无语的说道：“我这个人就是脸太小了，爱要面子，不懂得拒绝”<br/>　　<br/>　　    茅小草斜了着眼睛，笑眯眯的问道：“那面对女人的投怀送抱，你也不懂得拒绝，要面子是么小蛰蛰？”<br/>　　<br/>　　    王惊蛰顿时撇嘴说道：“别闹了，哪个女人那么瞎啊，还对我投怀送抱的”<br/>　　<br/>　　    小草磨着牙说道：“妈蛋，有种你再说一遍？”<br/>　　<br/>　　    “你可能误会我的意思了，其实我和你的关系，不是你的投怀送抱，而是我对你的死皮赖脸，你造么？这是两回事啊”王惊蛰顿时就泛起了小机灵劲，很有力度的回了一句过去。<br/>　　<br/>　　    茅小草伸出两根手指说道：“来，你过来，让我掐一下你脸”<br/>　　<br/>　　    王惊蛰眉开眼笑的说道：“妥，你看我用什么姿势凑过去，你能掐的身心愉悦一点？”<br/>　　<br/>　　    林汶骐很惊愕的看着他，低声跟菜刀文说道：“我一看，他这也是情场上的老中医啊，我是真服了，他的反应就跟无痛人流似的，人家小草还没开始收拾他呢，他就能把麻烦给弄结束了，这是个人才啊……”<br/>　　<br/>　　    菜刀文感叹的说道：“是啊，我他么有时候都觉得，我是不是应该也失忆一把了，真好使啊，以前泡妞我能甩他两条街，现在我就是给自己装个涡轮，都不能一定能追的上他了”<br/>　　<br/>　　    “真的，我看这个路数确实靠谱，要不咱俩组个团去重新把记忆回回炉吧，没准来个华丽的转身之后，咱们也是个情场浪子了……”
