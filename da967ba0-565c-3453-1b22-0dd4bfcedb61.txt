　　屠雄的实力，的确是要远远超过其他人。
<br/>　　所以擂台上的不少武者，都是不由的一齐出手，足足有着二十多尊武者，一齐出手。
<br/>　　向着屠雄，围攻而去。
<br/>　　“一群废物，还敢来围攻我？找死！”
<br/>　　屠雄冷哼一声，双眼之中尽是杀意。
<br/>　　只见他的身躯动了，他身上的肌肉高高隆起，猛然一用力，一拳打出。
<br/>　　这一拳打出，其中压缩的灵力，便是直接化为了一道霹雳，顿时周围围攻的无数武者，在这一道霹雳的爆炸中，全部化解。
<br/>　　周围的无数武者，都被直接震飞了出去。
<br/>　　“此人怎么会如此厉害？”
<br/>　　无数武者被震开，每个人的眼神中都是无比惊骇。
<br/>　　他们如此之多的人一起出手，围攻屠雄，居然是都无法奈何的了屠雄。
<br/>　　“死！”
<br/>　　屠雄避开众人，冷哼一声，直接向着围攻中其中最强的一尊中年人冲了过去。
<br/>　　“轰隆！”
<br/>　　他的一拳打出，那尊中年人，便是跟刚刚的青厉一样，直接被贯穿胸口，硬生生的打死。
<br/>　　“你们这些废物，正好可以来化为我的养分！我练就鲜血魔体，正好需要圣者精血来吞噬，如今你们都得死！”
<br/>　　屠雄狂笑连连，那中年人身上的精血，吞噬被他吸取的一干二净。
<br/>　　他的眼神，再次扫视过去，身影连连变幻。
<br/>　　“砰！砰！砰！”
<br/>　　又是三尊武者，直接死在了屠雄的手中，那些武者的精血，全部都被屠雄吞噬殆尽，化为了他身躯中的力量。
<br/>　　“不好，此人已经不是我们可以能够对付的了，快逃！”
<br/>　　看着屠雄，如今大杀四方，周围一些武者的眼神纷纷变了，眼神中尽是惧怕之色，一个个急忙的退后，想要认输。
<br/>　　“现在想要认输？晚了！你们都要成为我的养料，我会以你们的精血，来铸造身躯！”
<br/>　　屠雄大笑连连，身形猛然冲了上去，一瞬之间，这刚刚围攻他的武者，全部都是尽数被他斩杀。
<br/>　　无数精粹圣者精血，都是被他吞噬。
<br/>　　吞噬了如此之多的圣者精血，如今屠雄的力量，便是有增强了几分，这让他的一双双眼，又看向了剩下的武者。
<br/>　　“你们这些人，也都死吧！”
<br/>　　屠雄眼神疯狂，便是如同一尊魔头一般，直接猛然出手，开始疯狂屠杀这擂台上的无数武者。
<br/>　　几乎是要将这擂台上的所有人，都给杀光。
<br/>　　擂台上的无数武者，不是纷纷逃走，就都是急忙认输，一个个的眼神中尽是恐惧。
<br/>　　在又斩杀了几尊武者之后，屠雄的双眼，再次开始扫视着，正好一眼便是看见了，站在原地，脸色平静的楚风眠。
<br/>　　这令他的脸上，不由露出几分讥笑。
<br/>　　“小子，看来你是吓傻了吗？连逃都不会了？你过来，乖乖跪在地上，我可以放过你一条命如何？”
<br/>　　屠雄看向楚风眠，讥笑道。
<br/>　　一个御风境武者，在他眼中，简直就是如同蝼蚁的存在。
<br/>　　不入神力境，屠雄就算是想要杀的兴趣都没有，看到站在原地脸色平静的楚风眠，他的眼神中不由露出几分玩味。
<br/>　　屠雄指着楚风眠，大声骂道。
<br/>　　这言语之中，尽是威胁，丝毫没有将楚风眠放在眼中。
<br/>　　“小子，说你呢，快点滚过来！”
<br/>　　“恩？”
<br/>　　楚风眠看着屠雄，脸上突然露出几分笑意道。
<br/>　　“这样如何？你跪下，我可以放过你一命如何？”
<br/>　　“什么？小子，你莫非是疯了？”
<br/>　　听到楚风眠的话，屠雄先是一愣，随后脸上大笑道。
<br/>　　“饶我一命？就凭你一个御风境的蝼蚁？说你是废物，都算不上，你这样的实力，也敢来参加北邙学院的考核，真是自不量力。”
<br/>　　屠雄的眼神一变，笑意便是消失，直接化为了杀意。
<br/>　　“既然你自己找死，那我就成全了你！”
<br/>　　屠雄的身躯，突然动了。
<br/>　　这速度，快到惊人，几乎一瞬间，便是来到了楚风眠的面前。
<br/>　　他的手中，灵力凝聚，便是突然一拳，冲着楚风眠的脑袋打了过去。
<br/>　　“自不量力。”
<br/>　　楚风眠口中，平淡的吐出四个字。
<br/>　　他的手，仿佛随意挥动，一股灵力打了出去。
<br/>　　这屠雄的一拳，便是就被彻底化解，不留任何力量，没有任何一丝力量，打在了楚风眠的身上。
<br/>　　“此人是谁？”
<br/>　　“居然有人，可以抗衡屠雄？”
<br/>　　“一个御风境武者，怎么可能！”
<br/>　　擂台之上，其他的一些武者，看着眼前的这一幕，皆是不敢相信自己的双眼。
<br/>　　这所向披靡，近乎无敌的屠雄，如今他的拳风居然会被人如此轻易的化解了。
<br/>　　而且做到这一切的，却还只是一个御风境武者，就连实力媲美巅峰圣者的青厉都无法抗衡这屠雄的一拳。
<br/>　　如今楚风眠居然做到了。
<br/>　　这让人如何不惊。
<br/>　　“哦？小子你的实力，的确有些门道。”
<br/>　　屠雄看到他的一拳被如此轻易的化解，脸上也是不由的露出了几分惊讶之色。
<br/>　　不过随后很快，便是转为了冷笑。
<br/>　　“就算是你有着几分门道又如何？今日你还是要死！”
<br/>　　“鲜血魔体！”
<br/>　　屠雄的身躯，变的膨胀无比，一身气息，比起刚刚还要强大许多。
<br/>　　“什么？这才是屠雄的全力？”
<br/>　　“好恐怖的气息，这等气息，只怕是逼近生死境了吧。”
<br/>　　看着屠雄身上暴涨的气息，周围无数人皆是不可置信的喊了起来。
<br/>　　“小子，去死吧！”
<br/>　　屠雄狰狞一笑，再次出手，又是握拳，一拳冲着楚风眠，轰杀而来。
<br/>　　这一拳的力量，比起刚刚，只怕是要强大了数倍。
<br/>　　这才是屠雄的真正实力。
<br/>　　换做一尊巅峰圣者，只怕是都会死在这一拳之下。
<br/>　　“破！”
<br/>　　面对着极为恐怖的一拳，楚风眠这边，却只是轻描淡写的一般的随意开口了一句。
<br/>　　楚风眠的口中，一个字吐出，蕴含着几分灵力。
<br/>　　这恐怖的一拳，便是就在楚风眠的面前，轻松自如的化解了。
