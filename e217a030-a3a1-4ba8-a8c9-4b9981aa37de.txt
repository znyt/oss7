　　萧若曦自然不知道，
<br/>　　叶尘之所以要读大学，是为了更好的守护曦月。
<br/>　　按照曦月的计划，在她的神魂，没有达到元神离体的境界之前，只能待在萧若曦的身体里。
<br/>　　所以在此之前，对于叶尘来说，守护曦月最好的办法，就是进入天海大学。
<br/>　　至于，为什么分数是581，报考专业是生物工程。
<br/>　　那是因为，叶尘忽然想起，他前世的高考分数就是581分，而报考的专业，正是天海大学的生物工程专业，所以便顺口说了出来。
<br/>　　电话那头，
<br/>　　易山河听完叶尘的话，也顿时愣住了，
<br/>　　“叶帅，你不是在开玩笑吧？你当真要进入天海大学读书？”
<br/>　　堂堂华夏国元帅，当世第一人，竟然要进入大学当学生？
<br/>　　难道是想效仿古代皇帝微服私访？
<br/>　　“当然是真的！怎么？老易，这点小事应该难不住你吧？”
<br/>　　叶尘淡淡开口道。
<br/>　　易山河苦笑道：
<br/>　　“难自然是不难，不过您能不能告诉我为什么啊？”
<br/>　　叶尘想了想，回了一句，
<br/>　　“因为爱情！”
<br/>　　然后便直接挂断了电话。
<br/>　　易山河愣了好一会，最终一阵苦笑摇头，自言自语道：
<br/>　　“是了，我们一直把他当做神灵一般看待，可事实上，他毕竟也只是一个不到二十岁的少年啊！”
<br/>　　易山河感慨了片刻之后，又拨通了一个电话，
<br/>　　“小张吗？你立即跑一趟天海市，去天海大学帮我办一件事！”
<br/>　　……
<br/>　　一晃眼，
<br/>　　几日时间，匆匆而过。
<br/>　　这几天，叶尘几乎一直待在天海大学附近，每天中午时分，曦月便会占据萧若曦的身体两个小时，和叶尘相见。
<br/>　　这一天，
<br/>　　终于到了大一新生入学的日子！
<br/>　　“叶帅，你入学所需要的身份资料，已经全部都在这里面了，需要属下送您过去吗？”
<br/>　　天海大学外面，一名身穿军装、肩扛两杠三星的中年军官，将一份材料交到叶尘手上，毕恭毕敬的开口道。
<br/>　　而在中年军官的身后，还跟着两名少校，全都站得笔直
<br/>　　叶尘摆了摆手，
<br/>　　“不用！我自己进去报到就行了，你们都回去吧！”
<br/>　　……
<br/>　　等叶尘来到天海大学门口的时候，这里早已经是人满为患。
<br/>　　不过，天海大学的每个学院，都会派出高年级的志愿者，在门口举着各个学院的招牌，来迎接各自学院新生的到来，然后由他们带领新生完成入学报到。
<br/>　　叶尘扫视了一圈，很快便找到了生物学院所在，于是便缓步走了过去。
<br/>　　与其他学院的火爆场景相比，生物工程学院这边，人非常少不说，还在一个偏僻的角落里，如果不是叶尘眼力够好，还真不容易找到。
<br/>　　对此，叶尘也一点也不奇怪
<br/>　　生物类专业，一向都是冷门专业，报考的人不多，自然也就十分冷清。
<br/>　　“你好，我是生物学院的新生！”
<br/>　　叶尘走上前，缓缓开口道。
<br/>　　负责迎接生物学院新生的，一共只有五个人，四男一女。
<br/>　　看到叶尘走了过来，那唯一的一名女生，顿时两眼发亮，低呼了一声，
<br/>　　“好帅！”
<br/>　　其他四名男生，也顿时升起一股相形见绌的感觉。
<br/>　　相比之下，叶尘无论是气质，还是容貌，以及穿着，都是上上之选，一看就知道来头不凡。
<br/>　　“咳咳！”
<br/>　　为首的一名青年男子，轻咳了一声，笑道：
<br/>　　“你好！我们是生物学院学生会的，我叫林涛，先出示一下你的通知书和身份证件吧！”
<br/>　　叶尘当即把相关的证件资料，递给了林涛。
<br/>　　林涛大致检查了一遍之后，便还给了叶尘，笑道：
<br/>　　“叶尘学弟，我们代表天海大学生物学院，欢迎你的加入！”
<br/>　　说完这话之后，林涛望了一眼叶尘身后，脸上微微有些好奇，道：
<br/>　　“你的家人没有陪你来吗？还有你的行李在哪？我可以让人帮你去搬！”
<br/>　　叶尘淡淡一笑，
<br/>　　“就我自己一个，也没有带行李，你只要告诉我，去哪里报道就行了！”
<br/>　　虽然前世，叶尘曾经在天海大学度过一年时光，不过那毕竟是八百年前的事情了，除了和他同宿舍的那几位舍友之外，其他的相关记忆，已经十分模糊。
<br/>　　听到这话，五人中唯一的那名女生，当即站了出来，自告奋勇道：
<br/>　　“我带你去吧！”
<br/>　　叶尘微微一笑，
<br/>　　“多谢！”
<br/>　　两人前脚刚走，剩下的四人，顿时小声议论起来，
<br/>　　“这个戴丽丽，又犯花痴了！”
<br/>　　“不过这小子，确实很帅啊！比之前那个杨之凡还帅！”
<br/>　　“男生这边，有叶尘和杨之凡，女生那边有个陶婉清，看来咱们生物学院，今年新生的质量很不错嘛！”
<br/>　　……
<br/>　　四人的议论声虽然不大，但还是被叶尘清晰无比的听进了耳中。
<br/>　　“杨之凡……陶婉清……”
<br/>　　听到这个两个名字，叶尘前世的相关记忆，也渐渐浮现了出来。
<br/>　　他隐约记得，这杨之凡是他们这一届中，最为优秀的一个，而且背景也不弱，一进入学院，就成为了生物学院的院草。
<br/>　　至于陶婉清，这和杨之凡，并称为生物学院的金童玉女，两人后来貌似也真的走到了一起，羡煞旁人。
<br/>　　不过，这些记忆，也只是在叶尘脑海中，微微一闪，便直接抛到脑后了。
<br/>　　以他如今的心境，杨之凡和陶婉清，他自然不会放在心上。
<br/>　　“对了，应该马上就能见到我的那三位室友了吧？也不知道这一世，还会不会和他们分在同一个宿舍？”
<br/>　　想到这里，叶尘心中顿时隐隐有些期待起来。
<br/>　　……
<br/>　　在戴丽丽指引下，叶尘顺利完成了新生报到，拿到了报到证，和宿舍寝室号。
<br/>　　让叶尘感到惊喜的是，宿舍寝室号是602，
<br/>　　和前世一样！
<br/>　　戴丽丽倒是热情的很，一直帮叶尘办好了住宿手续方才离开。
<br/>　　叶尘拿到宿舍钥匙之后，循着上一世的记忆，来到6楼，找到了602房间。
<br/>　　房门是开着的，此时里面已经有了三个人。
<br/>　　原来，叶尘竟然是最后一个！
<br/>　　望着眼前的三人，叶尘的心情顿时微微有些激动，
<br/>　　‘陈东！唐威！李虎！还真是你们三个啊！’
