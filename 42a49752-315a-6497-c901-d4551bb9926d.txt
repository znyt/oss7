<br/>　　落长恨心中怒火中烧，一脸恨意，看向那些荒古门弟子，似乎是有些心有不甘。

<br/>　　荒古门这等庞然大物，落长恨在他的面前，简直就是蝼蚁。

<br/>　　突然间，落长恨的眼神中，扫视了一眼众人，突然开口道。

<br/>　　落长恨眼神一动，一双眼睛的视线，直接看向了楚风眠。
<br/>　　顿时楚风眠便是感觉到，无数视线，直接便是看向了他。
<br/>　　这一道道视线之中，有贪婪，有杀意，甚至是有不少人，听到落长恨的话后，都恨不得直接要杀了楚风眠，夺取楚风眠身上的宝物。

<br/>　　听到落长恨的话，楚风眠眼神中都是露出了几分杀意。
<br/>　　这落长恨的话，根本便是将楚风眠给推了出来，让楚风眠来跟那些荒古门弟子交手，左手渔翁之利。
<br/>　　“哦？”
<br/>　　那些荒古门弟子，一道道视线，也是突然都看向了楚风眠。
<br/>　　打量了一眼楚风眠的境界，这些荒古门弟子的脸上，都尽是不屑。
<br/>　　“就凭你们这些北域的蝼蚁，也敢跟我们荒古门来谈条件？”
<br/>　　一名荒古门弟子不屑的开口道。
<br/>　　“不过一个神力境四重的小子罢了，的确也不配拥有这等宝物，王智，你去杀了这小子，将他的宝物全部夺过来吧。”
<br/>　　一尊荒古门弟子，看了楚风眠一眼，直接开口道。
<br/>　　“是！”
<br/>　　顿时那名叫做王智的荒古门弟子，一下子便是站了出来，看向楚风眠的眼神中，顿时冷笑连连道。
<br/>　　“算了，我就先杀了你！让这些北域的土著们知道，这颜古龙帝洞府中的宝物，不是你们配拥有的！”
<br/>　　王智开口，他的双眼，顿时盯向了楚风眠，突然之间，就在楚风眠的身边，燃烧起无数熊熊火焰，将楚风眠的身躯，全然包围在了其中。
<br/>　　这火焰，近乎无声无息之间的出现，仅仅是王智的双眼看了过来，随着他的目光所出现。
<br/>　　“怒目双瞳！这是怒目双瞳！”
<br/>　　一些北域武者，脸色都是一边。
<br/>　　“传闻中愤怒之下，目光之下都会燃烧起熊熊火焰！这虽然并非是先天神体！却也是要远超常人的强大！”
<br/>　　“这怒焰，无声无息之间都可以出现，随着目光，都可以直接看过去，这是无敌的力量啊！”
<br/>　　“荒古门中，随便一个弟子，居然都拥有如此可怕的血脉？”
<br/>　　一些北域武者，都是震惊无比。
<br/>　　特殊血脉，并不少见，起码在北邙学院中，拥有特殊血脉的武者，都并不少。
<br/>　　像是楚风眠的远古战龙血脉，其实也算是特殊血脉，这特殊血脉并而只有先天得到，后天奇遇一样不少。
<br/>　　但是就算如此，能够拥有特殊血脉的武者，也都是一方势力的天才了。
<br/>　　可眼前，随意出来一个荒古门弟子，居然都是拥有特殊血脉，这实在是太可怕了。
<br/>　　跟荒古门相比，荒古门随意一个弟子，都比起许多号称的天才还要天才。
<br/>　　“哼，区区北域土著，就连特殊血脉都没有见过，真是一群废物，今日就让你们这些北域土著看看，我们荒古门的厉害！”
<br/>　　王智看着众人的反应，脸色更是自傲无比，看向楚风眠，冷冷的说道。

<br/>　　王智的血脉之力全开，无数怒焰凝聚而出。
<br/>　　无数怒焰，顿时咆哮起来，化为无数火蟒，向着猛然扑了过来。
<br/>　　滚滚热浪，席卷着整个密室之中，许多实力弱一点的武者，都是脸色苍白起来，在这热浪下，不得不动用灵力，来抵挡着其中的热浪。
<br/>　　“好可怕的火焰！在这火焰下，我们就连反抗的机会都没有。”

<br/>　　“完了，看来此人要被杀鸡儆猴了。”
<br/>　　一尊尊北域武者，看向楚风眠的眼神中，都十分可怜楚风眠。
<br/>　　在他们看来，楚风眠完全就是被这些荒古门弟子，当做鸡一样要杀掉了。

<br/>　　他们任何一人，都没有想过，特殊血脉，本就是同境界无敌的存在。
