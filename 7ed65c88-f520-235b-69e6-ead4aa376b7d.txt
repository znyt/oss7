　　叶少阳嘴角抽搐了几下，拱了拱手，“敢问这位师父法名，这里又是什么地方？”
<br/>　　“不敢当不敢当，和尚是林隐寺僧人，名字嘛……不记得了。”说完又喝了一口酒。
<br/>　　这个动作，给了叶少阳强烈的暗示，隐约想到了一个名字，但没有开口去问。
<br/>　　“大师，不知道这是什么地方，我明明在山洞里，怎么一下子到这儿了？”
<br/>　　老和尚伸手点着他，道：“走吧，跟我走就知道了。”
<br/>　　说着上来拉他。
<br/>　　“哎哎等下，我那个朋友呢？”
<br/>　　“放心吧，她没事。”老和尚眨眼笑着，“小天师，你也是个重情的人哪。”
<br/>　　叶少阳看看四周，也不知道这什么地方，只好跟着他走，方向是对面那座高山。
<br/>　　老和尚在前面走，一边走一边甩开了膀子，唱到：“世人都晓神仙好，惟有功名忘不了！古今将相在何方？荒冢一堆草没了。世人都晓神仙好，只有金银忘不了！终朝只恨聚无多，及到多时眼闭了。世人都晓神仙好，只有娇妻忘不了！君生日日说恩情，君死又随人去了。世人都晓神仙好，只有儿孙忘不了！痴心父母古来多，孝顺儿孙谁见了？”
<br/>　　唱完了哈哈大笑，好像突然想起身后还有人，回头看了叶少阳一眼，道：“小天师，你说我这唱得怎么样？”
<br/>　　“听上去不错，但太极端了。”
<br/>　　“哦？正好赶路无聊，小天师请解说一番。”
<br/>　　叶少阳本来不想聊这个，听他这么说，便开口说道：“第一，是人都会死，不能因为都会死，就什么都不做，反而，正因为人都会死，所以活着的时候，该追求的一定要追求，不然死了就没机会了。”
<br/>　　他说完，看着老和尚，老和尚微笑，没任何表情。
<br/>　　“关于钱的那个，我以前也觉得赚钱就该花，不然浪费，但花钱的目的，是为了开心，有些人虽然很抠……哦，就像我郭师兄那样，这种人花钱就像割肉，什么都不舍得，但只要一想到自己有很多存款，也会很开心，这种乐趣跟别人花钱一样，那就让他开心就好了，谁规定赚钱一定要花？还有你说娇妻的，这个简直太极端了，是有很多背信弃义的，但也有很多矢志不渝的爱情啊，不能以偏概全吧？再说孝子，那就更多了，这首歌也就是故作凄凉，好像人活着什么意义都没有，其实狗屁不通。”
<br/>　　叶少阳一口气说完，感觉很舒服，他一向就不太喜欢佛门这种消极的世界观，说起来就一发不可收拾了。
<br/>　　“哈哈哈！”
<br/>　　老和尚一点也不生气，反而狂笑起来，样子很滑稽，“好好好，说的好，痛快！”
<br/>　　叶少阳愣愣地看着他，“你……不反驳吗？”
<br/>　　老和尚摇摇头，“所谓佛法，只是教人超脱，不是向人传道，所谓道，因人而异，善恶，好坏，真假，是非，岂能一概而论？”
<br/>　　叶少阳听得迷迷糊糊，觉得他答非所问，但也不好多问。
<br/>　　这时候，离那座山越来越近了。能看到山下有一座寺庙，青砖红瓦，看上去古色古香。
<br/>　　老和尚带他一路走过去，来到寺庙门外，叶少阳抬眼看去，寺庙正门上方悬着一块黑色的木匾，上面三个古体篆字：法诀寺。
<br/>　　好熟悉的名字？
<br/>　　叶少阳一时没想起来这是哪儿，朝寺门左右看去，两边立柱上都刻着字，凑在一起就是一副对联，上联：大千世界。下联：不二法门。
<br/>　　暗想这寺庙口气倒是真大，用这种对联，一般都是某个宗派最具代表的那种大寺。
<br/>　　老和尚在前面引路，看他不走了，就招手催他进去。
<br/>　　“我是道士，见寺不入，抱歉。”叶少阳拱了拱手。
<br/>　　老和尚笑道：“小天师，这可不是人间寺庙，在这里，就不要拘泥这些了。”
<br/>　　不是人间？
<br/>　　叶少阳讷讷地抬头望着写着寺名的牌匾，心中咯噔了一下，猛然想了起来，法诀寺！无量界法诀寺！
<br/>　　“这里是法诀寺？”叶少阳震惊地看着老和尚。
<br/>　　老和尚朝寺庙上方的牌匾努了努嘴，“这么大的字你看不到吗？”
<br/>　　叶少阳呆了半天，道：“这么说，我们到了无量界了？”
<br/>　　“嗯，嗯，进去吧，菩萨想见见你。”
<br/>　　菩萨……果然啊。这法诀寺是地藏菩萨的道场，也是无量界的核心所在，地藏菩萨虽然经常在阴司呆着，但这里才是他的老家。
<br/>　　自己……为什么会来到这里？
<br/>　　叶少阳正踌躇着，被老和尚一把拉了进去。
<br/>　　从外面看寺庙很小，但一迈过门槛，立刻就像是进了另一方天地，眼前，宛如一座巨大的宫殿，建在一座长方形的高台上，背后就是山。
<br/>　　宫殿前面，是一座巨大的广场，都用金砖铺地，中间一道汉白玉的走廊，一直连到宫殿前面的台阶上，自己此刻就站在台阶上。叶少阳左右看去，两边都不见尽头。
<br/>　　数不清的和尚，或者站在地上，或者盘膝坐在地上，或者漂浮在空中，有的身下还坐着一片巨大的莲叶，人多的数不清，广场两边都看不到尽头。
<br/>　　这些人都闭着眼睛，似乎都入定了，对他们的出现完全没任何反应。前面老和尚催着走，叶少阳就一边走一边打量，发现这些和尚穿什么的都有：
<br/>　　有的是半身红袈裟的喇嘛打扮，有的是头戴法冠、身穿宝衣的禅师打扮，叶少阳看着像是禅宗的。还有头戴金箍的头陀……还有各种奇怪的装束，叶少阳见都没见过的，猜测应该是不同的宗派。
<br/>　　这么大的场面……
<br/>　　叶少阳走到老和尚身边，小声问道：“这些都是大帝的弟子吗？”
<br/>　　“哦，都是历代高僧，死后不愿去轮回的，都来了这里，不过你看到这些，都是入室弟子，诺，你看那个站在所有人前面的，那个是禅宗六祖慧能，哎哎，他很厉害的哦，你以后遇到他可别惹他……”
<br/>　　叶少阳吐血。禅宗六祖慧能！
<br/>　　他哪里是“厉害”两个字就能概括的，简直是神话一般的人物，这名字在人间虽然没几个知道，但他有一首诗人人皆知：身似菩提树，心如明镜台；本来无一物，何处惹尘埃。
<br/>(本章完)
