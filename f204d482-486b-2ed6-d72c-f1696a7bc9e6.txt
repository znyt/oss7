        童筱颖走到了连华生的实验室门口，见实验室的门是虚掩着的，里面依旧传来打印机的声音，于是直接推门而入，想要去把打印机关上。
<br/>
<br/>        应该是刚刚家里停电了，造成打印机突然断电，现在又来电了，打印机才会继续工作。
<br/>
<br/>        那边的窗户未关，从打印机里打出来的资料被风吹得满地都是。
<br/>
<br/>        童筱颖蹲下身去，一张张地将资料捡了起来，同时也忍不住好奇地翻看起资料里的内容来。
<br/>
<br/>        但是，她将资料全部翻了一遍后，发现自己居然一个字也看不懂。
<br/>
<br/>        因为，资料上写的全是摩斯电码。
<br/>
<br/>        这都什么年代了，还用这种电码写资料，难道……这资料里有什么不可告人的秘密吗？
<br/>
<br/>        算了，反正她也看不懂，也不打算去弄懂。
<br/>
<br/>        童筱颖捡起资料后，起身去关了窗户，然后将资料放回了打印机上，并将打印机给关了。
<br/>
<br/>        做完这些举手之劳，她刚一转身，就撞到了一堵肉墙上。
<br/>
<br/>        童筱颖摸了摸额头，站稳脚后抬起头来看了看。
<br/>
<br/>        “大小姐你人真好，还帮我整理资料啊！”连华生话中有话地说，嘴角噙着一抹意味不明的笑意。
<br/>
<br/>        “我只是……”童筱颖欲言又止，瞬间理会到他话中的另外一层意思，慌忙解释道，“我可不是进来偷东西的！”
<br/>
<br/>        “哈哈！大小姐严重了！能让阿沉带入私宅里的人，自然都是值得信任的人。”连华生微笑着说道，然后走到打印机前，将那叠资料用放入了一旁的碎纸机中。
<br/>
<br/>        童筱颖本不该多问，还是忍不住问了一句：“你和大哥，是不是特工？”
<br/>
<br/>        “特工？！”连华生一脸诧异地看着童筱颖，顿了顿后，忍俊不禁道，“这个名词，我喜欢。”
<br/>
<br/>        “你们是不是在调查与制毒有关的案子？”童筱颖又问道，顿时异想天开起来，“是不是像电视里所演的那样，与毒贩周旋？或者说，你们在搜寻贩毒集团的证据！再或许说，你们是在追查什么制毒集团？”
<br/>
<br/>        这并不是她的敏感所致，而是她通过一些事情结合起来得出的结论。
<br/>
<br/>        比如说，她第一次邂逅大哥傅夜沉的时候，他是遭人追杀；再比如说，邂逅大哥傅夜沉之后的那段日子里，他总是来无影去无踪；再比如说，她老家后山那片罂粟花田，还把她哥哥也牵涉到了其中；又比如说……
<br/>
<br/>        他连华生吧！
<br/>
<br/>        没事管她借阅她外婆的《千草集》。
<br/>
<br/>        她们云城西镇那种地方，前方虽然山清水秀，而她外婆家住的那一片深山里，拥有许多叫不出名字甚至未曾被任何资料记载过的奇花异草。
<br/>
<br/>        而且，她们那儿又处于边境，经常有那种东西的走私。
<br/>
<br/>        所以，童筱颖知道这些事情，不足为奇。
<br/>
<br/>        但连华生却一脸不可思议地看着童筱颖。
<br/>
<br/>        这小女人脑子里的想象力怎么这么丰富？不过，她也猜得八九不离十了。这种事情，除了内部的人，他无法跟任何外人暗示或者明示。
<br/>
<br/>        好在，她童筱颖不是敌方派来的卧底。
<br/>
<br/>        他和傅夜沉，确实被顶级大上司派下了秘密任务。
<br/>
<br/>        傅夜沉虽然表面上是退役从商，但他内在仍旧有任务在身。
<br/>
<br/>        而他连华生的任务，就是辅佐傅夜沉，听从傅夜沉的一切安排，顺便给自己赚赚外快，攒钱养老过日子。
<br/>
<br/>        “大小姐，有句话，我得告诉你。阿沉非常的信任你，那么你在私宅里的所见所闻，全都得守口如瓶。噢！对了！就连你的丈夫，傅昼景那儿也不可透漏一个字！能做到吗？”连华生微笑着问道。
<br/>
<br/>        童筱颖点了点头。
<br/>
<br/>        随后，连华生去那边抽屉里取来了童筱颖的外婆的《千草集》和《行医日记》，将这两本手札还给了童筱颖。
<br/>
<br/>        童筱颖小心翼翼地接过外婆的遗物，捧在怀里。
<br/>
<br/>        连华生突然想起了什么，又从裤兜里掏出一把钥匙递给了童筱颖：“可否帮我把这把钥匙藏起来！这把钥匙关系到我和阿沉的生死，你最好藏到一个我不知道的地方。藏好后，对任何人都不可以提起这把钥匙的事情。”
<br/>
<br/>        “交给我去藏？！”童筱颖怔愣。
<br/>
<br/>        连华生会心一笑，意味深长地说道：“阿沉的那把，傅老太爷已经转交给了大少奶奶。而我的这把，我交给你，因为我相信你。你会守口如瓶的，对吗？”
<br/>
<br/>        童筱颖垂眸看了看连华生递过来的钥匙，又抬眸看了看连华生的眼睛。
<br/>
<br/>        她犹豫了一会儿，才从连华生的手中接过钥匙。
