        &lt;h3&gt;第三卷第三十二章  欢乐三人行1&lt;/h3&gt;
<br/>
<br/>        陈子昂到洛城已经下午了。
<br/>
<br/>        江俞轩在酒店等他们。
<br/>
<br/>        “俞轩，你们聊得怎么样？”
<br/>
<br/>        “你这个老同学是个人才啊，对这方面的见解很独到啊，思想也比较超前。”
<br/>
<br/>        “他就是学的这个专业的。”
<br/>
<br/>        “你很了解他啊。”
<br/>
<br/>        “当初我们准备考一个专业。”
<br/>
<br/>        “江助理，这个项目有没有意向？”陈子霖问江俞轩。
<br/>
<br/>        “能聊得来，但是还没有说有没有意向，我们聊了一上午信息上的事情，听了傅总的意见。”江俞轩如实回答，在工作上他还是很认真的。
<br/>
<br/>        “那他什么意思？”
<br/>
<br/>        “项目资金已经审批下来了，要做他就要做到最好，后期看各家的方案，优先选择。”
<br/>
<br/>        “这样好，这样公平。”陈子霖点点头。
<br/>
<br/>        江俞轩和陈子昂相互看一眼，有马总和陈子乔在，公平存不存在呢？
<br/>
<br/>        马总和陈子乔是他们的人，当然后期还要看方案，傅海生说了，优先选择。
<br/>
<br/>        “你知道项目预算资金是多少吗？”陈子昂问江俞轩。
<br/>
<br/>        “一千二。”
<br/>
<br/>        “有得做。”陈子霖说
<br/>
<br/>        “是，一个县的这种项目确实空间很大。”江俞轩敲着桌子，发出哒哒的声音。
<br/>
<br/>        “按照正常的利润点走，保证合理收益，不出格。”陈子昂又开始叮咛了。
<br/>
<br/>        “后期还是会招标的，花落谁家也不到一定。”江俞轩看了一眼陈子昂。
<br/>
<br/>        “嗯，下午还见傅海生吗？”
<br/>
<br/>        “你觉得呢？”
<br/>
<br/>        “我觉得你还是有必要再和他聊聊。”
<br/>
<br/>        “那你联系他吧。”
<br/>
<br/>        “你不联系？”
<br/>
<br/>        “我拿什么和他联系？”他们分开的时候，傅海生已经喝醉了，就没有给陈子昂留电话。
<br/>
<br/>        “哦。”江俞轩给傅海生打电话，响了好几声电话才接通。
<br/>
<br/>        “傅总，你好。下午你还有时间吗？我们陈总回来了，想下午见见你。”
<br/>
<br/>        “明天吧，我现在在去泉城的路上。让陈总等一下我好吗？”
<br/>
<br/>        “他让你等一下他，他去泉城了，明天见你。”
<br/>
<br/>        “好吧，那下午我们去转转，我都没有来过洛城呢。”
<br/>
<br/>        “子昂，子昂，你太好了，我一个人呆得都发霉了。”江俞轩好看的桃花眼泛滥成灾。
<br/>
<br/>        “江俞轩，你才呆了一上午。”陈子霖总是看江俞轩不顺眼，一个大男人，时不时的在他妹妹跟前卖个萌。
<br/>
<br/>        “我工作了一个上午好不好？”江俞轩都不知道为什么陈子霖看他不顺眼。
<br/>
<br/>        “哼。”陈子霖扭头不理江俞轩了。
<br/>
<br/>        “哥，你去不去？”
<br/>
<br/>        陈子昂也不知道陈子霖见了江俞轩会那么大的气。
<br/>
<br/>        “我当然去，我为什么不去？”陈子霖没有想到妹妹这样问他。
<br/>
<br/>        “子昂，你哥哥好像对我有意见？”走在路上，江俞轩悄悄的在陈子昂耳边说。
<br/>
<br/>        “没有吧？”陈子昂不确定的看了陈子霖一眼。
<br/>
<br/>        “你们说什么呢？神神秘秘的？”陈子霖看见江俞轩和妹妹那么亲密的样子，直接拿肩膀一抗挤开江俞轩。
<br/>
<br/>        江俞轩又在陈子昂的左耳边说：“看见了没？就差点没揍我了。”
<br/>
<br/>        “你们说话能不能不要背着我？”陈子霖又挤在俩个人中间，这一次将江俞轩直接就挤了一个踉跄。
<br/>
<br/>        “陈子霖，你故意的吧？看我不顺眼是吧？我吃你的了还是喝你的了？”
<br/>
<br/>        “你离子昂那么近干什么？”陈子霖也不示弱。“不好好干好自己的本职工作。”
<br/>
<br/>        “呵呵，我天天都离陈子昂这么近，你能咋的？”江俞轩这才发现这货对他有意见的原因。
<br/>
<br/>        “子昂，你看他嘚瑟的，你离他远点。”
<br/>
<br/>        “哥，他是我们公司的副总，你的位置都在他之下，你让我离他远一点？”陈子昂也觉得哥哥有点无理取闹了。
<br/>
<br/>        “副总，不是你的助理吗？”
<br/>
<br/>        “我们俩个在一起搭档了很多年了。”
<br/>
<br/>        “你们这关系也太亲密了，我以为他对你有想法。”
<br/>
<br/>        “陈子霖，我是她哥，比你还亲的亲哥！”江俞轩大声对着陈子霖的耳朵喊。
<br/>
<br/>        “那你咋不姓陈？”陈子霖弱弱的问了一句。
<br/>
<br/>        “陈子霖，告诉你，我不姓陈，我也比你亲，这几年陪在她身边一起闯荡的是我，你这个便宜哥哥去哪里了？得了便宜还卖乖！”江俞轩被陈子霖的那一挤，是丝毫不会顾及陈子霖心理阴影面积有多大。
<br/>
<br/>        陈子昂看着他们俩个吵吵闹闹的竟然觉得无比的幸福。虽然她的个人感情世界一片苍白，但是身边的人对她这么好，人生如此？何所求？
<br/>
<br/>        看着俩个帅气的男人和小孩子一样站在阳光底下吵闹，竟然无比的养眼，江俞轩的1.78的个子，因为工作的原因，穿着工作服，打着领带，黑色的皮鞋，显得干练而精神，那张帅气的脸上满是洋洋得意嘚瑟的表情。
<br/>
<br/>        陈子霖1.85的个头，比江俞轩高一点，虽然很是无语，但是却是居高临下的看着江俞轩，一副：你再嘚瑟看我不收拾你的表情。
<br/>
<br/>        “你们俩个走不走？不走我就走了啊。”
<br/>
<br/>        “哼！”江俞轩对着陈子霖哼了一声，转身就跑，跑到陈子昂的身边，毫不犹豫的拉起陈子昂的手回头对着陈子霖摇了摇。
<br/>
<br/>        陈子霖的大长腿两步就追上他们，也拉起了陈子昂的手。
<br/>
<br/>        “你俩幼稚不？”陈子昂好笑的问。
<br/>
<br/>        “不幼稚！”俩个人异口同声。
<br/>
<br/>        “子昂，我们去哪里？”
<br/>
<br/>        “听说那边有个二龙山，风景不错，洛城的书画院就在哪里。”陈子霖说。
<br/>
<br/>        “俞轩，那我们去哪里看看？”
<br/>
<br/>        “随你，你去哪里我就去哪里。”其实，江俞轩知道陈子昂的计划，如果她去甸城了，陪在她身边的日子就不多了，甸城离京城是比较远的，来来去去不方便。
<br/>
<br/>        三个人一路朝二龙山走去，二龙山其实就是一个小山头，周边林荫密布，群鸟和鸣，有一条弯弯曲曲的小河，河水清澈透明，一群群的鱼儿悠闲自在的摇头摆尾，山清水秀、空气清新、一阵风来，清爽惬意。
<br/>
<br/>        陈子昂深深的吸了口气，好久都没有这么轻松自在过，从到北京之后，她一直都是在奔跑着，哪里有心情这么悠闲的看山看水？
<br/>
<br/>        络城书画院就藏在这山水之间，一副巨大的广告墙面上都是各书画家的简介。
<br/>
<br/>        陈子昂站在山头，看着脚下弯弯曲曲的河流绕山而去，在阳光下闪着凌凌波光。
<br/>
<br/>        “子昂，子昂，你快来，我看见你了，我看见你了。”陈子霖大叫。
<br/>
<br/>        “陈子霖，子昂就在这里，你发神经了？”
<br/>
<br/>        “子昂，过来，快点过来。”陈子霖不理江俞轩。
<br/>
<br/>        “哥。怎么了？”
<br/>
<br/>        “你看这里！这里有你的名字。”
<br/>
<br/>        广告墙左上角一块a4大小版面的地方，有几首小诗，还有简介；
