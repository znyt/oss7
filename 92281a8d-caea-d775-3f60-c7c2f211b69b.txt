　　第63章
<br/>　　柳芷捏着手指立起来，忐忑不安道：“大当家、主人、老爷……洗脚水已经打好，我伺候你洗脚吧……”
<br/>　　一连换三个称呼，连她也不知该如何称呼程大雷，说到最后，头已经垂下来，呆呆的看着自己鞋面。
<br/>　　程大雷一身瀑布汗。
<br/>　　他没有这个命，也享不了这个福。
<br/>　　即便以挑剔的眼光看，柳芷也绝对称得上标致。身体纵然还没有完全发开，但身材已经不矮，脸颊上的婴儿肥也正在褪去，一双眼睛从懵懂也快要布满水雾。再有几个年头，一定能出落得亭亭玉立。
<br/>　　可那是日后……
<br/>　　日……后……
<br/>　　好吧，不得不承认，程大雷的心思邪恶了一把。
<br/>　　如今的柳芷才十六岁，在程大雷心底还是个孩子。
<br/>　　“这个……哈……柳家姑娘，我觉得你可能是误会了，天色不早，你还是回自己房间休息吧。”
<br/>　　“大当家不用奴陪床么？”
<br/>　　“不用，绝对不用。”程大雷坚定拒绝，对方一口一个奴，一口一个陪床，很容易把自己拉入罪恶深渊。
<br/>　　“那大当家是嫌弃奴的身子么？”
<br/>　　“……”
<br/>　　程大雷目瞪口呆，说着说着，柳芷竟然落下泪来，啪啪打在手上，让程大雷觉得自己仿佛造了天大的孽。
<br/>　　不要妄图去理解一个青春期少女，尤其是一个跨时代的青春期少女。程大雷一咬牙一跺脚，你不走我走，再待下去，他生怕自己会把持不住。
<br/>　　“大当家，若是走了，奴也只有一死了之。”
<br/>　　程大雷回过头，见柳芷举着金钗抵在喉咙上，一双眼泛起水雾。
<br/>　　这一幕何其熟悉，苏樱这样做，不让自己靠近她，柳芷这么做，却是让自己不要不靠近他。
<br/>　　是自己撞鬼了么！
<br/>　　“这个……咱们之间是不是有什么误会，你先把东西放下，这个一哭二闹三上吊的法子就是你们传下来的。”
<br/>　　“大当家定是嫌弃我曾身在青楼，陪人喝酒卖笑，奴只好一死了之。”
<br/>　　“你误会了，我没有嫌弃你，只是你太小了。”程大雷摊开手，如这样的美人坯子，前生怕是连多看自己一眼都不肯，自己怎么有资格嫌弃对方。
<br/>　　“哪里小？”
<br/>　　“……”
<br/>　　“大当家是顶天立地的丈夫，自然不会和世间的俗人一样。可你今日走出这房间，旁人怕会所大当家看不上我，柳芷便是给人为奴为仆都没人要。”
<br/>　　“呃……”
<br/>　　程大雷再次无语，这人的脑回路还真是清新出奇啊。不过，她说得也未必错，人言可畏，以这个时代的观念来说，真会发生这样的事情也说不定。
<br/>　　他眼珠一转，想出个主意，铺了两床被子，将炕桌放在中间。两人各自睡在炕桌两边。
<br/>　　“先说好了，晚上不要偷偷趴过来，我这个人，很有底线的。”
<br/>　　程大雷啪地吹熄了蜡烛，房间里久久没有谈话声，只有二人均匀的呼吸。
<br/>　　“大当家……”
<br/>　　“还没睡么？”
<br/>　　“嗯，柳芷以前没做过伺候人的事，大当家有没有要我注意的？”
<br/>　　“注意……嗯，倒真有一件事，以后要注意多喝牛奶。”
<br/>　　“山寨没有牛。”
<br/>　　“呃……那就多吃木瓜。”
<br/>　　“什么是木瓜？”
<br/>　　“多做扩胸运动。”
<br/>　　“什么是扩胸运动？”
<br/>　　“呃……睡吧，睡吧。”
<br/>　　……
<br/>　　次日天明，天刚蒙蒙亮，山寨便响起锣声，大家被召集到校场上。
<br/>　　这些人刚睡醒，还有些迷糊，不知道程大雷这么早将自己叫起来做什么。
<br/>　　“训练。”程大雷惜字如金。
<br/>　　训练，什么训练……这帮人脑袋里稀里糊涂。
<br/>　　然后，这帮人立刻明白，程大雷所谓的训练是什么意思。
<br/>　　加入蛤蟆寨的人有92人，这里面老弱妇孺占多半，真正有战斗力的成年男子在四十人左右。
<br/>　　程大雷将所有人分成甲乙丙丁四个等级，并未每个人准备了统一款式的制服，当然，也可以称之为军装。之后，又将这些人分为若干小队，每个小队有一名队长管理，采用连坐的方式，一人违规，全队受罚。
<br/>　　训练的项目，并不复杂，无非是站军姿，走正步，跑操，程大雷把前世军训时接受的训练搬了过来。对于这些人而言，根本谈不上累，但是……很枯燥。
<br/>　　统一的服装，枯燥而不断重复的行为，再加上一些没什么意义，却必须喊得震天响的口号，比如『我爱山寨，山寨爱我』之类的……
<br/>　　这些加在一起，终究只有一个目的，统一他们的思想，让所有人对自己的命令言听计从。
<br/>　　程大雷既不会练兵，也不会统军，他前生没有任何这方面的经验。但在这个信息不畅，生产力落后的时代，多去几个城市就称得上见多识广。
<br/>　　而程大雷拥有得，则是五千年人类积攒下来的经验，每一条微不足道的经验得到，背后不知就有多少血汗。
<br/>　　训练一开始，众人实在苦不堪言，但在连坐惩罚的方式下，比如加重训练量，不许吃饭之类的，众人倒是没人敢闹事。最悲催的是，到晚上程大雷还将众人聚在一起，围着篝火谈你们为什么爱山寨。
<br/>　　说，你为什么不说，难道大家都爱，你不爱么。
<br/>　　大家绞尽脑汁的想，心中万分悲催，我特么到底为啥要爱山寨！
<br/>　　有些事情比想象中容易，程大雷知道在国外曾有一个试验，一中学教师在班级内展开为期一周的极权试验，其后果……令人恐怖。
<br/>　　七天后，这些人一改之前的涣散，行为动作变得整齐划一，这对于个人来说并非好事，但对于一个集体而言，某种力量正在凝聚。
<br/>　　这一天有发生了一件事，守山的秦蛮说，山下有两个人慕名前来加入。
<br/>　　程大雷赶紧将两人请到山上，问过才知道，这二人是一对兄弟，哥哥叫廖甲，弟弟叫廖乙，是因为听到程大雷的名声，想来蛤蟆寨入伙。
<br/>　　这倒是大大出乎程大雷的预料，山寨开张这么长时间，还是第一次有人来入伙。
<br/>　　他想来想去，忽然想到一个原因，可能最接近真相。
<br/>　　山寨大旗的招募技能发动了。
<br/>(本章完)
