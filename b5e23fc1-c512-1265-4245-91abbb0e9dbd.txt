　　第267章
<br/>　　双方大眼瞪小眼，同时都愣住了。程大雷此时也将对方认了出来，只是一时有些不敢相信，堂堂帝国六王子，怎么会落魄到如此模样，披头散发，满脸污垢，活脱一个人猿泰山。
<br/>　　“自己人，自己人，都认出来了还压着我做什么。”李行哉挣扎道：“放我起来！”
<br/>　　程大雷挥挥手，秦蛮和赵子龙这才将二人放开。
<br/>　　李行哉的精神头相当不错，站起来后又蹦又跳，冲李婉儿道：“小妹，你怎么和这山贼搞在一起，你被他绑架了！”
<br/>　　“说来话长，六哥还是先说说你吧。”李婉儿忽觉得有这样一个兄长是件丢人的事情。
<br/>　　“我的话说来就更长了。”李行哉负手长叹一声，但因为身上破破烂烂，委实也没有什么气质。
<br/>　　程大雷目光转移到另外一人身上，道：“你套索玩得不错，戎族人？”
<br/>　　李行哉兴奋的拍拍对方肩膀：“福德勒，我的朋友。”
<br/>　　程大雷没见过福德勒，他对帝国没感情，对戎族自然也谈不上什么深仇大恨，只是隐隐约约觉得，对方看自己的眼神有些畏惧。
<br/>　　可其他人和程大雷的心情不同，一听对方是戎族顿时如临大敌，各自已经握住兵器。
<br/>　　“别紧张，别紧张。”李行哉挥舞着双手：“这是我朋友，我的兄弟，他不是坏人，不是坏人。”
<br/>　　李行哉这话照道理说没有什么毛病，戎族也是有好有坏，福德勒未必就是坏人。
<br/>　　但他不是坏人，我们是啊。
<br/>　　“有吃得没，有穿的没？”李行哉道：“赶紧的吧，我们都饿好几天了。”
<br/>　　程大雷发现认识李行哉，真的是件很丢人的事情。
<br/>　　大道上，李行哉吃得狼吞虎咽，仿佛是饿死鬼投胎转世一般，诸人看着他的模样，都有些无语。
<br/>　　“六哥，您怎么到这里来了？”李婉儿捧着水囊。
<br/>　　“还能怎么过来的，逃过来的。”李行哉道：“在西北我被抓了，打完仗后逃到并州，又从并州逃到青州，在青州又差点被抓了壮丁，妈的，所以我又逃到这里了。”
<br/>　　“你为何不回长安，所有人都在找你。”
<br/>　　“我打算再走走，再看看。”李行哉边吃边说，突然噎得直咳嗽，李婉儿忙把水递过去。
<br/>　　“你们是怎么回事？”李行哉问：“打算去那哪儿？”
<br/>　　“嘿，你还不知道吧。”程大雷抖抖身上的官服：“我现在可是朝廷亲封的边关守将，现在正要去琴川关走马上任。”
<br/>　　“嘿，刚才光顾着吃了，可是没注意到，你穿这一套还挺精神。”李行哉终于舍掉将手中的干粮放下，忽然想起什么，看着程大雷道：“你刚才说要去哪儿？琴川，我劝你不要去？”
<br/>　　“琴川怎么了？”程大雷问。
<br/>　　“我们刚从那边逃过来的，你看看我现在这个样子，还不明白什么么？”李行哉。
<br/>　　“嗯，很符合你现在的气质。”
<br/>　　李行哉撇撇嘴，用树枝在地上画了一条线，一边画口中一边道：
<br/>　　“琴川八百里，是西北最重要一道防线，出了琴川关就是戎族草原，向来这条道都是东西通商要地。”
<br/>　　这李行哉不愧是接受精英教育出来的，寥寥几笔就画出琴川的大概地貌，其他人也看不太懂，程大雷却是极为认真。他对地图有某种莫名的迷恋。
<br/>　　“关内的丝绸，茶叶，铁器运到关外价值万金，关外的皮草，玉石，骏马在关内也是稀罕货。不过帝国不许和戎族做生意，所以这条商道就被禁止了。”
<br/>　　程大雷点点头：“那琴川关危险在哪里？”
<br/>　　“现在占据琴川关的人名叫莫鸣米，原先是一伙啸聚山林的山贼，不过他现在被招安了，封为琴川副将，整个琴川关在他手中控制。”
<br/>　　“山贼么？”
<br/>　　一听这话，众人都笑了，大家本来就是山贼，连戎族都打过，何况几个山贼。
<br/>　　“那看来是同行了，该让他们明白谁是祖师爷。”
<br/>　　“教教他们怎么样做人做事。”
<br/>　　李行哉愣愣的看着这一幕，心道：程大雷都带出一帮怎样的手下。
<br/>　　“莫鸣米手下有一万余人，你们准备怎么教他做人的道理？”李行哉。
<br/>　　众人皆无语了。
<br/>　　程大雷问：“是不是莫鸣米控制了这条路，做些走私的生意，如果真是如此，的确是不会允许别人染指。”
<br/>　　“倒也不是。”李行哉道：“琴川关发现了铁矿，他劫掠附近的平民为他采矿，卖给方圆的商行谋取重利，这些平民吃住都在矿井内，诸多人都是活着进，白骨出。我和福德勒也被他抢了进去，好不容易才逃出来。”
<br/>　　“那这是血汗工厂呐。”
<br/>　　“他们竟敢私采铁矿，六哥，你是要去调兵攻打他们么？”李婉儿。
<br/>　　“嘿，哪有那么容易，其中利益盘根错节，我根本无兵可调，而且以莫鸣米一人的本事，玩不转这么大盘子，他背后肯定还有人，只是不知道是谁。”李行哉。
<br/>　　程大雷隐约已经有些明白，采铁可是重利，这生意莫鸣米怎么会让自己染指，现在琴川关一万兵丁，自己去了就是一个死。而程大雷大概已猜出，莫鸣米身后之人是谁。
<br/>　　“那咱们就先不去琴川。”程大雷道：“找个地方扎营，接下来看看这山里有什么地方合适建山寨，日后咱们就在这里安家了。”
<br/>　　“大当家，新的山寨有什么要求么？”徐神机问。
<br/>　　“当然有。”程大雷口中吐出一句话：“要有水。”
<br/>　　“如果你们需要一个新的山寨，我却是知道一个地方。”李行哉道。
<br/>　　“咦，你有知道？”程大雷睁大眼睛：“原来你就是这次的指路npc，说说吧，你知道什么？”
<br/>　　“什么叫npc？”李行哉有些糊涂，晃晃脑袋道：“我说的地方是原先莫鸣米的山寨，或者可以说是水寨，没有其他特点，就是水多。”
<br/>　　“水多才好搞么。”程大雷道：“地方在哪儿？”
<br/>　　“地方是很好找，只不过怕是不太好搞，现在也是被占着。”
<br/>　　“被什么人占了？”
<br/>　　“不是人。”李行哉道：“是蛇……”
