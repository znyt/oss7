　　“这头魔龙，桀骜不驯，也该让他吃一个亏了。”
<br/>　　虚魔王看着楚风眠的目光闪烁，似乎想要报复，最终还是叹了一口气，放弃了这个想法。
<br/>　　“算了，我们再去找其他人联合。”
<br/>　　毕竟龙摩罗，可不是弱者，一身实力堪称九劫古帝中的霸主，除非是半步天人境的强者，不然谁也不愿意轻易得罪他。
<br/>　　这夜魔王，虚魔王，吃了这一个亏，也是只能再去寻找其他联手的人选。
<br/>　　另一边，在拒绝了夜魔王的邀请之后。
<br/>　　又有几名魔族强者，前来邀请楚风眠，想要让楚风眠加入到他们的团体之中去，一同是联手进入佛寺之中探索。
<br/>　　无一例外，都被楚风眠拒绝了。
<br/>　　“这条魔龙，真是狂妄自大从，迟早要吃个大亏！”
<br/>　　一尊魔族之中强者，被楚风眠拒绝之后，也是脸色一寒的离开了。
<br/>　　连番拒绝了不少邀请之后，楚风眠便是站在了距离这佛寺很近的一座山峰之上。
<br/>　　周围的无数魔族强者，也没有任何一人，还敢来靠近楚风眠了。
<br/>　　楚风眠安静了下来，也在仔细观察这佛寺之上的变化。
<br/>　　半个时辰的时间。
<br/>　　这佛寺之上的佛门禁制，已经是越发的凝聚显现出来，在场的气氛，也变的认真了许多。
<br/>　　无数魔族强者，都是已经联络好的帮手，如今一脸凝重的看着净土佛寺。
<br/>　　在净土佛寺之上，已经是随时可能开启了。
<br/>　　“这龙摩罗，真的以为他一个人，可以占到什么便宜？哼，此人如此狂妄，我们不如杀了他，吞了他的精血，也可以得到巨大的好处。”
<br/>　　就在距离楚风眠的不远处，一尊浑身黑袍的魔族，死死的盯着楚风眠的看向，眼神之中尽是贪婪之色，跟旁边的人交谈道。
<br/>　　在这黑袍魔族的身上，散发着一种浓浓的血腥味，这是一尊血魔。
<br/>　　在魔族之中，血魔一族却是极为独特的存在，他们是以武者妖神，甚至是魔族的鲜血为食。
<br/>　　吞噬的血液越强，他们的实力也就越强，所以血魔的实力，有低有高。
<br/>　　这一位血魔身上的气息，已经是逼近了九劫古帝的巅峰，在血魔一族之中都是位列王者的人物。
<br/>　　魔龙一族，血脉尊贵，对血魔来说正是最好的大补之物，他盯着楚风眠，眼神之中的贪婪之色，毫不掩饰。
<br/>　　“龙摩罗，可不好对付，此人的实力，我等单打独斗，谁也不是他的对手，不过这一次，的确是杀了他的好机会！”
<br/>　　另一名魔族，在一旁开口道，正是之前想要联合楚风眠的夜魔王，他的脸色，在提到龙摩罗的时候，也是无比阴沉。
<br/>　　显然是上一次楚风眠直接拒绝了他，让他颜面尽失的事，他还记忆犹新。
<br/>　　在听到这个提议之后，夜魔王的眼神之中也露出了几分异样光芒，考虑道。
<br/>　　“这龙摩罗，死在这佛寺之中，就算是魔龙一族的人，也不会知道，倒时候我们说杀了他，也没有人会来找我们麻烦。”
<br/>　　“都说龙摩罗强，我倒是要看看，此人到底能够比我们强横到哪里去，魔龙一族的血脉啊，我们三人平分，都是巨大的好处。”
<br/>　　那另一位虚魔王，也是冷声的开口道。
<br/>　　“等到了佛寺之中，我们可以试探一二，要是这龙摩罗是强者，我们三人便是退去，反正我们三人联手，这龙摩罗也奈何不了我们。”
<br/>　　血魔王目光看向楚风眠，冰冷的开口道。
<br/>　　“但若是这龙摩罗，只是一个徒有虚名的废物的话，那我们就杀了此人，夺去了此人的精血！”
<br/>　　“好，就这么办！”
<br/>　　虚魔王，夜魔王都是点了点头。
<br/>　　另一边，楚风眠已经是感受到了一股杀意，在死死的盯着他。
<br/>　　“想要杀我？”
<br/>　　楚风眠心中露出几分冷笑。
<br/>　　这血魔王，夜魔王等人，打算是杀了龙摩罗，来夺取龙摩罗一身的战龙精血。
<br/>　　可他们千算万算也想不到，龙摩罗早就被楚风眠杀了，真正实力，楚风眠可是要比起龙摩罗还要强横。
<br/>　　魔族之中九劫古帝级别的王者，一身精血蕴含的灵力也是无比雄厚。
<br/>　　虽然这精血之中，蕴含魔意，对寻常武者来说，是剧毒之物，哪怕是得到，也只能够是退避三舍，不敢炼化。
<br/>　　但是楚风眠可是有着吞天兽的血脉，吞天秘术，吞噬诸天，小小的魔意，又怎么可能抵挡得住吞天秘术。
<br/>　　哪怕是龙摩罗身上的魔意，都影响不到楚风眠，这些人身上的魔意，更是算不得什么。
<br/>　　只要是他们敢来动手，楚风眠就杀了他们三人，正好是以他们的精血，化为楚风眠的力量，为楚风眠踏入九劫古帝做准备。
<br/>　　不止是夜魔王等人。
<br/>　　在场的魔族，如今都在互相算计，各怀鬼胎。
<br/>　　魔族之中的争斗，比起武者之间，还要残酷，互相杀戮吞噬，几乎是家常便饭的事。
<br/>　　能够踏入九劫古帝，每一尊魔族强者的手中，都是沾满了鲜血。
<br/>　　出手杀人，对他们来说眼皮子都不需要眨一下。
<br/>　　楚风眠一脸淡然的，看着这一切。
<br/>　　在这凝重的气氛之下，时间一点点过去。
<br/>　　突然之间，在这佛寺之上，一道七彩霞光，冲天而起。
<br/>　　这七彩霞光的出现，楚风眠熟悉无数，他已经是见到过不少次了。
<br/>　　净土佛寺，开启了！
<br/>　　轰隆隆。
<br/>　　一股股的魔意，在这净土佛寺之中，涌现了出来。
<br/>　　这净土佛寺，跟外观看上去的金碧辉煌，截然不同，大门打开的一刻，魔意翻腾，鬼哭狼嚎。
<br/>　　如同一方魔域，其中带有着极大的怨气。
<br/>　　“好多的怨气，这里面，到底是死了多少的强者，这些怨气，经历数十万年的时间，非但是没有消散，反而是化为了怨灵一般的存在？”
<br/>　　楚风眠目光看了过去。
<br/>　　在这净土佛寺之中，他就看见了其中的数道人影。
<br/>　　这些人影，半虚半真，如同鬼魅，身上却带怨气冲天，乃是强者怨气，化为了怨灵。
<br/>　　在这净土佛寺之中，这样的怨灵，只怕是足有千万。
