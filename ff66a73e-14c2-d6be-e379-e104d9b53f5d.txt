　　　　寻找片刻，陆雪琪困惑，道：“这里貌似有些问题，你快过来看看。”
<br/>　　慕清霄走上前来，眼神一凝就看出此处有一隐秘的幻境阵法，必然是这里没错，就在准备进入滴血洞的时候，海面上忽然飘来一道绿色的倩影。
<br/>　　“有人漂过来了，这身影怎么有点熟悉？”不仅是慕清霄，连陆雪琪也第一时间发现了飘在海上的身影。
<br/>　　“遇见过好几次了，自然有些熟悉。”
<br/>　　慕清霄瞥了一眼，就认出飘在海面上的是鬼王宗的碧瑶，不过此时她处于昏迷状态，脸色也有些苍白。
<br/>　　“先将她捞上来在说吧。”
<br/>　　“嗯。”
<br/>　　陆雪琪闻言点了点头，虽然她表面上冷若寒霜，但心却很善良，身影飘向无情海，踏浪而行，很快就将碧瑶提回了岸上。
<br/>　　慕清霄握住她的手腕，发现伤势并不严重，只是昏迷了过去而已，用不了多久便能醒过来。
<br/>　　“小伤，黑水玄蛇已死，这里没有什么危险，她的队伍想必很快就会追上来，我们先走吧。”
<br/>　　说罢，牵起陆雪琪径直朝里面走去，很快两人便被环境吞噬，不过，此等幻境对他来说，只能算是小巫见大巫。
<br/>　　空间荡起一阵涟漪，两人脱离环境，视线在四周看了一圈，发现自己已经深处一处闷沉的山洞中，洞内漆黑一片，不知道通往何处。
<br/>　　慕清霄眼中一喜，忍不住在陆雪琪嫩滑的小脸上亲了一口，惹得她脸色绯红：“雪琪，你真是旺夫命，这里如此隐秘，肯定是滴血洞没错了。”
<br/>　　走在狭窄的洞穴中，转过拐角，呈现在两人面前的是一条完全不同的通道。
<br/>　　前方是一条长廊，宽敞许多，两侧的石壁上依然散发着荧光，把其中照耀的极其明亮，但经过长年累月的积累，脚下的灰尘极厚，一脚踩上去就留下一只明显的脚印。
<br/>　　踩在积满灰尘的路上，在中间留下一连串的脚印，片刻后，慕清霄抬头看了一眼，微笑道：“看来，我们已经抵达滴血洞的尽头了。”
<br/>　　此时，哗哗作响声传入耳中，不过多时，果然看到前方通道尽头从洞顶直挂下一幕水帘，水花四溅，晶莹美丽，最后落到通道尽头一个小水潭中，若不是在这绝地之中，倒也不失为一道风景。
<br/>　　“这万蝠古窟中，真是气象万千。”望着映入美眸中的水帘，陆雪琪不禁感叹道。
<br/>　　只是随意瞥了一眼，四周除了岩壁还是岩壁，也唯有水帘比较特殊，两人直接闯过水帘，瀑布后面便是坚硬的石壁，与通道两侧的石头没有什么不同。
<br/>　　小水潭更是清澈见底，也不见水流流向何处，在这狭小的空间中，只怕是渗入了地底，而在上方，滴水的地方更是在一片石壁洞顶，不知怎会布满水珠，不停滴落，根本没有出路。
<br/>　　陆雪琪看向洞顶，仔细观察，脸色有些古怪，声音清冷，道：“这死灵渊怪事真多，你看这洞顶石壁上有几块红色的地方，水珠流过，都被映成血液一样。”
<br/>　　顺着陆雪琪的视线，向洞顶望去，果然透过水珠，在洞顶石壁上共有七块半个巴掌大的红色石头镶在洞顶，石质纹理与旁边的石头一样，只是颜色不同。
<br/>　　只是，洞顶七块红色石头扭扭捏捏地布置在洞顶，看上去倒是向古怪的勺子形状。
<br/>　　尤其是颜色，也不知在这洞中被水冲刷了多少岁月，依然殷红如血，甚至连晶莹的水珠流过这些红石时，都被它映成了像鲜血一样的红色，然后滴滴落下，宛若鲜血滴落。
<br/>　　慕清霄嘴角牵起一丝笑意，道：“我们没有找错地方，雪琪，你看这水滴滴落的时候像什么？”
<br/>　　“像鲜血...”陆雪琪猛然惊醒，道：“难道，这里便是滴血洞？”
<br/>　　当今之世，魔教中四大派系为首并立，分别为合欢派，万毒门，长生堂，鬼王宗，但若论到声势之盛，却无一比得上当年盛极一时的炼血堂。
<br/>　　而在魔教之中，这八百年来，一直传说当年正魔大战之后，炼血堂主要首脑虽然尽皆战死，但多有密宝法器被收藏于炼血堂根基之地‘万蝠古窟’地下一个叫“滴血洞”的秘密所在。
<br/>　　这八百年来，不知道有多少魔教中人暗中偷下万蝠古窟，甚至连死灵渊也被找了个遍，但都是空手而归。
<br/>　　“倘若这里真是传说中的滴血洞，可我们要如何进去？”
<br/>　　慕清霄微微一笑，瞥了一眼水潭，此处多出一座水潭，明显是多此一举，但恰巧出现在这里，明显就是告诉他，水潭有猫腻。
<br/>　　后退一步，凝视水面，水波荡漾，果然有七颗红色石子渐渐浮现在水潭中，因为是倒影的关系，此刻它们的排列，已不再是古怪的勺子形状，反而有点像是一个人的手掌。
<br/>　　慕清霄指了指水面上的倒影，陆雪琪俯身看去，只见水面上水波荡漾，只需集中精神，便慢慢看清那七颗红色石子在水中如手掌一样的倒影。
<br/>　　“这就是开启滴血洞的方法了。”
<br/>　　话语刚落，慕清霄就毫不犹豫的走进水潭中，陆雪琪看了看水潭，心中有些担忧，一同走了下去。
<br/>　　慕清霄牵起她的玉手也不说话，感情在无声中升华，滴落的水珠打湿了陆雪琪的白衫，粉色内衣若隐若现，但她却不曾在意，只是静静的跟在他的身后。
<br/>　　片刻后，水波逐渐平息，慕清霄向岸上看去，只见那一颗颗如珍珠般晶莹的水珠从空中轻轻飘落，落在这空灵清绝的美丽女色秀发上，脸颊与衣衫上。
<br/>　　透明清澈的水珠，自她乌黑的发梢滑落而下，慢慢流过她雪白的肌肤，令她原本就美丽的脸蛋显得更加诱人。
<br/>　　慕清霄一时间竟有些痴了，只觉得洞中原本哗哗流水声渐渐远去，在他眼中只有面前这位站在谭中，宛若出水芙蓉仙子般的女子，带着动人心魄的美丽扑面而来。
<br/>　　“雪琪，你真美。”
