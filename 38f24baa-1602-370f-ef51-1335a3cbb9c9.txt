　　这个人形生物与一般的人类并没有什么两样，只不过他全身被黑暗气息笼罩，身上散发出腐朽、堕落的气息，仿佛是一尊绝世大魔王。
<br/>　　“杀！”
<br/>　　那人见偷袭不成，手持黑色长棍再次向两人砸来，这人的力量也就相当于大罗金仙境界的修士，道祖一掌拍出，便将此人拍成黑雾，可是，不过眨眼时间，黑雾凝聚人体，这人便再次复活，继续向两人杀来。
<br/>　　道祖继续出手，他手持造化玉牒，一道道玄妙的法则被他施展出来，惊天动地。一次次地击杀，一次次的复活，但随着复活次数的增多，那人的力量也越来越小。显然，这种复活也不是无限制的。最后，此人被道祖生生磨死。
<br/>　　两人继续向前，他们再次遇到了这种黑暗怪物，他们似乎是能够操控黑色物质，若是一个不慎被侵蚀了，自己恐怕也要变成这样的怪物。
<br/>　　元始甚至猜测，这些人便是被黑色物质同化了，变成怪物，早已非人了，因为他们的身体是受那诡异物质在操控着。
<br/>　　“道友，不知你是否发现，这些人都是至尊或者真仙境的修为，而且又好些都是以前在星空中闯出赫赫威名的存在？”道祖一边战斗一边说道。
<br/>　　“不错，他们的确是至尊与真仙境的强者。”元始点头道，至于是不是星空中人，元始表示，自己不是这个世界的人，又怎么会知道他们是不是。
<br/>　　“那猎食的“天”恐怕就在里面，这些人物都曾经是他的食物，可如今不知为何被黑暗感染了变成这幅样子？”道祖有些担忧道。
<br/>　　“这些人早已经死了，他们体内的灵魂、血肉都已经不是原先的自己，他们早已被黑暗控制了。”元始手持混沌钟，击碎一黑暗生物，开口说道。
<br/>　　“不错，的确如此。”道祖此时也明白了。
<br/>　　“当——”
<br/>　　此刻元始也放开手来战斗，他手中混沌钟声一响，泛起圈圈音波，空间便被定住了，静止不动，钟声一落，同时混沌钟向着空间一绞，便将这些黑暗生物震死，连元神都不剩下，以元始的修为，对付一群堪比大罗的真仙或者堪比金仙的至尊，又有混沌钟这等开天至宝在手，还不是手到擒来的事情。
<br/>　　“道友好法力，好法宝！”
<br/>　　道祖见了，深深感叹道。心中却直叹，这人到底是从何而来，怎会有如此高深的法力。还有那神钟，有点像太一的东皇钟，可却比东皇钟强大好几个层次。
<br/>　　元始不语，收了神钟，继续向前，他有预感，真相离他们已经不远了。
<br/>　　两人不知走了多远，再次遇到拦路之人，只不过，如今的拦路人修为已经达到准圣级别了。
<br/>　　“擅闯禁地者，杀无赦！”
<br/>　　这人形生物居然是一具骷髅，浑身破破烂烂的，几乎没有丁点完好的不为，每一根骨头都坑坑洼洼的，早已腐朽得不成样子了，但即便如此，他身上散发出的气息，却强大无比。
<br/>　　破烂的骷髅中，竟然还能够传出阵阵精神的波动，只见他手持一柄破败的战剑，如猛兽发狂一般，向元始两人扑杀过来，看似拙劣不堪，实则迅猛无比，就算是准圣中期高手，都难以抵挡。
<br/>　　“当——”
<br/>　　一道钟声悠扬响起，阵阵音波扩散开来，让那前行的神秘骷髅微微一滞，但仅仅只是一瞬间的功夫，那破败的战剑便毫不犹豫地斩了下来。
<br/>　　“嗡——”
<br/>　　战剑与神钟相撞，令人想不到的是，竟然让神钟发出嗡嗡响声，像是激起了战意，神钟迅速做出反应，变得巨大，开始与战剑交战起来。
<br/>　　“怎么可能？”
<br/>　　元始也不由得为之一惊，混沌钟可不是普通的先天至宝，那可是最为顶级的自开天至宝，距离混沌至宝也不远了，可如今一口破败的残剑，竟然能够与之大战而没有落入下风，难道这剑也是先天至宝？开玩笑，什么时候先天至宝也开始烂大街了？
<br/>　　虽然意外，但元始并没有放弃进攻，混沌钟蕴含了时间与空间的力量，更能聚集周天星辰之力，没有必要与对方硬碰，元始将钟身一转，变得强大无比，空间禁锢，时间静止双技能发动，那神秘骷髅还没反应过来，便被定住身形，紧接着变得巨大的神钟罩了下来。
<br/>　　“铛铛铛——”
<br/>　　一阵强烈无比的音波在神钟之内震荡开来，其内的空间尽皆被震得粉碎，时间已经失去了意义，这就是混沌钟的威能，镇压周天，可不是说说而已，就算是混元大罗金仙圆满的高手都不敢小觑。
<br/>　　天道境之前，混沌钟的作用都非常大。
<br/>　　神州收起，原地早已不见了神秘骷髅的身影，只留下一柄残破的战剑在斩破虚空，上下起伏着，元始顺势收起战剑，这剑能够与混沌神钟硬碰，肯定不简单。
<br/>　　道祖在一旁看得只觉得心头骇然无比，他还只是一个大罗圆满的修士，哪见过准圣大战啊，此刻看到神秘骷髅与元始大战，更让他觉得这元始已经超越了真仙。
<br/>　　元始双眼一眯，他知道这次自己可能要遇到劫数了，这才开始，就有准圣了，后面还不知道有多少高手，不过，想来应该不会太多。而且，他觉得自己的对手绝对超越了准圣。
<br/>　　机缘，还是劫数？
<br/>　　深吸一口气，元始继续向前，既然选择了这条路，便没有回头，即便再艰难，就算咬着牙也要走下去。
<br/>　　黑色迷雾之中，隐藏着太多未知的凶险，接下来的短短时间内，元始便再次遇到了数次突袭，袭击他的有人形生物，也有兽形生物，这些生物，无一例外，全部都是被黑暗腐蚀了大半，拖着残破躯体的骷髅，而且他们的兵器，也大多是残破不堪的，但却十分强大。
<br/>　　这些生物，有的生前是大罗，有的是准圣，甚至有的已经达到了混元大罗金仙的地步，可惜，如今都已经死了。
<br/>　　当初到底是发生了什么，竟然有如此惨烈的大战爆发，还有这些人是哪里来的。恐怕谁也不知道，这样一个小地方却也隐藏着惊天之谜。
<br/>　　这一刻，元始深深地感受到了宇宙的浩瀚，强大如混元大罗金仙也会战死，被黑暗侵蚀，他一个准圣又能如何？
<br/>　　不再多想，元始继续向前，修道无悔！
<br/>　　一路上，也让元始收到了好多残破兵器，到了此时，他也知道了，这些并不是先天至宝，这只是一种特殊的材料，硬度方面十分强悍罢了，能够与混沌钟硬碰。这倒是炼器的好材料，以后有时间再炼几件法宝。
<br/>　　终于，在行了近百里之后，两人发现了一个悬浮于虚空上的庞然大物，那赫然是一座岛屿，岛屿之上，隐隐能够看到有一片建筑，只是被黑色迷雾笼罩，难以看清。
<br/>　　近前，终于看得清晰些，一片残破的建筑群，破败腐烂的气息，坍塌的建筑，只有半边建筑还算完好，匾额亦只剩下了半截，上面刻着“太玄”两字，后面应该还有字，但是被毁去了。
