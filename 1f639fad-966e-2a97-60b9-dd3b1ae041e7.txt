　　回到黑印城客栈，已经是中午时分，简单的吃过午饭，慕清霄给了点小费，让客栈准备了两桶清水，两人就钻进了屋内。
<br/>　　小医仙双手抱腿，静坐在床榻上，美眸温柔的望着他。
<br/>　　慕清霄设下结界后，盘坐在床榻的另一头，将千年寒冰玉盒取了出来，直接打开，一抹金光就将卧室照耀的通透。
<br/>　　金光消散，将阴阳玄龙丹捏在手中，金光流转，两条细小的金色神龙，在丹内盘旋。
<br/>　　淡淡的龙吟声，透出丹外，仔细聆听，那一丝细微的龙吟声，似乎隐隐有着一股奇异的魔力，竟能够让人的灵魂有着瞬间的失神与颤抖。
<br/>　　慕清霄清楚，这是来自于血脉的威压，阴阳玄龙丹是两头龙系魔兽的魔核炼制而成，而龙系魔兽，拥有稀薄龙族血脉也很正常。
<br/>　　血脉虽稀薄，但龙毕竟是龙，不是普通血脉能比拟的，就像小医仙能感受到威严，但慕清霄却丝毫不感觉，因为他的血脉更强！
<br/>　　捏着阴阳玄龙丹，慕清霄微微一笑，几大势力拼命争夺，现在却落入自己手中，微微摇头，直接将丹药放进嘴中。
<br/>　　丹药入口即化，化作一股暖流，顺着喉咙灌入四肢百骸中。
<br/>　　澎湃的热流犹如洪水一般，铺天盖地的涌进身体中，七品丹药中蕴含的能量果然不俗，慕清霄面色如常，如同老僧入定，盘坐在床榻上。
<br/>　　几分钟过后，如同所料，药效并没有出现，澎湃如洪水一样的热流，犹如百川汇海，渐渐沉淀，直到最后完全消失。
<br/>　　慕清霄睁开双眼，攥了攥拳头，他知道药效潜伏在自己的体内，不过庞大的药力，倒是让他实力略微有些提升。
<br/>　　化神八重中期！
<br/>　　同时，他还发现，自己不仅仅实力有略微的提升，气质也产生了一丝变化，高贵中多出了一丝威严的气质，犹如君王。
<br/>　　沉吟片刻，慕清霄就有了头绪，这估计就是龙气所致。
<br/>　　当初他炼化过龙元，身上本就携有龙气，现在吞服阴阳玄龙丹后，帝王气质更加明显，甚至完美的继承了帝王的气质。
<br/>　　收敛气息，慕清霄就将对面的小医仙一把拉入怀中，望着她娇羞的模样，邪火蠢蠢欲动。
<br/>　　但他知道，现在可能乱来，急忙运转心法，将刚刚滋生的邪火压了下去，语气温柔，道：“小妮子，快把衣服脱了，开始修炼吧。”
<br/>　　话语刚落，小医仙羞得抬不起头来，但见慕清霄颇为认真的模样，微微点头。
<br/>　　伸出纤纤玉手，慢慢解开衣扣，雪白嫩滑的肌肤就暴露在空气中，待白裙完全褪下，一具完美无瑕的玉体就呈现在了眼前。
<br/>　　慕清霄舔了舔干涩的嘴唇，不禁多瞧了几眼，抱着她来到木桶前，伸出手指，测了下水温，随后将她放进木桶中。
<br/>　　“平心静神，只管吸收药力，过程可能会有一点痛苦，忍耐一下就过去了，乖乖听话。”
<br/>　　听得他那温柔至极的话语，小医仙痴痴的点了点小脑袋，随后取出七彩韵灵果，一口一口的吃了下去。
<br/>　　七彩韵灵果化作一股热流窜入她的体内，使得小医仙秀眉紧蹙，柔美至极的脸颊上噙着一丝痛楚。
<br/>　　不过，痛楚并不明显，可见这小妮子还是很努力的，只见她闭上美眸，静靠在木桶中，只露出雪白嫩滑的脖颈，开始吸收药效。
<br/>　　慕清霄掌心窜出一缕黑色火焰，取出凤火磷果，培髓根，以及在‘千药坊’中购买的金精，紫金千元果。
<br/>　　五种灵药皆是五品，正是五品淬体液的配方，其中两样是来自范凌的纳戒中。
<br/>　　根据范凌先前对七彩韵灵果一副志在必得，根基虚浮的模样，估计资质并不好，灵药想必是他搞来，准备自己淬体用的。
<br/>　　不过，现在倒是便宜了小医仙，七彩韵灵果搭配五品淬体液，绝对能将她的资质提高到一定程度。
<br/>　　估计范崂与范凌做梦也想不到，他们处心积虑想要得到的东西，全都便宜了别人，甚至连性命都给丢了。
<br/>　　几分钟后，七彩韵灵果生效，彩色的流光在小医仙玉体上流转，使得她的玉体渐渐红润，一丝丝白雾自木桶中升腾而起。
<br/>　　同时，也有一点点黑色物质，顺着小医仙毛孔钻出。
<br/>　　经历过淬体，慕清霄自然知道这是人体内的杂质，七彩韵灵果效果倒是出乎他的意料，当下也不再犹豫，灵魂力量控制着灵药，九霄琉璃焱开始提炼精纯的能量。
<br/>　　在九霄琉璃焱的焚烧下，凤火磷果化作一团赤红色液体盘旋在空中，紧接着是培髓根，最后是紫金千元果......
<br/>　　等三种灵药提炼完毕，化作三股精纯能量，慕清霄开始控制着能量渐渐汇聚在一起，当灵药相互接触，一股狂暴的气息就扩散而出。
<br/>　　抓准时机，将金精也融入其中，狂暴的气息竟然逐渐平和，最后消失不见，三种灵药也有了相融的趋势。
<br/>　　半小时后，一股浓郁的药香味蔓延至房间中每一个角落。
<br/>　　望着空中一团碧绿通透的液体，慕清霄眼中闪过一丝笑意，九霄琉璃焱消失，五品淬体液炼制成功。
<br/>　　取出六个玉瓶，将淬体液装进其中，剩余的淬体液，则在他的控制下，流淌进木桶中。
<br/>　　视线落在木桶中，小医仙的玉体，此刻被一团七彩色流光所笼罩，
<br/>　　淬体液刚进入木桶，水源就变成一潭绿色，强大的药效就发挥作用，顺着小医仙玉体上的每一个毛孔，钻进她的体内。
<br/>　　只听见小医仙痛哼一声，银牙咬着粉唇，脸颊上浮现一抹明显的痛楚，淬体液开始生效。
<br/>　　七彩韵灵果改造身体所带来的痛苦，淬体液所带来的痛苦，坚持下去，便是新生。
<br/>　　“仙儿，淬体液助你淬体的同时，也在帮你稳固根基，虽有一丝痛楚，但忍耐过去，往后控制厄难毒体后，才能展现出最大的优势。”
<br/>　　望着木桶中，柔美脸颊上带着痛苦的小医仙，慕清霄叹息一声，他又何尝希望自己的女人受苦，但这一步终究是要走的。
<br/>　　否则，往后厄难毒体带来的痛苦，将会是致命的。
<br/>　　通往强者的路上，资质极其重要，但有一样东西，比资质更加重要，那便是毅力。
